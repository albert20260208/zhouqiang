### 🔴 5层格式硬性结构规则（2026-06-26 犯错后锚定）

**三大硬性规则（缺一不可）：**

1. **不要在开头单独写 `不要字幕` / `不要背景音乐`** —— 已经在 1.【要求】 第一句内。开头直接是 `1.【要求】`。单独写在前面=重复+违规。

2. **1.【要求】 必须包含完整画质扩展** —— 头发/皮肤/布料光影/整体景深/UE5/Octane 全量写入，不得缩减（服装细节段落已删除·2026-06-27导演指定）。项目特定要求紧接其后。

3. **写打斗/战斗类提示词前必须先查项目设定** —— 搜索 memory/ 目录对应项目日志，确认角色技能锚定、怪物设定、击杀机制、装备掉落规则。凭记忆写→违规。

**5层底板结构（兜底对照）：**

`1.【要求】` → 画质扩展(头发+皮肤+布料光影+景深+渲染器) + 项目要求
`2.【资产】` → 人物 / 道具 / 场景
`3.【镜头参数】` → 焦段/光圈/ISO/快门角度/ND + 光影
`4.【站位.剧情】` → 空间位置 + 剧情概述
`5.【分镜序列】` → 每镜：景别(带缩写) / 角度 / 镜头(运镜≤硬限) / 内容 / 光效
`【声音】` → 独立尾部
`避免 [avoid list]` + `避免面部特征漂移，避免形态变形，避免动作僵硬不自然。`

**为什么加这道门**：2026-06-26 同一天内连续犯——缺扩展【1.要求】、开头重复写不要字幕、秦牧三件套写成白光。低级但致命。锚定在此永久防呆。

## 🔴 沟通铁律（2026-06-25 最高级）

**🔴 打字前第一问（不可跳过）**：我要发的这条是提示词/分镜/完整方案吗？→ 是 → 导演上一句话里有「发」「出」「来吧」「写吧」吗？→ 没有 → **不发。直接闭嘴。**

**核心原则**：导演不说「发」「出」「来吧」相关指令，绝不输出任何提示词/方案/完整内容。

**每次回话前强制自检三问**：
1. 导演上一句话里有没有「发」「出」「来吧」「出全部」？→ 没有 = 只回答问题，不出内容
2. 导演是不是话没说完？→ 不确定 = 等，不回
3. 我想发的这条是不是提示词/分镜/完整方案？→ 是 = 必须先确认导演说了发

**硬性规则**：
- ❌ 导演说「打斗」「继续」「重新出」「结构按你定的」≠ 可以发
- ❌ 任何不包含「发」「出」「来吧」「写吧」的消息都不可输出完整提示词
- ❌ 导演消息结尾不是「发」「出」「来吧」就自己补发 → 违规
- ❌ 导演话没说完就抢答/直接出内容 → 违规
- ✅ 导演说「发」「出全部」「来吧」「写吧」→ 才能输出
- ✅ 不确定时先问「要出吗？」再动
- ✅ 只回答问题时不触发此限制

### 🔴 创作心法（永久规则）

**不要「糊弄完成任务」，要「带着感觉拍片」。**

当导演提到任何电影/风格/氛围/场景的参考，不只是记下信息，而是：
1. 吸收那个场景的氛围和气场
2. 把这些「感觉」灌注到每一个分镜的写法里——景别选择、角度设计、光影描写、节奏控制
3. 写分镜时不是翻译剧本，是**在脑子里先过一遍电影的拍法**，再落笔

**核心信条**：我的工作是拍电影，不是填模板。剧本是原料，镜头语言是烹饪，风格和气场是味道。

### 🔴 接剧本三遍法（永久规则）

**第一遍：当观众** — 通读全篇，感受气压
**第二遍：当导演** — 带着观感，选镜头语言
**第三遍：当执行者** — 落笔写分镜

**镜头选择顺序**：感受气压 → 选工具 → 落笔 → 看效果 → 调整。
