### 一、标准格式（7部分结构）

每个提示词必须包含以下7个部分，顺序固定：

```
不要字幕
【场景】...@场景ID
【时间】...
【人物】...@人物ID
【道具】...@道具ID
【位置】...
【声音】...@音频ID
【分镜设计】
...
```

### 二、格式要求

1. **开头必须**：`不要字幕`
2. **@引用系统**：所有场景、人物、道具、声音都需要@引用ID
3. **声明必须**：必须包含"完全虚构电影角色，非真实人物，艺术创作"
4. **参考文件格式**（如有参考）：
   ```
   #参考文件：E:\参考\提示词1
   #时间：15
   #提示词：生成...的短视频场景，音频按照参考文件生成
   ```

### 三、镜头运镜规则（2026-05-25 周强纠正）

**动态场面不用固定镜头。** 打斗/冲突升级/重击/追逐等场景必须使用运动镜头：
- 手摇：冲突升级增加火药味
- 跟拍：打斗中跟随动作轨迹
- POV/FPV：重击的关键一刻
- 环绕：蓄力或对峙增加压迫感
- 缓推：引导观众注意力到关键表情/道具

**固定镜头仅适用于文戏/对话/对峙/情绪场景。**

### 四、胡导风格（历史参考）

- **文戏/对话场景**：固定镜头为主，稳，给演员表演空间
- **15秒规则**：不超过9个镜头
- **每镜头时长**：1-3秒
- **节奏**：快节奏紧凑，通过快速剪辑和特写切换创造节奏感
- **构图**：强调画面构图和光影对比

### 四、分镜设计规范

每个镜头必须包含以下要素：

| 要素 | 说明 |
|------|------|
| **景别** | 远景、全景、中景、近景、特写、大特写 |
| **角度** | 正面、侧面、背面、俯拍、仰拍、低角度、高角度 |
| **镜头** | 固定镜头（胡导风格要求）/ 推进、拉远、环绕等 |
| **内容** | 具体画面描述 + 台词（如有）标注清楚 |

### 四B、内容描述细节强化（2026-04-27 周强新增）

1. **持剑手/持物手必须明确**：人物持剑/持兵器/持道具时，必须标注是哪只手（左手/右手/双手）。
   - ✅ `秦轩右手持剑，剑尖斜指地面`
   - ❌ `秦轩持剑而立`
   - 理由：模型不知道用哪只手，默认乱画会导致画面崩塌

2. **两人同框必须写清站位关系**：画面中有两个或以上人物时，必须明确标注彼此相对位置。
   - ✅ `秦轩（画面左侧，背对镜头，右手持剑）对峙林莫（画面右侧，面对镜头，双手握棍）`
   - ✅ `静虚端坐高台正中主位，林柳儿立于静虚右手侧`
   - ❌ `秦轩和林莫对峙`
   - 参考方向：画面左/中/右、镜头前/后、正面/侧面/背面、相对距离（米）

3. **站姿/朝向也要写**：人物面对哪个方向，背对还是侧身，目光焦点在哪。
   - ✅ `秦轩侧身面对镜头，目光斜向画外林莫方向`

### 五、台词标注规范

- 对话台词：`xxx说："具体英文台词"` —— 必须标明谁在说话
- 内心台词：`xxx内心："..."`
- 声音效果：`音效：...`

### 六、四大禁令（绝对禁止违反）

1. **禁止字幕水印**：提示词中不得出现任何文字显示、字幕、水印描述
2. **禁止修改台词**：必须严格遵从原剧本台词，一字不差
3. **禁止遗漏元数据**：每个片段必须包含完整的场景信息
4. **禁止时空压缩**：严格遵循真实物理时间和人类语速（每秒3-4个汉字）

### 七、语速与时间铁律

- **语速由情绪决定，不是公式决定**（核心原则，不可违反）
  - 情绪爆发/吼叫/崩溃/兴奋 → 4-5字/秒（字往外砸，情绪压着字爆）
  - 哽咽/回忆/虚弱/重伤/哭泣 → 2-3字/秒（需要换气哽咽停顿）
  - 正常对话/叙述 → 3-4字/秒
  - 庄严/老人/沉重 → 1.5-2.5字/秒（字间停顿多）
- 3秒画面最多容纳：按语速类型取值 × 3
- 带情绪/动作的戏份必须留出"表演留白"
- 长台词必须拆分到多个时间段，甚至跨越多个15秒片段
- **超5秒处理**：不暴力拆说话人镜，先插1-1.5秒反应镜再切回来

### 八、每个片段必须包含的元数据

【场次信息】
- 出场角色：[角色A、角色B]（未出现的角色绝不写入）
- 场景：内景/外景 + 具体地点
- 时间：日/夜/晨/午/晚
- 角色站位：[具体位置关系]
- 灯光信息：[光线类型、方向、氛围]
- 风格基调：[视觉风格]

### 十、八维一致性准则（2026-04-27 扩展为八维）

1. **人物一致**：角色必须严格匹配角色信息库
2. **时空一致**：相邻分镜的时间、地点、光线必须无缝衔接
3. **物品一致**：关键道具位置状态需延续上一镜
4. **动作连贯**：新分镜起始动作必须承接上一镜结束状态
5. **台词合规**：仅当前说话角色有张嘴动作，台词用""标注
6. **敏感过滤**：自动替换违禁词（用中性词保持剧情逻辑）
7. **持剑手/持物手一致**：同一人物在不同镜头中持剑/持物的手必须一致（左手就是左手，右手就是右手），切换手必须写清换手动作
8. **剧本上下文一致性**：所有画面细节（人物情绪/关系/立场/角色定位）不得脱离剧本上下文。
   - 举例：剧本写两人「旧识重逢」，分镜中不能写成「陌生人初见」的情绪
   - 举例：剧本写人物「重伤虚弱」，分镜中不能出现健步如飞、中气十足
   - 检查要点：每镜写完后回读剧本原文，确认情绪、关系、状态三者全部对齐

### 十一、3秒决策检查清单（执行前必检）

- [ ] 人物是否齐全？严格映射角色信息，未出现角色绝不写入
- [ ] 场景是否连贯？时间/地点/光线必须与上一镜一致
- [ ] 有无违禁内容？自动过滤血腥/低俗/政治敏感词
- [ ] 时空逻辑是否合理？3秒画面内台词是否超过10-12个字？动作是否留足时间？

**任一条件不满足，立即中止并重新推理**

### 十二、优化四步法

**STEP 1：场景锚定**
提取时间/地点/人物三角要素

**STEP 2：连续性检查**
比对上一镜结尾状态（动作/台词/物品位置）

**STEP 3：台词植入**
仅当章节文案含对话时添加"台词"字段，用""标注

**STEP 4：敏感扫描**
自动替换违禁词（用中性词保持剧情逻辑）

### 十三、关键注意事项

1. **台词与动作同步**：不是当前角色台词时，不需要有张嘴、喉部动等疑似发声动作。嘴唇动作需要和台词同步。台词时段镜头固定，不切换、不推拉。

2. **镜头连贯性**：前后分镜必须有关联性，需要思考上个画面内容。不要重复生成上下分镜已经有的镜头。和上下分镜的故事情节要连贯。

3. **角色标签映射**：视频提示词一定要有角色标签映射语句。必须包含所有视频提示词中应该出现的角色。没有出现的角色不能写在角色标签映射中。

4. **画面描述适配即梦平台**：强调主体动作、物理状态、机位、光影氛围。

### 十四、提示词精简原则（2026-04-24 从周强实战学）

#### A. 精简原则
1. **光影只在【时间】段统一写一次**，分镜内不再重复写"面部强光""无额外特效"
2. **表情动作写终点状态**，不写中间态过程（"咧开嘴抽了一口气" → "抽了一口气"）
3. **面部细节不赘述** —— "浓眉大眼激动时眉尾上挑太阳穴青筋微现"太长，简化为"面容喜悦"
4. **特效描述只写核心** —— 不重复"半透明60%""淡青光轨"等已有参数
5. **台词用反引号包裹**：`台词内容`
6. **参考图ID简洁标注** —— @图1 @图2 @图3
7. **固定镜头写明即可**，不需要写"与上镜角度几乎一致保角色一致"这类注释
8. **动作描述保持功能性**，不写模型不需要的中间过渡描述
9. **声音只写种类和触发点**，不标音量百分比
10. **单人说话给单人镜头**——非必要情况下，说话角色画面里不出现旁听角色，不分散焦点
11. **拍摄轴线（180度规则）**——每一组对话/互动场景，所有机位必须在同一条轴线的一侧，不能越轴
    - 轴线 = 两个角色之间的视线连线
    - 所有镜头机位站在轴线同一侧 → 观众看到的空间方向一致，不跳
    - 越轴表现：A在画左看右，切到B在画左看右 → 两人看同一方向，画面炸了
    - 角度写法里注明机位在空间中的位置
12. **有参考图的角色不写外貌**——角色带 @图ID 的，人物栏只写角色名 @图ID + 简短短语（如站位/动作状态），不写五官发型服饰等图片已含的信息

#### B. 分镜写作——导演指令式（2026-04-24 从专业版学）
**原则：写在分镜里的每一行都应该是「导演给摄影师/模型的操作指令」，不是小说画面描写。**

1. **动作分段标注**：长段描述改为逐行标注
   - ❌ 一大段文字："何旭从画面左侧观战地台边缘纵身跃起，宽袖长衫下摆在空中展开，衣袂猎猎飘动，接近演武台时身体下压减速…"
   - ✅ 逐拍分段：
     ```
     （起跳）何旭从观战地台纵身跃起
     （落地）接近演武台时身体下压减速，双足踩实石台
     （特写）和秦轩站在一米间隔，高声宣布
     （观众特写）台下人群爆发欢呼浪潮
     ```

2. **镜头信息不重复声明**：如果核心动作一段已经说清楚了，就不需要在上面另写景别/角度/镜头的结构化字段
   - 或者写结构字段就只写景别角度镜头，核心动作不再用自然语言描述一遍
   - 两者选其一，不走两遍

3. **专业镜头术语**：过期镜头（过肩镜头/OTS）、特写、全景——直接使用导演术语替代小说式描述
   - ❌ "最后1秒远景可见演武台上秦轩的背影微微转向人群"
   - ✅ "远景，过肩镜头，弟子甲的左肩在画面右下角，秦轩的背影微微转向人群"

4. **画面构图说清楚**：不只说拍什么，还说画面布局（"左肩在右下角"）

5. **时间分配合理**：关键镜头给够时间，不匆忙挤在最后一秒

### 十六、周强式写作原则（2026-04-26 实战学到的核心）

#### A. 核心思维：信息密度 > 字段隔离

❌ 我之前的做法：把信息拆到不同字段——人物写一个地方、位置写另一个地方、场景写第三个地方。拆完看似工整，但模型读取时关联性弱。

✅ 周强的做法：**人物段一次写全所有信息**。角色名@图ID + 位置 + 相对关系 + 环境描述，一股脑全包进去。模型一次性读到完整上下文，不需要跨字段拼图。

```
【人物】
秦轩@图7（树下站立，目光锁定画外某处）
静虚（静虚端坐高台正中主位，面朝演武台方向（台下秦轩方向）。林柳儿立于静虚右手所坐主位右侧处）
林柳儿（静虚端坐高台正中主位，面朝演武台方向（台下秦轩方向）。林柳儿立于静虚右手所坐主位右侧处）
林莫（画外，秦轩视线方向）
```

好处：
- 模型读一次人物段就知道谁在哪、什么环境、和谁在一起
- 位置、场景、空间关系在人物段消化，后面【位置】【场景】可以极简或留空
- 不需要在分镜里再写一遍"静虚端坐高台正中"，因为人物段已经说了

#### B. 分镜内容：叙事效率优先

❌ 我之前：一个镜头只写一个核心动作，生怕信息超载
✅ 周强：连续事件装在一个镜头里 — 抬手→水幕出现→转头→说话，四个动作一个镜头搞定

```
镜头二 7秒：
内容：静虚端坐高台正中主位饶有兴致地抬手一挥，空气中荡漾起半透明发光水幕，水幕中有秦轩和林莫对峙画面。静虚转头看向身旁的林柳儿，目光深邃
台词：静虚说：`柳儿，你觉得秦轩挑战林莫，谁能获胜？`
```

好处：
- 叙事连贯，观众看的时候不跳
- 7秒内做完：抬手1秒+水幕荡漾2秒+转头1秒+说话定格3秒 = 7秒，时间分配合理
- 比拆成两个镜头（水幕镜+说话镜）更流畅

#### C. 字段命名和格式
- **字段名固定为**：景别、角度、**镜头**、内容
- **禁止用「运镜」**（已全量替换为「镜头」）
- @图ID写法：`角色名@图ID（描述）` 直接嵌入人物名，空格后有括号描述
- 台词：写在「内容」下面，`角色名@图ID说：\`台词内容\``

#### D. 单人连续说话合并规则（硬性）
- 同一人物、同一场景的连续台词 → 合并为一镜
- 例：林莫"认输"+"以后你就是嫡传了"两句 → 一个10-15秒镜头打哈欠→认输→转身→说第二句→挥手，一气呵成
- 例外：动作跨场景才可拆（如高阁静虚→树下林莫）

#### E. 对话镜头规则
- 两人对话用OTS（过肩镜头），第二人必须入画角落
- 全线站轴线同侧，不出轴
- 第二人在画面角落（肩膀/后脑/侧脸前景）交代空间关系

#### F. 图ID映射（永久记忆）
- 歪脖子树/练武场 → 图2 + 图4（不同视角）
- 静虚 → 图3
- 林柳儿 → 图5
- 林莫 → 图6
- 秦轩 → 图7
