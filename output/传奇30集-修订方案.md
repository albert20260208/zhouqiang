## 逐集修订

### 第1集：组队

**修改1：开场加速**
原文本以「老李头趴在柜台上」慢节奏开局。
改为：30秒蒙太奇快剪→进入杂货铺对话。
在「一、杂货铺·日」之前增加一段：

```
（蒙太奇快剪·约20秒）
画面：比奇城全景，鸡在街道上跑。战天涯在森林里砍半兽人。斧头砍在硬皮上溅出火花。怪物倒地的瞬间，他摸了摸空钱包。切回比奇城，他站在杂货铺门口，阳光照在青蛙皮上。
（画外音：人在传奇——不是在打钱，就是在去打钱的路上。）
```

**修改2：矿洞挖矿闪回**
在战天涯付钱与老李头对话之间插入：

```
战天涯数金币的手指停了一下。他低头看了一眼指尖的灰——那是矿区石灰岩的印子，洗了三遍都没掉干净。
（画外音：传奇玩家的第一桶金——要么靠爆装备，要么靠挖矿。战天涯选的是后者。在比奇矿洞里蹲了四天，挖的铁矿够老张头打十把菜刀了。卖矿的钱凑够了青蛙皮的尾款——这就是战士的浪漫，连新盔甲都是用命换的。）
```

### 第19集：道心（原缺失标题）

**修改1：加标题**
原「【第十九集】」改为「第十九集：道心」

**修改2：灵魂黑夜强化**
在道小仙走进太和殿门槛之前，加入内心独白：

```
道小仙站在太和殿门槛前停了一步。从比奇城组队到现在，她第一次一个人走这条路。以前的她从来不怕走夜路——她是整理经书的，白天黑夜对她来说只是纸页的颜色不同。但今天她站在殿门口，脑子里想的不是藏经阁的危机，而是城墙上的那个晚上——战天涯说"计划永远赶不上变化"的时候，她第一次感觉自己算不过来了。
```

---

### 第26集：沙巴克前夕

**修改1：加入被守尸梗**
在法小冷城墙上说完「最后一个阵型——我打前锋」之前，插入被守尸练习：

```
（前一天·盟重城外）
战天涯在沙地上画了一个圈，代表守尸的法师火墙覆盖范围。
法小冷蹲在他旁边——昨晚她在模拟PK训练中被一个法师守了尸体。复活→站起来→被雷电术劈死→再复活→再被劈死。连续四次，对面那法师就站在她的尸体旁边，一动不动。
"什么感觉？"战天涯问。
"不爽。"法小冷说这个字的时候没表情，但声音比平时低了半个调。
"记住这个感觉。沙巴克攻城的时候，对面也会这样守我们的人。我们要做的就是——比他们还狠。"
```

### 全局修改

**1. "白光一闪"画外音（加在爆装备时刻）**

第11集（白野猪爆龙之戒指）：
```
（画外音：白光一闪——地上多了一枚戒指。老玩家管这个叫"出装备"——三个字里装的从来不只是一件装备，是整个星期的期待。）
```

第16集（爆战神盔甲）：
```
（画外音：战神盔甲爆出来的那一刻，战天涯还不知道——这件衣服他得再打好几天的工才能穿得上。但这不重要。重要的是白光闪了，装备出了。这就够了。）
```

第21集（天魔神甲紫光爆出时）：
```
（画外音：紫光——极品装备的专属色。每一个见过这道光的老玩家，都知道接下来要发生什么。）
```

第24集（裁决之杖爆出时）：
```
（画外音：白光一闪——地上出现了一件装备……然后第二件……再然后第三件。祖玛教主大爆，是传奇玩家能遇到的最爽的画面之一。）
```

**2. 28级/35级等级梗**

第2集（战天涯提出"上了"前）：
```
道小仙补了一句："等你到二十八级学烈火剑法，打半兽勇士直接一刀暴击带走。"
战天涯看了她一眼："你怎么知道我多少级？"
"进门就看面板了——二十三级的战士，攻击力刚好够用炼狱斧的门槛。"
```

第19集（道小仙学完神兽后）：
```
师父说："三十五级才能学神兽，你没到三十五就先把书打出来了。"
道小仙低头看了看背包里的技能书："运气好。"
"运气好的人多了。能活着把书带回来的，不多。"
```

**3. 安全区强化（第9集）**

追加一段战天涯站在安全区内的内心活动：
```
（画外音：盟重土城的安全区——老玩家管它叫"精神故乡"。不管外面的风沙多大、打架多凶、被怪追得多惨——只要走进那个圈，就安全了。战天涯在安全区站了十秒，第一次觉得在盟重这地方，自己算是有个家了。）
```
