# 输出前必检清单
**写任何视频提示词前必须逐条过完，缺一条不输出**

## 🔴 核心原则（优先于一切）
- [ ] 我做的是导演级别优化，不是格式套用。先判断结构：哪些段该合并、哪些信息量不够独立成段、节奏是否合理
- [ ] 不要在导演给的分段上一套格式就还给他

## 格式类
- [ ] 开头写「不要字幕」了吗？
- [ ] 格式是标准多行字段吗？（景别/角度/镜头各一行，内容单独一行，谁说单独一行）
- [ ] 每一镜都写了景别、角度、镜头、内容四要素？
- [ ] 台词写的是「谁说：`台词`」格式吗？（不是「台词：谁说」）

## @引用类
- [ ] 提示词里没有任何@图ID和场景图引用？导演自己配

## 时长与语速类
- [ ] 每段不超过15秒？
- [ ] 每个单镜不超过5秒？（有台词也限）
- [ ] 超过5秒的单一镜头必须切镜，不能硬塞
- [ ] 没台词的画面每秒推进剧情，不灌水？
- [ ] 段8收尾内容少就不做满15秒？
- [ ] 台词时长按**人物情绪**算，不按固定公式
  - 激动/愤怒/紧急 → 快，1秒4-5字
  - 虚弱/重伤/喘息 → 慢，1秒1-2字+停顿喘息
  - 庄严/沉重/冷笑 → 字间有停顿
  - 3-4字/秒只是基准参考，不是铁律

## 镜头类
- [ ] 全是固定镜头？（没有推拉摇移）
- [ ] 连续动作一镜到底不拆？（喘气→抬头→喊话不分成两镜）
- [ ] 单人说话用近景/特写？（不给中景）
- [ ] 说话人说完一句完整台词前不切成旁听角色反应镜？
- [ ] 反应镜只放在台词结束后？
- [ ] 画外音覆盖时不切换主体画面？

## 场景类
- [ ] 场景不变时，是持续写场景状态变化（不是只写「承接上一段」）？
- [ ] 上一段的环境状态（兽人、火光、声音等）下一段还在吗？

## 空间类
- [ ] 每段人物站位关系是否明确？（画左/右/前/后/距离）
- [ ] 拍摄轴线和机位是**按该段人物站位重新定**的吗？（不是一条线打全剧）
- [ ] OTS肩膀方向对吗？谁的肩膀入画哪个角占多少比例？
- [ ] 拍烈焰是正面偏左，不是偏右

## 物理与一致类
- [ ] 持剑手持物手都写清了？（左手/右手/双手）
- [ ] 人物服装/武器/位置每镜一致？
- [ ] 打斗收尾做完了吗？（打完了就逼近/定格/反应，不悬半空）
- [ ] 所有规则对照USER.md和MEMORY.md再过一遍有无遗漏？
