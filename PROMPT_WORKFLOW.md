# 🔴 每次出提示词前强制工作流

**此文件高于一切。每次出提示词前必须从门0走到门4，跳一步=违规。**

---

## 门0：沟通铁律（是打字之前）

```
[ ] 导演上一句话里有「发」「出」「来吧」「写吧」吗？
    → 没有 → STOP。只回答问题，不输出提示词/分镜/方案。
    → 有 → 进门1。
```

---

## 门1：查数据库（是动手写之前）

```
[ ] memory_search + grep 查 SOUL.md/SKILL.md/MEMORY.md 相关规则
    → 特效固定写法、技能锚定、格式模板、打斗规则
[ ] 读 SKILL.md §角色特效固定描述（确认能量刃/冲击波/弧光锚定文字）

[ ] 读 SELF_CHECK.md（逐条勾）
    → 以上全过 → 进门2。
```

---

## 门2：写前锁定（是动笔之前）

```
[ ] 导演给的格式模板是什么？——复制下来，一字不动当底板
[ ] 角色有没有参考图？——有→不写外貌。不知道→问，不准猜图号
[ ] 场景类型？——打斗/文戏/空镜→匹配对应规则
[ ] 15秒还是其他时长？——确认
[ ] 段头：第一个镜头就是动作进行中，不写传送/降临/光团炸开
[ ] 段尾：最后一击即结束，不写站定/pose/定帧
    → 以上全过 → 进门3。
```

---

## 门3：写完自检（逐条勾）

### 格式
```
[ ] 格式和导演给的模板逐行对比——一字不多一字不少
[ ] 没有导演模板里不存在的行（如不要字幕独立行）
[ ] 子项冒号/括号跟模板一致
[ ] 每镜秒数正确（不是全写0-3秒）
[ ] 不确定的图号不填（留空让导演配）
```

### 特效锚定（每镜核对）
```
[ ] 挥剑→弧光（火焰在剑划过处留下发亮光痕残影）+能量刃（白色月牙形能量刃从剑身飞出）
[ ] 命中→冲击波（白色透明能量波从撞击点扩散至全场）
[ ] 剑身火焰→每镜出现剑必须锚定火焰包裹剑身
[ ] 怪攻击有自己的特效描写（颜色/形态具体）
[ ] 弧光≠能量刃≠冲击波（弧光是剑轨不飞出去，能量刃是飞行弹道，冲击波是碰撞扩散）
```

### 段头段尾
```
[ ] 段头：第一个镜头是动作进行中（不是传送/降临/特效铺垫）
[ ] 段尾：最后一击即结束（不站定不摆pose不定帧）
```

### 打斗规则
```
[ ] 打斗镜头是运动镜头（非固定镜头）
[ ] 每镜有力度词+速度词
[ ] 秦移动是弧线/折线（不直线不站桩）
[ ] 怪也会攻击秦（有来有回）
```

### 技术参数
```
[ ] 焦段+光圈+ISO+快门角度+ND ——全写
[ ] 景别带英文缩写（WS/MS/CU/ECU/LS/MCU）
[ ] 3秒镜≤2种运镜
```

### 台词（如有）
```
[ ] 格式：谁说/情绪+说：`台词`
[ ] 台词带语气符号
```
