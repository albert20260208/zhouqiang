# 🔴 出提示词前必过自检清单（v2.0 · 5层格式通用版）

> 更新：2026-06-29 | 对齐5层底板 · 放开运镜约束 · 区分图片/文字参考 · 通用化
> 此文件每次出提示词前必须逐条勾完。一条不过=不许发。

---

## 门0：沟通铁律（打字之前）

- [ ] 导演上一句话里有「发」「出」「来吧」「写吧」吗？
  - 没有 → STOP。只回答，不输出完整提示词/分镜/方案。
  - 有 → 进门1。

---

## 门1：参考类型确认（写之前）

- [ ] 角色/场景/道具有参考吗？已确认类型？
  - **图片参考**（即梦@图ID·实际图片文件）→ 不写外貌，只写位置+动作状态
  - **文字参考**（对话中给的描述/角色档案）→ 必须写入提示词，不能跳过
  - 不确定 → 追问区分，不准猜，不准一刀切

---

## 门2：格式底板对照（写之前）

- [ ] 已读格式底板？默认 `PROMPT_FORMAT.md`（格式一·5层）；导演说「格式二」→ `PROMPT_FORMAT_V2.md`（格式二·7字段）
- [ ] 逐段对照结构？
  - `1.【要求】` — 开头直接写，不单独写"不要字幕""不要背景音乐"
  - `2.【资产】` — 人物/道具/场景，图片参考去外形，质感尾缀照加
  - `3.【镜头参数】` — 焦段/光圈/ISO/快门角度/ND + 光影（全量写入）
  - `4.【站位.剧情】` — 空间位置 + 剧情概述一句话
  - `5.【分镜序列】` — 每镜：景别(缩写)/角度/镜头/内容/光效
  - `【声音】` — 极简，环境音一行，角色声+打斗声+特效声句号分隔
- [ ] 结尾有 `避免面部特征漂移，避免形态变形，避免动作僵硬不自然。`？

---

## 门3：逐镜检查（写完后逐条）

### 景别←→可见粒度
- [ ] 当前镜景别 + 英文缩写（LS/MS/MCU/CU/ECU）
- [ ] 内容只写了这个景别能拍到的东西？
  - 全景/远景 → 身体轮廓/位移/空间关系/光影方向，不写表情细节
  - 中近景 → 脸部表情/眼神/嘴型，不写全身动作
  - 特写 → 面部肌肉/眼部微动/单颗汗珠
  - 单人对话镜 → 永远是近景或特写

### 角度←→空间关系
- [ ] 角度写了两样：机位位置 + 脸与镜头的关系？
- [ ] "正面"后面标了谁的正面？
- [ ] 人物朝向和他该面对的方向一致？

### 运镜（已放开·文戏打斗皆可运动）
- [ ] 3秒/镜≤2种运镜（引导线·5秒+可多于2种·不叠同一窗口·不叠同类）
- [ ] 不叠两种冲击型运镜（手摇+POV）
- [ ] 不叠两种过程型运镜（跟拍+环绕）
- [ ] 文戏可用固定镜头，但根据情绪节奏可选运动镜头
- [ ] 打斗/动态场景 → 运动镜头（手摇/跟拍/POV/环绕/缓推）

### 人脸交代
- [ ] 脸与镜头的关系写清（侧脸30°/四分之三侧脸/正脸/背面/剪影）
- [ ] 拍不到脸 → 没写表情和光在脸上？
- [ ] 全段扫描：关键情绪节点至少一镜拍到正脸？

### 特效锚定
- [ ] 每镜特效一笔写穿形态，不写概括词（❌"光效华丽" ✅"火焰螺旋环在暗雾中切开环形橙色光道"）
- [ ] 项目固定特效描述一致？（先查项目设定，不凭记忆写）

---

## 门4：跨镜连续性

### 180度轴线
- [ ] 所有镜头中，角色的左右位置关系一致？
- [ ] 相邻镜头没有越轴？（A画左看右 → B画右看左才算越轴）
- [ ] 镜头角度都在轴线同一侧（除非有意越轴）

### 景别递进
- [ ] 相邻镜头角度不重复？
- [ ] 景别变化有理由？
- [ ] 中全景为主（70-80%），近特插关键节点

### 物理连续性
- [ ] 动作承接上一镜结束状态？（不跳步）
- [ ] 物体状态不可逆？（打飞的剑不会自己回来，物品变化需写过渡）

---

## 门5：全段总检

### 时长与节奏
- [ ] 每镜时长够动作做完？
- [ ] 单镜≤5秒？
- [ ] 不硬凑15秒？（内容够了10/12/14秒都行，不灌水）
- [ ] 台词字数÷语速（情绪定语速）→ 分配秒数够？留白够了？
- [ ] 长台词不暴力截断？（插反应镜再切回来）

### 角色
- [ ] 只列该段实际出现的角色？未出现=不写
- [ ] 人物名用导演指定的简称
- [ ] 有参考图的角色 → 外貌由图片兜底，只写位置+动作+质感尾缀
- [ ] 有文字描述的 → 外貌已写入
- [ ] 质感尾缀全量追加（`服装面料细节拉满…肤质统一`等完整段落）

### 台词
- [ ] 谁说/情绪+说：\`台词\`
- [ ] 台词带语气符号
- [ ] 台词不做任何修改（与原剧本一字不差）

### 违禁词
- [ ] 全文扫描违禁词库？（`skills/VIDEO/违禁词审查技能/references/word-list.md`）
- [ ] 含违禁词→已替换？

### 段头段尾
- [ ] 段头：第一个镜头是动作进行中（不是传送/降临/光团铺垫）
- [ ] 段尾：最后一击即结束（不站定/不pose/不定帧，除非导演指定）
