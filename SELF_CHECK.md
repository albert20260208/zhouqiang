# 🔴 出提示词前必过自检清单

**此文件每次出提示词前必须逐条勾完。一条不过=不许发。**

---

## 第一阶段：空间逻辑推演（写任何镜头之前）

> 🔴 多人打斗（3+人）→ 必先读 `skills/prompt-optimizer/references/multi-character-fight-positioning.md`

### 三事实锁定
- [ ] 秦在空间中的位置+朝向？（画左/画右/画中，面朝哪）
- [ ] 攻击者/对方在空间中的位置+方向？（在秦的什么方向）
- [ ] 摄影机在轴线的哪一侧？（这个决定了所有镜头的左右关系）

### 空间布局表（3+人打斗必填）
- [ ] 已填写多人空间布局表？（模版见 multi-character-fight-positioning.md §二）
- [ ] 轴线定义：秦牧与主对手的连线/包围圈整体单元切线？
- [ ] 焦点组合已分配：每镜只聚焦秦牧+1个对手？
- [ ] 主镜（镜1）建立空间→分镜逐焦点覆盖→末镜重立主镜？

### 空间合理性检查
- [ ] 秦的朝向和他的对手在空间中对得上吗？
- [ ] "正面"标了谁的正面？秦正脸对镜头=秦背对敌人？→ 冲突则改
- [ ] 全段镜头角度是否都在轴线同一侧？（除非有意越轴）

## 第二阶段：每镜落笔前（逐镜逐条）

### 景别←→内容匹配
- [ ] 当前镜景别是___ → 能写什么不能写什么？
  - 全景/远景 → 只能写身体轮廓、位移、空间关系、光影方向
  - 中全 → 全身动作、脸的大致朝向、基本姿态
  - 中景 → 腰部以上、手势、脸上大致表情
  - 近景 → 胸部以上、表情细节、眼神、嘴型
  - 特写 → 面部肌肉、眼部微动、单颗汗珠
- [ ] 内容中没写超过景别能拍到的细节？（全景不写牙关/目光，中全不写眼珠方向）

### 角度←→空间关系
- [ ] 角度写了两样：机位位置 + 脸与镜头的关系？
- [ ] "正面"后面标了谁的正面？
- [ ] 人物朝向和他该面对的方向一致？（秦面朝右打猩猩→不能正脸对镜头）

### 画面里有啥写啥
- [ ] 内容只写了这个角度+景别下画面内真实可见的东西？
- [ ] 没编造画外内容？（正面拍一人=画面里就一个人）
- [ ] 没写镜头拍不到的脸部细节？

### 人脸交代
- [ ] 脸与镜头的关系写清了吗？（侧脸30°/四分之三侧脸/正脸/背面/剪影）
- [ ] 拍不到脸→没写表情和光在脸上？
- [ ] 侧脸→只写侧面可见的？没写正面专属的（眉间/鼻翼/双眼同时）？
- [ ] 全段扫描：关键情绪节点至少有一镜明确拍到正脸？

## 第三阶段：跨镜连续性

### 180度轴线
- [ ] 所有镜头中，秦和对手的左右位置关系一致？（秦画左→永远画左，除非画面内转身）
- [ ] 相邻镜头没有越轴？（A画左看右 → B画右看左）

### 景别递进
- [ ] 相邻镜头角度不重复？
- [ ] 景别变化有理由？（不是同一景别同一人物无变化硬切）
- [ ] 中全景为主（70-80%），近特插关键节点？
- [ ] 打斗镜头是运动镜头？（非固定镜头）

### 物理连续性
- [ ] 动作承接上一镜结束状态？（不跳步）
- [ ] 物体状态不可逆？（打飞的剑不会自己回来）
- [ ] 没有同景别同人物无动作变化切镜？

## 第四阶段：全段总检

### 时长与节奏
- [ ] 每镜时长够动作做完？（不把多个动作挤在1秒里）
- [ ] 单镜≤5秒？
- [ ] 不硬凑15秒？（内容够了就收）
- [ ] 台词字数÷语速 = 分配秒数够？

### 格式头
- [ ] `不要字幕`
- [ ] `真人写实风格`
- [ ] `不要背景音乐`
- [ ] 【光影】只是一句话？

### 人物道具
- [ ] 有参考图的不写外貌
- [ ] 【人物】只列该段实际出现的角色（未出现=不写）
- [ ] 人物名用导演指定的简称

### 台词
- [ ] 谁说/情绪+说：\`台词\`
- [ ] 台词带语气符号

### 动作锚定（2026-06-24新增）
- [ ] 秦所有动作带力度词+速度词？（裸动词=废稿）
- [ ] 大猩猩所有动作带力度词+速度词？
- [ ] 挥剑→月牙白色冲击波发射？（每挥必写）
- [ ] 格挡→火星迸溅+力量回震+手臂绷紧？
- [ ] 砍中→伤口可见（裂开/渗血）？
- [ ] 收势→段尾有收招定格？（不可硬切）
- [ ] 节奏→每镜标注快/中/慢/骤停？（不全段一个拍子）
