## 🎯 项目定位

每次为该项目生成视频提示词时：
1. **必须先读** `scripts/` 下的最新剧本内容
2. **必须参考** `CHARACTERS.md` 的角色视觉建模
3. **必须遵守** `LOCATIONS.md` 的场景设定
4. **提示词风格**：喜剧反转、仙侠古风、青蓝色调为主

---

## 🎬 核心看点

1. **林莫的\"摆烂\"艺术**：表面酗酒废物，实则暗中白嫖系统奖励
2. **静虚的\"慈祥\"伪装**：表面仙风道骨，实则老谋深算要吸干弟子
3. **秦轩的\"内卷\"执念**：原著气运之子，正义感爆棚却只是下一棵蛊苗
4. **何旭的\"打脸\"日常**：每次阴谋算计都精准撞在林莫外挂上
5. **林柳儿的\"纠结\"内心**：对林莫怒其不争，内心深处却暗藏情愫

## 📌 生成提示词时的核心要求

### 视觉风格
- **色调**：青蓝色调为主（青莲宗）
- **年代感**：古风仙侠
- **场景**：宗门、练武场、古风建筑
- **特效**：剑气、灵力、罡气（透明微弱气流）

### 人物基调
- **林莫**：慵懒、醉意、眼神常带三分醉意与慵懒
- **静虚**：表面慈祥实则阴冷
- **秦轩**：坚毅、凌厉、正气
- **林柳儿**：古典仙女、冷艳
- **何旭**：肥胖、阴柔、讥讽

### 禁止项
- 字幕/水印
- 修改台词
- 破坏人物设定
- 时空压缩
