## 🏛️ 青莲宗

**整体风格**：古风仙侠宗门，青蓝色调

---

## 📍 主要场景

### 1. 练武场 / 演武台

| 属性 | 描述 |
|------|------|
| **类型** | 外景，日景 |
| **描述** | 练武场四周旌旗招展，数千名弟子齐声呐喊，声浪震天 |
| **重要道具** | 歪脖子树（位于最边缘）、演武台 |
| **灯光** | 日光，明亮 |
| **氛围** | 热闹、喧嚣、宗门大比 |

**拍摄要点**：
- 旌旗招展 → 风吹旌旗飘动
- 数千弟子 → 群演阵容
- 歪脖子树 → 林莫的专属位置，画面边缘

---

### 2. 歪脖子树

| 属性 | 描述 |
|------|------|
| **位置** | 演武场最边缘 |
| **描述** | 百年老树，枝干歪斜 |
| **意义** | 林莫的专属位置，与热闹氛围格格不入 |
| **关键道具** | 朱红酒壶（林莫的） |

**拍摄要点**：
- 林莫半躺在树干上，一只脚悠哉地晃荡
- 手里拎着朱红酒壶，正仰头猛灌
- 与远处的火热气氛形成强烈对比

---

### 3. 静虚房间

| 属性 | 描述 |
|------|------|
| **类型** | 内景，夜间 |
| **描述** | 深夜的宗门寂静得可怕，房间内灯火通明 |
| **重要道具** | 金色光罩、茶座茶具、檀香 |
| **灯光** | 灯火通明，金色光罩发光 |
| **氛围** | 神秘、压迫、仙气氤氲 |

**拍摄要点**：
- 静虚盘腿坐在金光罩之中闭目吐纳
- 檀香袅袅
- 宝相庄严
