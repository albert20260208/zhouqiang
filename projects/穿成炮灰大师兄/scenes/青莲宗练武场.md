# 青莲宗练武场 - 核心场景

## 场景信息
- **类型**: 外景，宗门核心建筑
- **出现集数**: 第1-4集, 第15-20集
- **核心功能**: 宗门大比、生死擂台、大决战

## 空间布局
```
            高台（静虚端坐）
          ↗
  演武台 ←──── 众弟子方阵
          ↘
            歪脖子树（林莫专属位置）
```

## 场景元素
1. **演武台**: 中央高台，比武用，石质。生死擂时升起四道半透明光柱形成光幕
2. **高台**: 老祖观礼台，位于演武台正前方，高于地面
3. **众弟子方阵**: 演武台四周，人群密集
4. **歪脖子树**: 演武场最边缘的一棵老树，林莫的专属躺平位置
5. **旌旗**: 四周招展的宗门旗帜
6. **护宗大阵**: 半球形半透明光幕（第17集升起，后被静虚捏碎令牌解除）

## 不同时段视觉特征

### 宗门大比（白天）
- 热闹、人声鼎沸
- 晴天，阳光普照
- 旌旗招展
- 弟子服饰统一（青莲宗道袍）

### 生死擂（白天）
- 肃杀、凝重
- 云层压顶（气氛烘托）
- 擂台光幕升起，隔绝内外
- 弟子围在擂台四周

### 大决战（白天→魔气弥漫）
- 静虚魔气扩散，光线变暗
- 地面裂痕
- 碎石废墟（高台被毁）
- 众弟子倒地或逃散

## 关键镜头位置
1. **高台俯视** - 静虚视角，俯瞰整个练武场
2. **演武台正面** - 标准对战视角
3. **歪脖子树方向** - 林莫的独特视角，从边缘看热闹
4. **人群中央** - 弟子视角，仰视擂台

## 灯光氛围
- 常态：自然日光，青色主调
- 生死擂：云层遮挡，半阴天
- 魔气扩散：暗青色，光线变暗
- 仙盟降临：金色光芒（周敏降临）

## 场景一致性锚点
- 青石地面，演武台居中
- 四周旌旗（颜色统一）
- 歪脖子树在场边缘（第1-3集重要，后被打碎但位置不变）
- 高台位置不变
