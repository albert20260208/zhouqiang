# 静虚房间 - 关键场景

## 场景信息
- **类型**: 内景，宗门核心建筑
- **出现集数**: 第4-6集, 第12集
- **核心功能**: 深夜密谈、试探、阴谋策划

## 空间布局
```
   门 ←── 入口
    │
    ↓
   茶座（林莫跪坐/对坐）
    │
    ↓
   蒲团（静虚盘坐）
    │
    ↓
   金光罩（静虚修炼时）
```

## 场景元素
1. **檀香**: 袅袅升起的檀香
2. **金色光罩**: 静虚盘腿坐在光罩中修炼，宝相庄严
3. **茶座茶具**: 静虚挥袖即出现
4. **蒲团**: 静虚的坐垫
5. **法阵**: 地面有法阵，泛着血光（第6集召唤天魔）
6. **黑暗密室**: 法阵所在（在房间深处）

## 不同时段视觉特征

### 日常状态
- 灯火通明
- 檀香袅袅
- 静虚盘坐金光罩中
- 仙气氤氲

### 天魔召唤（第6集）
- 昏暗
- 地面法阵泛着血光
- 黑气翻滚
- 天魔虚影凝聚

## 关键镜头位置
1. **门口视角** - 林莫站在门口拱手
2. **茶座对坐** - 林莫与静虚面对面
3. **俯视蒲团** - 静虚盘坐

## 灯光氛围
- 日常：暖黄烛光，祥和
- 密谈：烛光摇曳，阴影多
- 天魔召唤：暗红血光，黑气弥漫

## 场景一致性锚点
- 蒲团/茶座位置固定
- 檀香始终存在
- 地面法阵在深处
- 房间空间大小不变
