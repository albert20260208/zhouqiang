# 梵火仙牢 - 核心场景

## 场景信息
- **类型**: 内景，地下
- **出现集数**: 第5-14集
- **核心功能**: 林莫的"躺平圣地"

## 空间布局
```
   通道入口 ←─── 石门（厚重，有禁制）
       │
       ↓
   仙牢主体 ─── 中央地火裂缝（翻滚的赤红岩浆）
       │
       ↓
   最深处牢房 ─── 林莫的冰床位置
```

## 场景元素
1. **石门**: 厚重，添加了重重禁制，滚烫
2. **通道**: 赤红一片，两侧石壁涌动着火毒
3. **中央地火**: 地底裂缝中喷涌着赤红的岩浆火毒，灼热气浪翻腾
4. **冰床**: 林莫用五行仙法制造的玄冰大床，在地火之上
5. **风扇**: 灵力凝聚的风扇，挂在墙壁上吹冷气
6. **毛毯**: 林莫盖着毛毯睡觉
7. **金色符文锁链**: 第11集静虚设置的封印，数十道大树般粗壮，封锁石门

## 不同时段视觉特征

### 改造前（第5集初入）
- 红光冲天，火毒弥漫
- 灼热、压抑
- 只有岩浆和石壁

### 改造后（第5集起）
- 冰床蓝光与岩浆红光交织
- 有冷气流通
- 林莫躺在冰床上盖毛毯
- 形成冰火两重天的独特视觉

### 封印后（第11-14集）
- 石门被金色符文锁链封锁
- 光幕覆盖石门
- 其他不变（林莫继续躺平）

## 关键镜头位置
1. **门口视角** - 看冰床和岩浆的强烈对比（林柳儿推门而入的震惊视角）
2. **冰床俯视** - 林莫躺平，盖毛毯
3. **岩浆特写** - 赤红翻滚
4. **石门特写** - 符文锁链封印

## 灯光氛围
- 基础：暗红色（岩浆反射）
- 冰床：幽蓝色光
- 封印时：金色符文光芒
- 整体：暗色调，只有红蓝金三种颜色

## 场景一致性锚点
- 中央地火裂缝位置固定
- 冰床在地火之上（悬浮/拔地而起）
- 墙壁上的风扇位置
- 石门厚重且有禁制光芒
