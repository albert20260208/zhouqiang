# 2026-05-18 工作日志

## 周强纠正·短片节奏（17:12-17:14）

### ❌ 不需要的：灌水停顿
- 7个字台词就是3秒，不额外加1秒"酝酿/停顿"拖长单镜
- 该3秒走完的台词不多给

### ✅ 必须做的：剪辑切镜反应帧
- 单人说话超5秒 → 单镜画面没新意了
- 中间必须插对面反应镜/环境镜（1-2秒），再切回来
- 这是剪辑节奏的必要手段，不是灌水

### 区别
灌水 = 拖长单镜时长
切镜 = 打破单调画面，维持视觉新鲜感

### 🔴 做分镜的核心流程（17:44 周强纠正·当天最重要纠正）

**错误习惯（我以前的做法）：**
先选机位/景别 → 再往镜头里填内容 → 卡住了找过渡方案 → 越修越死

**正确流程：**
先想叙事逻辑（这段要讲什么、什么信息优先级最高）→ 再推机位和切镜

**关键判断：** 逻辑对了，切镜不会有问题。机位是逻辑推出来的，不是先定好机位再往里硬填内容。

### 🔴 入场+说话一镜包干（17:34 实战案例）
- 战天入场+说第一句「互不干扰」= 连续事件链
- ✅ 镜头一镜到底：正面平视中景，入场步伐→站定→开口，5秒全包
- ❌ 拆成两镜：低角度仰拍入场3秒 + 中景站定说话5秒 =多了一个无意义的切

### 🔴 长台词切镜法：过听者肩拍说话人（17:29 周强纠正）
说话人说连续长台词时，要切镜但不能断焦点：
- 镜1：说话人近景/中景
- 镜2：**过听者肩膀拍说话人**——听者的肩膀/后脑在画面角落前景
- 好处：画面变了（观众不疲劳），焦点没断（镜头里还是说话人主导），听者的反应也能顺带交代
- **不是三镜：说→反应→说，是一镜就把两件事都做了**

### 🔴 说话人台词没说完时，切镜规则
- ✅ 切焦点在说话人身上的机位变化（近景→过听者肩拍说话人）
- ❌ 切给听者的独立反应镜头

### 🔴 细节是为了传递情感（18:51 周强纠正）
- 道具细节不只是"交代信息"，更是"传递情绪"
- 冰瑶摩挲药瓶→法力流转→收起来 = 犹豫→权衡→下决定的情绪弧线
- 如果只是写"冰瑶犹豫"再切"冰瑶决定"，那是干巴巴的信息堆砌
- 通过一个道具的物理变化来承载人物内心变化，才是好的细节
- **关键判断**：这个细节在传递什么情感？不是"这个道具出现了没有"

### 🔴 做分镜前先筛原稿细节（18:49 周强纠正）
- 导演原设计里的每一个道具动作细节都有叙事功能
- 蓝药瓶=从"诚意"到"权衡"到"决定"的视觉载体，不是多余的
- 做提示词前先过一遍原分镜：哪些是"叙事链条里的动作节点"，一个都不能漏
- ❌ 错误习惯：只把台词拎出来就开始做镜头设计
- ✅ 正确做法：先理解每个道具/动作在叙事链中的角色，再判断它是保留还是砍

### 🔴 台词也可以精简（18:13 周强纠正）
- 看到长台词先问：能不能砍掉多余字？
- 去掉"如果""就算""了""吧""的"等无信息量的虚词
- 精简后意思不变、节奏更紧——这是优化的一环
- 前提：导演允许的情况下做

### 🔴 画外音技巧（18:04 周强纠正）
- 提到不在场的人（如"他要烈火剑法"战天）→ 玄灵画外音覆盖，画面切听者反应
- 好处：信息没断（台词继续推进），画面有新内容（听者反应），不提前穿帮人物出场
- 画外音是重要的过渡工具，熟练运用

### 🔴 景别选择铁律：景别跟着台词情绪走（17:18 周强纠正）
- 亮身份、秀底气、展自信 → **近景锁脸**，不是中景
- 道具信息在近景里一样能看到，不需要为道具拉到中景/全景
- 中景/全景只在需要交代空间关系时使用（比如"我们从这里穿过去"）
- **核心原则**：先问这句话在传递什么情绪/信息，景别为情绪服务，不为道具服务
