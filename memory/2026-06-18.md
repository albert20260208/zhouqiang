## 🔴 今日被纠正的错误（已写入SOUL.md/MEMORY.md）

### 1. 台词铁律 ⚠️ 反复犯的错
- ❌ 裸台词：`还不错，换上试试。` / `升级神力。`
- ✅ **每句台词必须标注：谁说 + 情绪语气 + 面部三部位 + 台词**
- 格式：`秦牧无奈道：\`台词内容\``
- 缺一不可！模型不知道裸台词是台词

### 2. OTS左右铁律 ⚠️ 第N+1次犯错
- 背对镜头：人物右肩=画面**左下**，人物左肩=画面**右下**
- 写OTS前停一秒确认方向

### 3. 入镜部分才写面部三部位
- 白光吞没/背影/特写道具 → 脸看不清，不硬凑
- 有脸的镜头 → 每镜每人眼+嘴+脸三部位

### 4. 人物名不带数字编号
- 秦牧4 → 秦牧（导演自己配图，不需要编号）

### 5. 换装特效不花哨
- 柔白光水波流转，不堆粒子炫光

### 导演给了一段降龙掌vs碧海曲的山巅对决让我分析并改进
- 场景从山巅换到竹林，融入风/碎光/竹叶/竹竿参与打斗
- 提炼出五维升级标准：

| 维度 | 原版问题 | 升级后 |
|------|---------|--------|
| 节奏 | 5×3秒均匀 | 不对称节奏（炸药引信） |
| 运镜 | 客观记录 | 镜头参与动作（急停/拉升/微颤） |
| 动作 | 段落描述 | 秒级节拍拆解（@0-1s/@1-2s） |
| 光影 | 镜1强后面淡 | 每镜独立写光源/面光/色温 |
| 特写 | 无 | 微型炸弹嵌入（手/眼/道具特写切出再切回） |

### 升级版已出稿（15秒/7镜 含微型特写），镜头5上帝视角俯拍冲击波收尾

## 🔴 打斗新标准（后续全部应用）

1. **秒级节拍**：一镜多攻防，每秒标注动作节点
2. **动力链**：蹬地→转腰→送肩→发力，不只写结果
3. **多视角**：背后跟拍/仰拍/环绕/俯拍/特写，五镜五个不同空间位置
4. **光影每镜**：光源→方向→色温→打在谁脸上，每镜写完反问
5. **场景参与**：竹竿摇晃/落叶切开/地面结霜/冲击波扫弯→场景是动作的同伴不是背景
6. **传奇技能融入**：野蛮冲撞/半月弯刀/刺杀剑术/攻杀剑术/烈火/逐日/开天斩

4. **Style Anchors** — 引用知名电影/品牌风格激活整组视觉参数，比堆形容词效果好
5. **Negative Anchors** — 负面锚点明确告诉模型什么不该变，防漂移有效
6. **Lens Switch关键词** — `lens switch →` 或 `cut to` 实现单Prompt多镜头切换
7. **Identity Blend** — 多参考图光照/角度不一致导致"平均"出新人物
8. **拆镜生成策略** — 大项目不要一Prompt吃所有镜头

### 🟢 打斗分镜核心原则（17:16学习）
- **三要素**：动作设计（编舞）→ 摄影（手持/多角度/Cutting on Action）→ 节奏（快慢交替，呼吸点）
- **打击力三层**：预备（蓄力）→ 接触（特写撞击点）→ 反应（受击方后退/踉跄）
- **节奏铁律**：持续快=麻木，快-慢-快交替，激烈<30秒插呼吸点
- **镜头配合**：广角建立→中景预备→近景出击→特写接触→中景反应→广角新位置
- **Cutting on Action**：在打击爆发点切镜

### 🔴 20:15 表情僵硬·学习方法论（周强指导）

### 🔴 20:25-20:30 打斗镜头方法论·多维补充（周强指导）
- **镜头要丰富**：不只全景/中景/近景三档，穿插武器特写、打击特写、手部特写、脚步特写（15秒≥6镜，每镜≤3秒）
- **角度不重样**：相邻两镜不用同角度，同一招内也可切镜
- **慢动作在爆发点**：卡在重劈/刺击/格挡瞬间，击飞和被打退也上慢拍
- **俯拍+低角度各用一次**：俯拍拍包围圈/劈落轨迹，低角度仰拍拍压迫感/重招气势

### 🔴 23:05-23:23 深夜补充规则（周强追加）
- **剑气是平A默认视觉效果**：每次挥剑都带，不是技能不是大招，不分轻重
- **剑气延伸攻击距离**：物理剑身之外有能量延伸段，1米剑打到2米外
- **剑气穿透效果**：正面砍一只怪，剑气穿过伤到身后一群
- **战斗风格**：穿梭走位不站桩，边移动边砍，剑气远程
- **中景全景拍游龙感**：中景拍动作舒展，全景拍穿梭走位
- **出招节奏**：起手近景拍蓄力→释放切全景/俯拍拍轨迹和命中
- **难度梯度**：越往后怪越硬，不能每波都碾压一剑清一排

- **怪的攻击镜头**：拍怪打秦，有来有回，不只秦一个人是主体
- **剑气可变巨剑形**：能量剑影放大砸下
