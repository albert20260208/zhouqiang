### 重要沟通规则（2026-05-16 周强纠正）
1. **先给示例确认→等导演说「出全部」→再继续**：不得未经确认就直接输出完整方案
2. **每段必带三要素**：
   - 【拍摄轴线】→ 后来发现模型不认得抽象概念，改为用**物理坐标**锁死在【人物】站位和分镜内容里（画左/右/前/后/距离/朝向），去掉了【拍摄轴线】字段
   - 【人物】精确站位：画左/画右/前/后/距离/转向
   - 所有机位锁定在轴线同一侧（通过物理坐标实现）
3. **同场景人物站位变了**→下段重写【人物】字段，用新站位
4. **换场景**→人物站位可以重新设计

### 景别与信息量规则
- 中景拍3米外的人：看不到肩甲爪痕、瓶身小字等细节
- 踢瓶子这类动作中景够用，不需要切特写
- 每个镜头写之前先想清楚「这个景别在这个距离实际能看到什么」

### 单人镜头时长规则（2026-05-16 周强确认）
- **连续帧同人累计≤5秒**：不是单镜≤5秒就够，要看前后帧——第1镜5秒战天，第2镜又是战天=10秒违规
- 中间必须插反应镜（对方反应/环境细节/特写道具，1.5-3秒）隔开
- 单人镜之间有其他人/道具/环境镜隔开 → 重置累计计数
- 双人中景/全景多人 → 不触发此规则
- 写完一段分镜后，按顺序逐镜扫描累计，不只是看单镜

### 对话OTS规则
- 两人对话必须用过肩镜头（OTS）交代空间关系
- 谁说话→过对方肩膀拍谁，第二人肩膀入画（占画左/右下角约15-20%）
- 两人对话不能各自拍单人近景，否则空间关系断裂

### 台词时长规则
- 语速由情绪决定：情绪爆发4-5字/秒，正常3-4字/秒
- 说多长台词就要给多少秒，不能为了塞进固定时长而压缩

**OTS肩膀入画位置（导演纠正）：**
- 两人对话场景（面对面），前景人站在画面一侧：
  - 前景人在**画右** → 过该人的**右肩** → 右肩从画面**左下角**伸入
  - 前景人在**画左** → 过该人的**左肩** → 左肩从画面**右下角**伸入
- 原因：面的方向决定肩膀的朝向。面朝画左的人右肩在画面左侧方向

### 最终输出格式定稿（17:26 导演确认）
每段提示词字段独占一行，不合并：
```
不要字幕
【场景】...
【时间】...
【人物】
xxx（画左前方行走。xxx在他画右后方约3米）
xxx（画右后方行走。神兽紧跟玄灵脚边画右）
神兽（棕毛巨犬，在玄灵脚边画右）
【道具】...
【声音】...
【分镜设计】

镜头N（X-Y秒）：
景别：
角度：
镜头：固定镜头
内容：...
```
关键：
- 景别、角度、镜头、内容各占一行
- 镜头永远写`固定镜头`
- 每个人物单独一行
- 【人物】【道具】【声音】内信息用句号分隔，不跨行
