## 🔴 最终确认规则（2026-05-07 导演最后一次纠正）

### 空间布局与拍摄轴线
- 画左=牢门/秦轩/何旭（面朝右），画右=林莫玄冰床（面朝左）
- 所有机位站在正前方/观众这一侧，不出轴
- 提示词必须包含【拍摄轴线】字段

### 机位规则
- 画左人物（秦轩/何旭）→ 正面偏左
- 画右人物（林莫）→ 正面偏右
- 全景/定场 → 正面

### OTS过肩镜头使用规则
- **关键对话/情绪点** → 用OTS（锁空间关系）
- **单人说话/普通对话** → 正面单人（不需要每句都过肩）
- 不是全用也不是全不用，该用的时候用
- OTS写法：
  - 过秦轩**右**肩（右肩入画**左**下角约20%）朝右拍林莫
  - 过林莫**左**肩（左肩入画**右**下角约20%）朝左拍秦轩
  - 规则：两人的**面朝方向那一侧的肩**入画对应的底部角落（对称）

### 写提示词必须一次性检查所有维度
1. ✅ 台词一字不差
2. ✅ 每镜≤5秒
3. ✅ 单人一律近景/特写，不给中景
4. ✅ 轴线统一、拍摄轴线写清
5. ✅ OTS只用于关键对话/情绪点
6. ✅ 没台词没信息的空镜全砍（林莫抬眼、嘴角微翘等不做独立镜）
7. ✅ 时长由内容量决定，不硬凑15秒
8. ✅ 角度不写虚假度数（不用"偏左30°"，写"正面偏左"即可）

### 我反复犯的错（绝不再犯）
1. 画左画右搞反
2. OTS肩膀方向写反/绕路越轴
3. 双人场景全用OTS或全不用，走极端
4. 乱编角度度数
5. 没信息量的反应镜独立占用镜头
6. 硬凑15秒
7. 每次只改一个问题、忘了其他维度
