## 📊 执行摘要

### 关键发现 (Top 10)

1. **Seedance 2.0 @ 引用系统** - 字节跳动的多模态引用语法，支持图片≤9 + 视频≤3 + 音频≤3，可精确控制首帧/尾帧/运镜/音乐，是当前最灵活的视频生成提示词系统

2. **"写意图、不写细节" 理念** - 在有世界知识的场景下（如"兰州拉面广告"），只需描述意图，避免过度细节化，让AI发挥导演思维

3. **HunyuanVideo 1.5** - 腾讯开源的轻量化模型（8.3B参数），支持10秒720p视频，引入SSTA稀疏注意力机制，推理速度比FlashAttention-3快1.87倍，单A100约180秒生成

4. **LTX-Video 2.0** - Lightricks推出首个DiT架构的音视频同步生成模型，支持4K@50FPS，成本比竞品低50%，支持多关键帧+LoRA风格定制

5. **CogVideoX 1.5** - 智谱AI开源，支持5-10秒视频，提供2B/5B两版本，FP16下最低4GB显存即可运行（diffusers优化）

6. **提示词优化关键** - 所有主流模型均强调"长提示词 > 短提示词"，详细描述场景/人物/运镜/光线的提示词效果提升30-50%

7. **视频扩散模型架构演进** - 从2+1D（分离空间+时间注意力）→ 纯3D全注意力 → 稀疏DiT（跳过冗余计算），训练成本和推理速度持续优化

8. **Prompt压缩技术** - LLMLingua-2等方法可将提示词压缩3-6倍，同时保持质量（GPT-4数据蒸馏）

9. **测试时计算缩放** - 类似o1/R1的长思维链策略也可应用于视频生成，通过增加推理步数而非模型参数提升质量

10. **开源工具链成熟** - ComfyUI/Diffusers/LightX2V等工具支持一键部署，量化技术（FP8/INT8）使消费级GPU可运行专业模型

## 🎯 分类整理

### 1. 提示词技巧

#### 核心原则
- **明确性 > 简洁性** - 详细描述比简短提示效果好30-50%（所有主流模型共识）
- **结构化提示** - 推荐格式：`[主体]+[场景]+[动作]+[运镜]+[光线]+[风格]`
- **时间分段** - 10秒以上视频建议用 0-3秒/3-6秒/6-10秒 分段描述
- **只增不改** - 多幕视频保持角色一致性：后续修改只在原提示词基础上增加，不删除

#### Seedance 2.0 特有技巧
```
@Image1 作为首帧
@Image2 作为尾帧
@Image1 的人物作为主体
@Video1 的运镜方式
@Audio1 作为背景音乐
场景参考 @Image2
```

#### 镜头语言速查（中英对照）
| 中文 | 英文 | 效果 |
|------|------|------|
| 推进 | Push in / Dolly in | 镜头向前，聚焦强调 |
| 拉远 | Pull back | 镜头向后，展示环境 |
| 跟拍 | Track / Follow shot | 跟随主体移动 |
| 环绕 | Orbit / Arc shot | 绕主体旋转 |
| 希区柯克变焦 | Hitchcock zoom | 推进+缩放，眩晕效果 |
| 一镜到底 | One-take / Oner | 无剪辑连续镜头 |
| 鱼眼镜头 | Fisheye lens | 超广角变形 |

#### Prompt优化工具
- **DSPy** - 自动优化提示词的编程框架（斯坦福ICLR 2024）
- **OPRO** - 使用LLM作为优化器，优化后提示词效果提升50%（Google NeurIPS 2024）
- **TextGrad** - 通过文本反馈进行"自动微分"（Nature发表）
- **HunyuanVideo Prompt Handbook** - 腾讯官方提示词手册（强烈推荐阅读）

### 3. 工作流程

#### 6阶段标准流程（来自本地知识库）
```
1. 构思 → 确定故事核心
2. 选风格 → 希区柯克/库布里克/诺兰/王家卫等
3. 写剧本 → 对白+动作描述
4. 设计镜头 → 运镜+景别+构图
5. 生成参考图 → Seedream 5.0组图功能
6. 生成视频 → Seedance API
```

#### 提示词模板（15秒产品广告）
```
参考@Video1的剪辑风格和镜头转场。用@Image1替换@Video1的产品。
生成15秒产品展示视频。

0-3秒: 产品动态入场，特写表面纹理和logo细节
4-8秒: 多角度转场（正面/侧面/背面），扫描光效
9-12秒: 产品放入使用场景，展示用途
13-15秒: Hero shot，品牌标语出现，背景音乐达到高潮

声音: 参考@Video1的背景音乐。添加产品交互音效。
```

### 4. 画质优化

#### Seedance画质增强5步法（海哥实战）
1. **升级模型** - 切换到Seedance 1.5 Pro完整版
2. **智能增强** - 勾选"画质链式增强"
3. **4K直出** - 分辨率3840×2160，渲染"极致细节"
4. **提示词前缀** - 开头加 `画质强化:4K直出+极致细节+锐利焦点`
5. **双重导出** - HD按钮重绘，尺寸倍率选2x

#### HunyuanVideo优化策略
- **VAE优化** - 3D Causal VAE，16×空间+4×时间压缩
- **稀疏注意力SSTA** - 剪枝冗余KV块，加速1.87倍
- **Step Distillation** - 480p I2V模型8-12步即可生成，速度提升75%

### 5. 一致性控制

#### 多幕角色一致性5要素（必记！）
1. **人物特征** - 发色/发型/发饰/面部特征/身材
2. **服装特点** - 类型/颜色/细节/配饰
3. **场景环境** - 建筑/环境元素/氛围色调
4. **人物比例** - 多人场景必须写明身高比例
5. **运镜方式** - 每幕运镜要明确（避免抖动）

#### 技术手段
- **参考图锚定** - `@Image1的人物作为主体` 锁定外观
- **提示词累加** - 后续幕不删前面描述，只增加
- **风格统一** - 同一项目用同一组参考图

## 🔬 技术深度分析

### 视频扩散模型架构演进
```
2022: 2+1D架构
  ├─ 空间注意力 (2D)
  ├─ 时间注意力 (1D)
  └─ 代表: Latte, VideoCrafter

2024: 纯3D全注意力
  ├─ 联合时空建模
  ├─ 更好的运动连贯性
  └─ 代表: CogVideoX 1.5, HunyuanVideo

2025: 稀疏DiT (Skiparse/SUV)
  ├─ 剪枝冗余时空KV块
  ├─ 35%+加速，质量持平
  └─ 代表: HunyuanVideo 1.5 SSTA
```

### 推理优化技术栈
1. **量化** - FP8/INT8/GGUF，显存降低50-70%
2. **卸载** - CPU卸载+VAE切片，牺牲速度换显存
3. **蒸馏** - Step Distillation，8步=原50步质量
4. **缓存** - DeepCache/TeaCache，重复帧跳过计算
5. **编译** - torch.compile，速度提升10-30%

### Prompt Engineering前沿
- **长思维链(Long CoT)** - o1/R1风格的测试时缩放，通过增加推理步数提升质量
- **自动提示词优化(APE)** - 使用LLM优化提示词，效果超人工50%
- **多模态提示** - 图文+视频+音频组合，@引用系统是代表
