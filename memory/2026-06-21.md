### ppt-master 使用体验
- **工作流**：Source → Project Init → Strategist (8 Confirmations) → Executor (SVG) → Quality Check → Post-process → Export
- **优点**：输出原生可编辑 PPTX（DrawingML），非图片；深色科技风效果不错；自带质检脚本
- **坑**：
  - 流程复杂，Eight Confirmations 需要来回确认（QQ Bot 无法用网页 UI，只能 Chat 确认）
  - SVG 必须手写（不准脚本批量生成），4页还行，大量页数会累
  - `<g opacity>` 在 SVG 中不认，必须逐元素设透明度
  - `<animate>` SMIL 动画元素也不认
  - 质检脚本的 spec_lock drift 为 warning 级别，不阻塞
- **生成文件**：
  - 项目：`ppt-master/projects/smart_security_ppt169_20260621/`
  - SVG：`svg_output/slide_01~04.svg`

- SVG 使用 `font-family="Microsoft YaHei"`，Linux 服务器无此字体

- 需替换为服务器有的中文字体（如文泉驿微米黑 `WenQuanYi Micro Hei`）后再转 PNG
