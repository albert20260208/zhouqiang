### 🔴 本次学到的核-心教训

#### 1. 连续台词同景同人不拆镜（再次强化）
段2冰瑶23字挑衅宣言：原文拆成两镜中间插火墙镜 → 导演纠正合并为一镜5秒（4.6字/秒挑衅语速合理）

#### 2. 同人物同景别连续动作无信息变化不切镜
段3玄灵：看火墙方向（镜头1）+抬手说话（镜头2），导演说合并 → 一镜完成

#### 3. 反应镜+反应后对话应合并为连续事件
段3镜头4（战天反应1.5秒）+镜头5（战天反问）→ 合并为一镜9.5-15秒，战天先反应再反问，连续不拆

#### 4. 周导提示词风格模板（永久使用）

**【人物】段格式：**
```
角色名 位置（描述.描述.描述）
```
- 括号内用句号分隔多个信息
- 空间位置写在角色名后面，不写括号里
- 示例：`冰瑶画左（站火墙左端，侧身面对镜头.面对二人4米对峙站立）`

**不写@引用ID：**
- 【人物】段和【场景】段都不写@图ID
- @UUID只在分镜内容中直接嵌入：`冰瑶@8e4106b9-cfa9-47b5-b585-c0ee3d4b4834右肩占画左下角约20%`

**角度写法：**
- 写"度"不写"°"：`正面偏左10度`

**景别写法：**
- 用"中景"、"近景"、"特写"、"全景"，不用"中近景"（除非裁切部位标注）
- 需要标注裁切部位时：`近景（冰瑶胸部以上景）`

**位置段：**
- 简明：`左端冰瑶（画左），右端散人剪影（画右）对峙画面`

**场景描述：**
- 砍废话："竖着一道"比"横亘一道"利索
- 直接，不要文学修饰

**道具段：**
- 末尾不加句号

**镜头编号：**
- 无用镜头直接跳过编号，不补号
- 镜头1→镜头2→镜头4（跳过无用镜头3）

**内容：**
- 简明扼要，砍掉不需要的过渡描述
- 不写⏱、✅等内部思考注释到提示词里

### 我的错误记录（导演反复骂了我半个多小时）

#### 错误1：删了推进剧情的台词
段1原稿 "我们是来谈合作的"在全景0-5秒，我合并到中景里删了这句 → 导演骂

#### 错误2：灌水反应镜
我自作聪明加了冰瑶握拳、手指收紧、掌心微汗等无台词细节 → 导演骂"拍这个干什么？"

**导演铁律：只要台词推进剧情，不需要的反应镜/装饰性镜头全砍**

#### 错误3：⏱ 计算出现在输出里
示例段里写了"⏱ 21字÷4.2字/秒=5秒 ✅" → 导演骂"怎么又有这个，以后不要带上这个"
→ 已存在于skills/prompt-optimizer/SKILL.md的铁律：提示词中不能带注释/计算

#### 错误4：未经指令就全量输出
导演给我示例后没说"出全部"，我自己输出全部10段 → 导演骂"我没让你发 你又发"

**沟通铁律强化：不给示例等确认，不输全部**（必须听到"出全部"、"来吧"等明确指令）

#### 错误5：优化=格式转换，不是真优化
导演的原意是"优化"=合并同场景段落、压缩冗余 → 我给回去了还是10段，等于白干了

**优化=做结构性合并（10段→8段），不是把原稿的格式整理一遍还回去**

三人谈判戏：玄灵(道士35级) + 战天(战士34级) vs 冰瑶(女法师)，地宫打尸王前组队谈判。

### 全局站位锁定（18:42 导演纠正）
- **玄灵**→画左，**右脸**对着镜头，面向画右（看冰瑶）
- **冰瑶**→画右，**左脸**对着镜头，面向画左（看玄灵）
- **战天**→段1-2在镜头后方位置不出镜，段3从镜头后方迈入画面中央
- **神兽**→段1-2不出镜，段3开始在战天脚下趴卧
- **全线轴线**：摄影师站位 = 战天/镜头方位

### 🔴 核心教训（永久保存）

#### 1. 台词对话的长度决定了段的边界（18:42+）
- 原10段合并为8段（段3+4战天长台词；段5+6+7冰瑶条件+双方回应）
- 但合并后战天53字长台词无法在15秒内装完 → 需要自然分段

#### 2. 5秒说话 + 2秒隔断反应 + 5秒说话 = 标准模式（18:55 导演纠正）
- 单人说话：5秒说话→2秒隔断反应（查四）→5秒说话→结束
- 末尾不要加无用的第三反应镜来凑15秒（我犯了）
- 例：段2原始稿加了冰瑶最终反应镜凑15秒=灌水，砍掉只剩12秒

#### 3. "提到他就给他镜头"（19:00 导演纠正）
- 段1玄灵说"他是战天，战士34级"→必须给战天点头微笑镜头
- 例：段1镜4加战天近景反应镜+画外音

#### 4. 台词秒数要真实（19:01 导演纠正）
- "我们是来谈合作的"（7字）3秒够了，不需要4秒
- 入场+说话一镜搞定，不需要拆成两个镜头（段3：入场3秒+说话5秒=8秒，实际一镜5秒搞定）

#### 5. 出场人物必须出现在人物栏（19:10 导演纠正）
- 段1战天给了镜头但人物栏没写战天→错误
- 人物栏必须和实际出镜人物完全匹配

#### 6. 不出镜角色不写入（延续之前规则）
- 段1战天在镜头后方(7-9s切近景)，神兽全程不出镜→不写
- 段3战天入场，神兽在脚下→写入
