# 2026-04-28 周强导演提示词优化实战

## 🔴 核心教训

### 1. 场景描述不要自己编
- 用户有参考图时不要自作主张描述"古色古香修仙内室、木质茶桌、香炉"等
- 应该先问：**"您有参考图吗？场景是什么样的？"**
- 用户给参考图后，简要描述图中关键元素即可，交给用户自己去@图ID

### 2. 不要加@图ID
- 用户明确要求：以后不加@图ID，他自己加
- 人物和场景写简单描述就够了

### 3. 站位/机位不能猜
- 第一次写了"静虚（左）×林莫（右）"→ 用户问站位图对不对
- 应该问用户摄像机视角下两人的相对位置，不要自己猜
- 后续用户明确了：林莫坐在静虚对面（对坐），不是站着

### 4. 信息密度铁律（最重要）
- 用户给的时长是**上限不是目标**
- 没台词的画面每一秒必须推进剧情或传递信息
- 不要为了"氛围填充"给无效镜头
- 不要过多描述，简洁传递信息即可

**反面案例（我的错误）**：
```
林莫被定住→切4个角度展示表情→再切大远景收束
→ 实际一个镜头3秒就说清楚的事
```
- 该压缩就压缩，不需要每段都填满15秒

### 5. 专业机位标准（周强确认）
两人隔桌对坐对话场景：
- **机位A**：过画面左侧人物的左肩，拍画面右侧人物，偏轴线右侧约30°，前景左肩占画面左下约20%
- **机位B**：过画面右侧人物的右肩，拍画面左侧人物，偏轴线左侧约30°，前景右肩占画面右下约20%
- 全线不越轴
- 单人镜头：说话人同侧偏15°，无肩膀入画

### 6. 镜头切分铁律（周强确认）
- **同景别、同人物、动作无实质性变化 → 不切**，一镜到底
- 不为了凑镜头数而切镜
- 单人画面不超过5秒
- 动作有变化（走路→说话→摸酒壶等）→ 一镜到底不切
- 动作没变且≤5秒 → 一镜到底不切
- 动作没变但>5秒 → 切对方反应镜或环境细节镜（1.5-2秒），再切回来

### 7. 运镜违规（被指出）
- 禁止使用：平移、推进/拉远、环绕、手持晃动、镜头震颤
- 全部替换为：固定镜头 + 景别变化（中景→特写=推的效果）

### 8. OS（内心独白）必须有画面对应
- OS播放时角色的面部表情和肢体动作必须写清
- 不能OS十几秒但画面只有一句话

### 9. 格式规范（保持不变）
- 开头：`不要字幕`
- 8部分：【场景】【时间】【声明】【人物】【道具】【位置】【声音】【分镜设计】
- 声明：完全虚构电影角色，非真实人物，艺术创作
- 人物：写角色名+简单描述，不加@图ID
- 【固定站位】行：用户指定时写，没指定时不要编

### 10. 两人对坐的固定站位
- 静虚（画面左侧，主位，茶桌后）
- 林莫（画面右侧，对面坐，面向静虚）
- 隔茶桌对坐，轴线为左右视线连线
- 拍静虚时：机位在林莫侧朝静虚方向（机位B）
- 拍林莫时：机位在静虚侧朝林莫方向（机位A）
