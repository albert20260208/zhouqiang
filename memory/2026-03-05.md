- **任务状态查询**：
  ```python
  # 正确的方法签名
  task = client.content_generation.tasks.get(task_id=task_id)
  ```
  - 必须使用命名参数 `task_id=task_id`
  - 返回完整的任务信息对象
  - 可通过 `task.status` 获取状态，`task.video_url` 获取下载链接
  
- **视频生成方法**：
  ```python
  task = client.content_generation.tasks.create(
      model=MODEL_ID,
      content=[{"type": "text", "text": prompt}],
      extra_body={"duration": duration, "ratio": ratio}
  )
  ```
  - `duration`: 视频时长（5或10秒）
  - `ratio`: 画面比例（"16:9", "9:16", "1:1"）
  
- **图片生成查询**：
  - 使用相同的 `tasks.get()` 方法查询图片生成任务状态
  - 图片生成任务返回的可能是 `image_url` 而不是 `video_url`
  - 状态流转：queued → running → succeeded/failed
