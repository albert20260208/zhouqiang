### 4. 5层提示词格式——永久标准（SOUL.md 已更新）
排版顺序：
```
不要字幕 {{Mixed N}} 或 无

1.【要求】不要字幕.不要背景音乐.人物无变形.禁止广角.实拍.全画幅手持+稳定器混合. 真人写实，电影质感，高对比度.
2.【资产】人物 / 道具 / 场景
3.【镜头参数】焦段/光圈/ISO/快门角度/ND + 光影
4.【站位.剧情】空间位置 + 剧情概述（合并）
5.【分镜序列】每镜：景别(带缩写) / 角度 / 镜头(运镜≤硬限) / 内容 / 光效
【声音】（全片，独立尾部）
```

- 角度可带英文标注（Low Angle/POV/OTS）
- 分镜内技术标记保留（[crash zoom] [whip pan]等）

### 打斗视觉风格（2026-06-28 独孤九剑参考·火焰版）
- **全程丝滑无卡顿**：动作衔接自然无断点，敏捷流畅带强节奏感
- **快慢镜头闪电切换**：快速挥刀斩击→火焰半透明残影拖尾扩散（橙红渐变炽白，火焰余烬粒子从重影边缘飘散），慢镜头定格肢体舒展的格挡架刀与卸力反制瞬间
- **火焰粒子贯穿全程**：火星/余烬/焰尘随刀速增减密度——快刀时粒子爆量溅射，慢镜时粒子悬浮缓飘
- **残影拖尾**：每次挥刀刀身火焰在空中拖出2-3层半透明火焰残影，层层错开，短暂残留后化为火星散去
- **节奏曲线**：快→慢→快→慢交替，不在同一速度停留超过2秒
- **卸力特写**：慢镜头中兵器架住碰撞瞬间——火花炸开凝滞0.3-0.5秒，火焰粒子以碰撞点为中心呈球形扩散缓飘
