## 项目B：热血传奇短剧 第一集分镜制作

### 本轮重点：前三段反复修正+格式定稿

### 格式最终规范（导演周强确认）

#### 关于不要第一行
- 第一行写：`不要字幕.不要背景音乐`（两个强制约束用点点号连在一行）
- 不再写「宣传CG级别的次世代3D写实风格」等风格词（用户已切换到真人实拍方向）

#### 字段格式（纯文本，无格式）
- 每个字段单独一行，不合并
- 【场景】后跟描述文字，按长度可同行或换行
- 【人物】只写角色名，不写位置/状态描述（更简洁）
- 【道具】后跟道具名，多个道具逗号或空格分隔
- 【位置】一行说明人物相对位置（必要时才写）
- 【声音】只写：说话声、环境声、打斗声、特效声、OS声——不自己乱加
- 时间写`白天自然光`

#### 分镜格式
- `镜头N（X-Y秒）：`单独一行
- `景别：`单独一行
- `角度：`单独一行
- `镜头：固定` 单独一行
- `内容：`单独一行，后面跟详细动作和台词
- 所有字段后不撞行，纯文本

#### 台词规则
- 台词必须严格遵从剧本，一字不差，不能删减
- 除非导演明确要求，否则不能自己砍台词
- 用反引号 \` \` 包裹台词
- 连续台词不切OS，当面说的话就正常说
- 剧本里的画外音才用OS/画外音标注

### 导演纠正的Bug记录

#### 镜头设计Bug
1. **第一镜就上过肩**：开场要定场，先让观众认识空间和人，再上OTS
2. **段尾段头动作断裂**：段1结尾"吃瘪手伸向怀里"→段2开头"掏出金币"，中间不能断
3. **单人连续帧过长**：段1镜头3+4连续拍战天涯9秒，段2开头又是战天涯，超5秒限
4. **同景别同人物无动作不切换**：连续拍同一人报装备时不该切镜
5. **段尾收太死**：段1结尾应该把动作带到段2（吃瘪→摸钱），让段2直接接

#### 格式Bug
1. 人物不要写位置描述，只写名字
2. 声音只写剧本里提到的（环境声/说话声/打斗声/特效声/OS）
3. 不要自己加道具音效（如"瓶罐碰撞声"）
4. "不要字幕"后跟背景音乐约束用点号

#### 道具/装备Bug
1. 战天涯整体改成灵魂战衣（白色布质道袍），从头到尾
2. 武器是裁决（右手一直握着），不是修罗斧
3. 参考图已有的内容不写文字描述（如老李头不写外貌）
