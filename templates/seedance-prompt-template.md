## 🎯 胡导拍摄指导应用

### 核心原则：
1. **多角度多景别，单个静态镜头以固定镜头为主**
2. **人物描述格式：人物+情绪+动作+台词**

**详细指南请参考**：`/root/.openclaw/workspace/templates/hu-director-style.md`

### 核心要点：
1. **多角度多景别**：每个场景使用多种角度和景别组合
2. **固定镜头为主**：90%以上镜头使用固定镜头
3. **标准格式**：人物描述必须使用"人物+情绪+动作+台词"格式
4. **微表情重点**：注重面部微表情的情感传递
5. **情感递进**：通过镜头语言控制情感节奏

### 技术实现：
- **角度多样性**：正面、侧面、俯拍、仰拍、过肩、主观视角
- **景别序列**：远景→中景→近景→特写→极特写
- **固定镜头**：三脚架稳定，精心构图
- **标准格式**：严格遵循"人物+情绪+动作+台词"描述顺序
- **微表情捕捉**：眼神、嘴角、眉毛等细节描述

## 📋 使用指南

### 如何替换素材
1. **图片1**：替换为男主图片
2. **图片2**：替换为女主图片
3. **图片3**：替换为场景背景图片
4. **图片4**：替换为特殊场景图片
5. **音频1**：替换为男主声音
6. **音频2**：替换为女主声音

### 技术要点
1. **固定镜头**：所有镜头都是固定镜头，避免位置混乱
2. **多角度**：正面、侧面、俯拍、背后等多种角度
3. **多景别**：特写、中景、远景、中近景等不同景别
4. **情感递进**：镜头语言跟随情绪变化
5. **安全声明**：添加虚构声明，避免审核问题

### 如果过审有问题
1. 降低写实度：`--realism 0.6`
2. 调整风格：`--style artistic`
3. 简化描述：减少微表情细节
4. 增加安全声明：强调"完全虚构"

### 生成顺序
1. 按分镜顺序生成（分镜1→分镜2→分镜3→分镜4）
2. 每个分镜15秒，4个镜头
3. 按顺序拼接，形成完整场景
