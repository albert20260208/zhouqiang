## 一、核心理念

### "写意图、不写细节" (liangdabiao)

在有世界知识的场景下，只需描述意图，让 AI 发挥导演思维：

✅ 推荐：`生成一个精美高级的兰州拉面广告，注意分镜编排`
❌ 不推荐：描述如何揉面、拉面、浇汤的每个步骤细节

**适用条件：**
- 有世界知识（餐饮品牌/健身动作/历史时期等）
- 不需要特定角色一致性
- 不需要特殊运镜风格

## 二、@ 引用系统

### 引用格式
```
@Image1  @Image2  @Image3  ...  (最多9张)
@Video1  @Video2  @Video3       (最多3个)
@Audio1  @Audio2  @Audio3       (最多3个)
总计: ≤12个文件
```

### 角色分配表

| 用途 | 语法示例 |
|------|----------|
| 首帧 | `@Image1 作为首帧` |
| 尾帧 | `@Image2 作为尾帧` |
| 人物形象 | `@Image1 的人物作为主体` |
| 场景背景 | `场景参考 @Image3` |
| 运镜方式 | `参考 @Video1 的运镜` |
| 动作编排 | `参考 @Video1 的动作` |
| 视觉特效 | `完全参考 @Video1 的特效和转场` |
| 背景音乐 | `BGM 参考 @Audio1` |
| 服装穿搭 | `穿着 @Image2 的服装` |

### 组合引用
```
@Image1的人物作为主体，参考@Video1的运镜和动作，
BGM参考@Audio1，场景参考@Image2
```

## 三、输入限制

| 类型 | 限制 | 格式 | 时长 |
|------|------|------|------|
| 图片 | ≤9张 | jpeg/png/webp | - |
| 视频 | ≤3个 | mp4/mov | 2-15秒 |
| 音频 | ≤3个 | mp3/wav | ≤15秒 |
| **总文件** | **≤12个** | - | - |

**重要限制：**
- ❌ 不支持真人脸部素材
- 输出分辨率：480p-720p
- 输出时长：4-15秒

## 四、提示词结构

### 基本公式
```
[主体/角色] + [场景/环境] + [动作/运动] + 
[镜头运动] + [时间分段] + [转场/特效] + 
[音频/声音] + [风格/氛围]
```

### 结构化模板 (推荐)
```
【场景】环境描述
【人物】角色+动作
【运镜】镜头运动方式
【风格】导演风格参考
【色彩与光线】色调+光效
【质感】渲染方式
```

### 时间分段（10秒以上推荐）
```
0-3秒: [开场描述，镜头，动作]
3-6秒: [中段发展]
6-10秒: [高潮或关键动作]
10-15秒: [结尾，品牌露出]
```

### 参数格式
```
--resolution 1080p    # 分辨率
--duration 5          # 时长(秒)
--camerafixed false   # 镜头是否固定
--watermark true      # 水印
```

## 五、镜头语言

### 基础运动
| 中文 | 英文 | 效果 |
|------|------|------|
| 推进 | Push in / Dolly in | 镜头向前，聚焦强调 |
| 拉远 | Pull back / Dolly out | 镜头向后，展示环境 |
| 平移 | Pan | 水平转动，扫描场景 |
| 摇镜 | Tilt | 垂直转动，展示高度 |
| 跟拍 | Track/Follow | 跟随主体移动 |
| 环绕 | Orbit / Arc | 绕主体旋转 |
| 一镜到底 | One-take | 无剪辑连续 |

### 高级技巧
| 中文 | 英文 | 效果 |
|------|------|------|
| 希区柯克变焦 | Hitchcock zoom | 眩晕、不安 |
| 鱼眼镜头 | Fisheye | 超广角变形 |
| 低角度 | Low angle | 仰拍，权力感 |
| 高角度 | High angle | 俯拍，脆弱感 |
| 鸟瞰 | Bird's eye | 正上方俯视 |
| 第一人称 | First-person POV | 主观视角 |
| 快速横摇 | Whip pan | 动态模糊转场 |
| 升降 | Crane up/down | 垂直升降，史诗感 |
| 航拍 | Aerial / Drone shot | 俯视全景 |

### 景别
| 中文 | 英文 | 范围 |
|------|------|------|
| 极特写 | Extreme Close-up | 眼睛/细节 |
| 特写 | Close-up | 面部 |
| 中景 | Medium shot | 腰部以上 |
| 全景 | Full shot | 全身 |
| 远景 | Wide shot | 全环境 |

## 六、场景模板

### 模板1：一句话生成
适用：有世界知识、不需角色一致性
```
生成一个精美高级的[品牌/场景]广告，注意分镜编排
```

### 模板2：品牌广告
```
【风格】_____风格，_____秒，_____比例，_____氛围
【时间轴】
0-X秒：[镜头] + [画面] + [动作] + [特效]
X-Y秒：[镜头] + [画面] + [动作] + [特效]
【声音】_____配乐 + _____音效
【参考】@图片1 _____，@视频1 _____
```

### 模板3：人物一致性
```
@Image1中的[人物描述]，[动作描述]，[场景描述]。
镜头[运镜方式]。[情绪/表情变化]。
```

### 模板4：运镜复刻
```
参考@Image1的角色。完全参考@Video1的运镜方式。
[场景描述]。[动作描述]。
```

### 模板5：视频延长
```
延长@Video1 [X]秒。
1-5秒：[延续描述]
6-10秒：[发展描述]
```

### 模板6：视频编辑
```
颠覆@Video1的剧情——[新的剧情走向]
```

### 模板7：短剧片段
```
场景(0-5秒)：[画面描述]
对白1（角色A，情绪）："台词"
场景(6-10秒)：[画面描述]
对白2（角色B，情绪）："台词"
声音：[配乐] + [音效]
```

### 模板8：产品展示
```
静态镜头。[产品]悬浮旋转。
[产品分解/组装动作描述]
保持[产品特性]的极致美感。
```

### 模板9：家装参观
```
一镜到底跟拍，从[起点]穿过[路径]，
最终到达[终点]。全程无剪辑。
```

### 模板10：口播视频
```
[人物描述]面对镜头，[表情/动作]。
对白："[台词内容]"
背景：[场景描述]
```

### 模板11：音乐卡点
```
@Image1 @Image2 @Image3... 
参考@Video1的视觉节奏和音乐节拍进行卡点剪辑。
角色动作更有张力，整体风格更梦幻。
```

### 模板12：科普教育
```
[时长]秒健康科普短片。
0-5秒：[可视化描述1]
5-10秒：[可视化描述2]
10-15秒：[对比/总结]
整体色调[描述]。
```

### 模板13：舞蹈视频
```
让@Image1中的角色复刻@Video1的舞蹈动作和音乐节拍。
生成[X]秒视频。动作流畅无卡顿。
```

### 模板14：风景蒙太奇
```
@Image1 @Image2 @Image3... 风景图片。
参考@Video1的视觉节奏、转场风格和音乐节拍进行卡点剪辑。
```

### 模板15：特效复刻
```
将@Video1的角色替换为@Image1。@Image1作为首帧。
完全参考@Video1的转场特效和视觉风格。
[场景切换描述]
```

### 模板16：角色打斗
```
参考@Image1的角色形象。完全参考@Video1的运镜和动作编排。
[时间分段描述打斗动作]
```

## 八、多幕一致性原则

由于每一幕都是单独生成，必须在提示词中明确写入以下要素保持一致性：

### 1. 人物特征（必写）
- 发色、发型、发饰（如：粉色长发、猫耳发饰）
- 面部特征（如：大眼睛、精致五官、深邃黑色眼睛）
- 身材描述（如：年轻少女身材、25岁亚洲男性）
- **身高比例**（如：身高约175cm，与周围人物比例协调）

### 2. 服装特点（必写）
- 服装类型和颜色（如：淡蓝色明朝襦裙、深蓝色科技外套）
- 服装细节（如：交领短衫、百褶长裙、电路纹理微光）
- 配饰（如：玉佩、流苏、AR眼镜）

### 3. 场景环境（必写）
- 建筑特征（如：木质店铺、霓虹摩天楼、飞檐翘角）
- 环境元素（如：青石板路、全息广告牌、薄雾、晨光）
- 氛围色调（如：暖黄色调、冷蓝赛博朋克、水墨风）

### 4. 人物比例关系（多人场景必写）⭐
- 主角与周围人物的身高比例（如：身高与周边行人差不多高）
- 正常人体比例
- 人物之间身高比例协调一致
- 前景/中景/背景人物大小符合透视关系

### 5. 运镜方式（建议写）
- 镜头类型（如：中景、特写、航拍）
- 镜头运动（如：缓慢推进、360度环绕、俯视下降）
- 特效（如：景深、动态模糊、lens flare）

### 原则
- **只增不改**：后续修改只在原有提示词基础上增加，不随意改动
- **参考图一致**：尽量使用相同的参考图片保持风格统一
- **明确描述**：所有要素都要明确写入提示词，不能依赖AI自动推断

## 九、画质增强技巧

### 海哥研究成果（Seedance 1.0适用）

#### 1. 升级模型版本
- 从 "Seedance 1.0 mini" 切换为 "Seedance 1.0（完整版）"
- 这是开启所有增强功能的前提

#### 2. 开启智能增强
- 在编辑界面的"Agent"设置中
- 选择 "高表现力创作流"
- 勾选"启用画质链式增强"
- 让AI自动优化画质

#### 3. 设置4K直出
- 在高级参数中手动设置输出分辨率：3840×2160 (4K)
- 渲染质量调至"极致细节"
- 绕过默认的较低分辨率

#### 4. 画质提示词（重要！）
在提示词开头固定加上：
```
画质强化:4K直出+极致细节+锐利焦点
```
AI会优先提升纹理和锐度

#### 5. 双重导出（关键一步）
1. 生成预览图后，先点击 **HD按钮** 重绘
2. 导出时在"尺寸倍率"中选择 **2x**
3. 获得真正的超清素材

## 十、常见错误

### ❌ 避免的问题

1. **模糊引用** - 不要只说"参考@Video1"，要说明参考什么
   - ❌ "参考@Video1"
   - ✅ "参考@Video1的运镜和动作编排"

2. **指令冲突** - 不要同时要求相互矛盾的效果
   - ❌ "静态镜头"+"环绕拍摄"
   - ❌ "镜头固定"+"推进特写"

3. **内容过载** - 短视频不要塞太多场景
   - ❌ 5秒视频包含5个场景转换
   - ✅ 5秒视频聚焦1-2个场景

4. **忽略音频** - 音效能大幅提升效果
   - ✅ 加入BGM描述、音效描述

5. **真人脸部** - 系统会拦截真人脸素材
   - ❌ 上传真人照片
   - ✅ 使用插画/3D角色

6. **忽略比例** - 多人场景不注意身高
   - ❌ 不写人物比例，生成后主角可能是巨人
   - ✅ "主角与周围人物身高比例协调"
