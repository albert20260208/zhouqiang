# 视频制作知识库索引

当海哥要制作视频时,按此流程调用知识并给出专业建议。

---

## 🎯 核心原则

1. **每个阶段给出2-3个选项** - 让海哥选择,不替他决定
2. **说明每个选项的优劣** - 专业建议要有理由
3. **按需加载知识** - 不一次性倒出所有知识点

---

## 📋 制作流程 & 知识调用

### 阶段1: 需求理解 (必问)

**问海哥:**
```
1. 视频主题/内容是什么?
2. 目标受众? (大众/专业/特定群体)
3. 时长要求? (15秒/30秒/1分钟/更长)
4. 风格偏好? (电影感/轻松/炫酷/温馨)
5. 有参考视频吗?
6. 有对白/旁白吗?
7. 有背景音乐吗?
8. 想要哪位大师的风格? (希区柯克/库布里克/斯皮尔伯格/诺兰/王家卫/昆汀/无特定)
```

**如果选择了大师风格,立即调用:** `director-styles.md` 获取对应风格的视觉元素

**我的输出:**
- 确认理解是否正确
- 初步方向建议

---

### 阶段2: 故事结构 (创意开发)

**调用知识:** `screenwriting.md`

**给海哥的选项:**
```
方案A: 三幕结构
- 建置(25%) → 对抗(50%) → 解决(25%)
- 适合: 有明确冲突和解决的故事

方案B: 情绪曲线
- 平静 → 上升 → 高潮 → 收尾
- 适合: 氛围类、情感类视频

方案C: 并列展示
- 场景1 + 场景2 + 场景3 + 总结
- 适合: 产品展示、多角度介绍

推荐: [X] 因为 [理由]
```

**如果有对白/旁白,额外检查:**
- [ ] 对白写作技巧: 角色说话风格是否区分?
- [ ] 潜台词: 是否需要"话外之意"?
- [ ] 角色声音: 多角色时声音特征是否明确?

---

### 阶段3: 分镜设计 (场景规划)

**调用知识:** `directing.md` + `screenwriting.md`

**给海哥的选项 (每场戏):**
```
场景1: [场景名称]
├─ 情绪目标: [要传达的情感]
├─ 时长建议: [X秒]
└─ 镜头方案:
   
   方案A: 远景开场 → 中景对话 → 特写反应
   方案B: 特写开场 → 拉远揭示环境
   方案C: 一镜到底
```

**碎片化知识点检查:**
- [ ] **180度法则**: 对话场景镜头是否在同一侧?
- [ ] **分镜要素**: 每镜头包含(画面/类型/运镜/时长/备注)?

---

### 阶段4: 镜头语言 (细节设计)

**调用知识:** `directing.md` → 镜头类型 + `knowledge.md` → 镜头语言速查表

**给海哥的选项:**
```
这个画面的镜头选择:

方案A: 中景 + 缓慢推进
方案B: 特写 + 固定
方案C: 远景 + 平移

推荐: [X] 因为 [理由]
```

**碎片化知识点检查:**
- [ ] **Debate-Judge**: 镜头方案有争议时,列出正反理由让海哥裁决
- [ ] **时间分段**: 10秒以上视频是否需要分段描述(0-3秒/3-6秒/6-10秒)?
- [ ] **音乐卡点**: 有背景音乐时,重点节拍对应什么画面?

---

### 阶段5: 提示词生成

**调用知识:** `knowledge.md` → 场景模板 + 一致性控制

**给海哥的输出:**
```
第X幕提示词:

[完整提示词]

参数建议:
- 时长: X秒
- 比例: 16:9 / 9:16
- 画质: 标准 / 4K

一致性检查:
✅ 人物特征已包含
✅ 服装描述已包含
✅ 场景要素已包含
✅ 运镜方式已包含
```

**碎片化知识点检查:**
- [ ] **时间分段**: 长视频是否按时间段描述?
- [ ] **音乐卡点**: 节拍点是否对应画面变化?
- [ ] **对白同步**: 对白时机是否与画面匹配?

---

### 阶段6: 质量检查

**调用知识:** `screenwriting.md` → 质检清单 + `directing.md` → 镜头组合原则

**基础检查项:**
```
□ 镜头类型是否有变化? (避免连续相同景别)
□ 情绪曲线是否合理? (有起伏)
□ 多幕人物是否一致? (特征描述)
□ 节奏是否匹配? (与音乐/内容)
□ 开场是否吸引人? (前3秒)
□ 结尾是否有力? (收束感)
```

**碎片化知识点检查:**
```
□ 180度法则: 对话场景镜头位置是否一致?
□ 音乐卡点: 节拍与画面是否同步?
□ 时间分段: 长镜头描述是否清晰?
□ 对白节奏: 语速与画面是否匹配?
□ 角色声音: 多角色是否可区分?
```

## 💡 建议话术模板

**开场:**
> 好的,我们来做这个视频。先确认几个问题...

**给选项时:**
> 这里有几个方案,各有优劣:
> - 方案A: ... (优点/缺点)
> - 方案B: ... (优点/缺点)
> 我推荐X,因为...你觉得呢?

**确认时:**
> 明白,那我们用方案X。下一步是...

**交付时:**
> 提示词已生成,参数建议如下。要我调整什么吗?
