## 🎯 快速入门

### 制作视频流程
```
1. 构思 → 读 workflow-index.md
2. 选风格 → 读 director-styles.md
3. 写剧本 → 读 screenwriting.md
4. 设计镜头 → 读 shots-guide.md + directing.md
5. 生成参考图 → 读 seedream-5.0.md
6. 生成视频 → 读 seedance-guide.md
7. 检查 → 读 production-tips.md
```

### 核心原则速记
1. **只增不改** - 修改只增加,不删除
2. **明确描述** - 不依赖AI推断
3. **参考图一致** - 同系列用同一参考图
4. **人物比例** - 多人场景必须写身高比例
5. **16:9比例** - 图生视频用2848x1600

## 🔧 API快速参考

### Seedream 5.0 (图像)
```python
model = 'doubao-seedream-5-0-260128'
size = '2848x1600'  # 16:9适配视频
sequential_image_generation = 'auto'  # 组图
tools = [{'type': 'web_search'}]  # 联网
```

### Seedance 1.5 Pro (视频)
```python
model = 'doubao-seedance-1-5-pro-251215'
# 图生视频
content = [
    {"type": "text", "text": prompt},
    {"type": "image_url", "image_url": {"url": image_url}}
]
# 参数放在提示词末尾
# --duration 5 --camerafixed false --watermark false
```
