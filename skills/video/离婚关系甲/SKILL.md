## 🔧 技术要点

### 人物一致性控制
- **外貌特征**: 发型、妆容、服装风格连续
- **年龄感**: 30-40岁都市精英的成熟感
- **气质变化**: 职场强势 vs 情感脆弱
- **微表情**: 眼神、嘴角、手势的细微变化

### 场景一致性控制
- **办公环境**: 现代简约，冷色调
- **时间感**: 夜晚为主，霓虹灯光
- **空间布局**: 四面墙方法确保空间逻辑
- **道具细节**: 办公用品、装饰品连续

### 情感连续性
- **情绪弧线**: 从冷静到爆发再到算计
- **情感层次**: 表面理性下的情感暗流
- **关系动态**: 权力平衡的变化
- **心理距离**: 物理距离反映心理距离

## 🎬 导演风格参考

### 适合风格
1. **王家卫风格** - 情感抽帧，霓虹色彩，内心独白
2. **诺兰风格** - 复杂叙事，时间折叠，冷静算计
3. **大卫·芬奇风格** - 冷峻色调，心理悬疑，细节控
4. **是枝裕和风格** - 日常生活下的情感暗流

### 视觉关键词
- 冷色调办公室，暖色调情感时刻
- 城市夜景，玻璃反光，霓虹色彩
- 特写面部表情，眼神交流
- 慢动作情感爆发，快速剪辑对峙

### 运镜技巧
- 推拉镜头制造压迫感
- 环绕镜头展现情感复杂
- 正反打镜头对话张力
- 主观镜头情感代入

## 📝 提示词撰写要点

### 结构公式
```
【关系背景】 + 【当前冲突】 + 【情感核心】 + 【权力博弈】 + 【未来悬念】
```

### 示例模板
```
【关系背景】前夫妻，商业伙伴，未公开的婚姻
【当前冲突】女方提出复仇，男方即将再婚
【情感核心】爱恨交织，未了情感，复仇欲望
【权力博弈】商业权力，信息优势，情感操控
【未来悬念】复仇之后的关系走向
```

### 情感关键词
- 复仇、背叛、未了情、复杂情感
- 权力、控制、博弈、算计
- 理性、感性、冲突、妥协
- 过去、现在、未来时间维度

## 💼 商业元素整合

### 商业场景
- 高层办公室，董事会会议室
- 商业谈判，权力交接
- 商业晚宴，社交场合
- 私人会所，隐秘会面

### 商业术语情感化
- "并购" → 情感吞噬
- "股权" → 情感份额
- "合同" → 情感契约
- "利益" → 情感投资

### 权力象征
- 办公室大小与位置
- 服装品牌与配饰
- 座驾与出行方式
- 社交圈层与资源

## 🚫 敏感点处理

### 需要谨慎的内容
- 暴力复仇具体细节
- 违法商业操作
- 过度负面情感渲染
- 性别角色刻板印象

### 安全处理方式
- 复仇改为"正当竞争"或"法律手段"
- 情感冲突强调心理层面而非暴力
- 商业操作符合法律法规
- 性别关系平等尊重

### 价值观导向
- 强调法治精神
- 倡导理性解决问题
- 尊重个人选择
- 正能量结局导向
