# 敏感词库（审核避坑指南）

## 概述

本指南列出视频生成中需要避免的敏感词汇，提供替代方案，确保内容顺利通过平台审核。

## 敏感词分类与替代方案

### 1. 灵异/迷信类

#### 禁止词汇
- 巫师、法师、巫术、魔法
- 灵魂、鬼魂、幽灵、鬼
- 妖怪、妖精、魔物
- 诅咒、符咒、法术
- 复活、转世、轮回
- 占卜、算命、风水
- 僵尸、丧尸

#### 推荐替代词
| 原词 | 替代词 | 说明 |
|------|--------|------|
| 巫师 | 神秘黑袍法师 | 保留神秘感，去掉迷信色彩 |
| 魔法 | 能量操控、特殊能力 | 科幻化处理 |
| 鬼魂 | 光影化身、能量体 | 科技感描述 |
| 妖怪 | 异变生物、特殊物种 | 生物学术语 |
| 诅咒 | 能量印记、负面效果 | 游戏化描述 |
| 复活 | 意识恢复、身体修复 | 科技解释 |
| 算命 | 数据分析、趋势预测 | 科学化处理 |
| 僵尸 | 行动迟缓的异变体 | 去掉恐怖元素 |

#### 示例对比
- ❌ "黑暗巫师施展复活魔法"
- ✅ "神秘黑袍法师使用能量修复技术"

### 2. 黑暗/负面类

#### 禁止词汇
- 黑暗宝玉、邪恶宝石
- 邪气、魔气、杀气
- 血腥、残暴、虐杀
- 恐怖、惊悚、骇人
- 死亡、杀戮、屠杀
- 地狱、恶魔、魔鬼

#### 推荐替代词
| 原词 | 替代词 | 说明 |
|------|--------|------|
| 黑暗宝玉 | 神秘宝玉、深色能量石 | 去掉负面形容词 |
| 邪气 | 暗色能量、特殊气场 | 中性描述 |
| 血腥 | 红色液体、战斗痕迹 | 淡化暴力 |
| 恐怖 | 紧张、悬疑、刺激 | 情绪调整 |
| 死亡 | 失去意识、退出战斗 | 游戏化处理 |
| 地狱 | 异次元空间、特殊领域 | 科幻化描述 |

#### 示例对比
- ❌ "血腥的战斗场面"
- ✅ "激烈的战斗，能量碰撞产生光效"

### 3. 封建/迷信类

#### 禁止词汇
- 皇帝、皇后、太子（历史题材需谨慎）
- 修炼、修仙、修真
- 门派、宗派、教派
- 神佛、菩萨、神仙
- 祭祀、祭拜、供奉

#### 推荐替代词
| 原词 | 替代词 | 说明 |
|------|--------|------|
| 修炼 | 训练、修行、提升 | 去掉迷信色彩 |
| 修仙 | 能量修炼、能力开发 | 科幻化处理 |
| 门派 | 组织、团体、学院 | 现代术语 |
| 神仙 | 高级生命体、能量存在 | 科幻描述 |
| 祭祀 | 纪念仪式、传统活动 | 文化性描述 |

#### 示例对比
- ❌ "修仙者修炼法术"
- ✅ "能力者在进行能量训练"

### 4. 暴力/武器类

#### 需要谨慎处理的词汇
- 枪械、武器具体型号
- 爆炸、爆破具体描述
- 伤害、攻击具体方式
- 战争、战斗血腥细节

#### 安全处理方式
1. **模糊化处理**
   - ❌ "AK-47步枪扫射"
   - ✅ "能量武器连续射击"

2. **艺术化表现**
   - ❌ "鲜血喷溅"
   - ✅ "光效碰撞，能量扩散"

3. **象征性表达**
   - ❌ "残忍的杀戮"
   - ✅ "激烈的对决，胜负分晓"

### 5. 政治/历史敏感类

#### 绝对避免
- 现实政治人物、事件
- 敏感历史时期、事件
- 民族、宗教敏感话题
- 领土、主权相关表述

#### 处理原则
1. **虚构化** - 使用虚构国家、时代
2. **模糊化** - 不涉及具体细节
3. **正能量** - 强调团结、和平、发展

### 6. 社会负面类

#### 需要调整的词汇
- 犯罪、违法具体行为
- 负面社会现象
- 歧视性语言
- 不道德行为

#### 正向引导
- 强调正义战胜邪恶
- 展现积极向上的价值观
- 体现人文关怀
- 传播正能量

## 审核原则

### 1. 核心需求保留原则
在保留核心创意的前提下进行修改：
- **核心**: 冲突、对决、成长
- **形式**: 调整表现方式，不改变本质

### 2. 意象表达原则
用意象代替直白描述：
- 暴力 → 力量对决
- 恐怖 → 紧张悬疑
- 血腥 → 能量碰撞

### 3. 科技奇幻化原则
将迷信、玄幻转化为科幻：
- 魔法 → 科技/超能力
- 鬼怪 → 外星生物/能量体
- 修炼 → 训练/进化

### 4. 文化适应性原则
考虑不同文化背景：
- 避免文化冒犯
- 尊重多元价值观
- 体现文化包容性

## 具体场景处理示例

### 场景1：武侠打斗
**原描述**:
```
黑衣杀手施展血影刀法，刀光带血，杀气腾腾。
一刀斩下，对手鲜血喷溅，倒地身亡。
```

**安全版本**:
```
神秘黑衣人施展影刃刀法，刀光如影，气势逼人。
一招制敌，对手被击退，退出战斗。
能量碰撞产生红色光效，象征激烈对决。
```

### 场景2：魔法战斗
**原描述**:
```
黑暗巫师召唤地狱火焰，焚烧一切。
灵魂哀嚎，恐怖景象。
```

**安全版本**:
```
神秘法师操控暗色能量，形成高温火焰。
能量场剧烈波动，产生视觉冲击效果。
光影变化象征力量对抗。
```

### 场景3：恐怖场景
**原描述**:
```
古宅中鬼魂出没，阴森恐怖。
突然出现血腥鬼脸，吓人一跳。
```

**安全版本**:
```
古老宅院中光影变幻，营造神秘氛围。
突然的光影效果产生惊喜感。
整体保持悬疑风格，避免恐怖元素。
```

- 考虑国际观众

## 常见问题解答

### Q1: 如何平衡创意与安全？
**A**: 核心创意保留，表现形式调整。用意象、象征、科技化手段表达。

### Q2: 历史题材如何处理？
**A**: 虚构时代背景，避免具体历史事件，强调普世价值。

### Q3: 动作场面如何避免暴力？
**A**: 强调技巧对决而非伤害，用光效代替血腥，展现武术美感。

### Q4: 恐怖元素如何转化？
**A**: 转化为悬疑、神秘、探险元素，用心理紧张代替视觉恐怖。

### Q5: 如何应对不同平台要求？
**A**: 建立平台专用词库，针对不同平台调整内容强度。

## 总结

敏感词处理不是限制创意，而是让创意以更安全、更可持续的方式呈现。通过：

1. **了解规则** - 掌握各平台审核标准
2. **预先规避** - 创作时即考虑安全性
3. **巧妙转化** - 用安全方式表达核心创意
4. **持续学习** - 跟进规则变化，积累经验

可以创作出既安全又有创意的优秀作品。
