```

**核心规则：**
1. ✅ 5层递进：要求→资产→镜头参数→站位剧情→分镜
2. ✅ 技术参数每段必写：焦段+光圈+ISO+快门角度+ND
3. ✅ 景别带英文缩写：WS/MS/CU/ECU/MCU/LS
4. ✅ 角度可带英文：Low Angle/POV/OTS
5. ✅ 运镜密度根据时长灵活配比：3秒镜≤2-3种快速过渡，5秒镜≤3-4种，7秒+镜可4种以上
5.1 搭配底线：不叠两种冲击型（whip pan+crash zoom），不叠两种过程型（缓推+缓拉）
6. ✅ 每镜运镜写过渡链，不写一种到底：手摇→突停→crash zoom / 跟拍→甩镜→跟拍
7. ✅ 光效每镜独立标注
7.1 🔴 **光效写法铁律（2026-06-26 导演确认）**：
- ❌ 禁止空词：不准写「火焰光效」「冲击波光效」「能量光效」等概括性空词
- ✅ 写光与物质的交互：`尘埃粒子被阳光穿透形成体积光柱`
- ✅ 写光的二次效应：`火焰剑光芒在地面投下摇曳橙红光斑`
- ✅ 写光的多源叠加：`阳光从秦身后打来形成金色轮廓光，火焰剑光与阳光在秦身上交织`
- ✅ 写光的时序变化：`轮廓光边缘火焰剑光微微脉动，地面尘埃缓缓沉降，空气中残留热浪扭曲`
7.2 🔴 **内容必须按机位分段**：逐拍标注机位标签 — `[crash zoom急速推进]` `[甩镜whip pan横切]` `[特写CU]` `[中景MS]`，每个机位一个动作块，读完能直接出画面。
8. ✅ 台词用反引号
9. ✅ 声音归尾部统一
10. ✅ 灵活运用，规则是底线不是公式
11. 🔴 **角色质感尾缀（2026-06-26 导演指定·永久必加）**：每个【人物】描述末尾必须追加角色质感尾缀（完整文本见下方§角色质感尾缀）。缺尾缀直接出稿→违规。

**用途**：每个【人物】描述末尾必须追加以下尾缀，解决 AI 视频生成中表情呆滞、眼神死板、肤质塑料感问题。

```
服装面料细节拉满，褶皱丰富，层次分明，布料肌理清晰，丝绸高光通透，薄纱半透质感，色彩冷暖层次丰富，光影塑造立体感，真人写实质感，表情淡然嘴角若有若无微扬——重压之下举重若轻的从容。真人写实，眼神灵动，缓慢自然眨眼，眼球轻微转动，面部带有细微微表情，眉部、嘴角有微弱肌肉起伏；皮肤毛孔纹理全程保留，肤质前后帧保持一致，不会自动磨皮；面部固定单侧柔光，光影稳定不跳动；五官立体，眼神有神，杜绝木偶脸，仅保留微小生理动态，脸部不发生大幅度形变，肤质统一

**使用规则**：

### 🔴 提示词末尾 avoid 约束行（2026-06-26 导演指定·配合尾缀使用）

**用途**：Seedance 2.0 不支持原生 Negative Prompt，用正向 `avoid` 约束等效排除常见问题。

**每段提示词末尾（【声音】之后）必加一行**：
```
避免面部特征漂移，避免形态变形，避免动作僵硬不自然
```

**约束释义**：
| 约束词 | 排除内容 |
|--------|----------|
| `避免面部特征漂移` | 帧间脸变样/五官移位 |
| `避免形态变形` | 面部/身体扭曲拉伸 |
| `避免动作僵硬不自然` | 机器人感/木偶式运动 |

**使用规则**：
- ✅ 放在【声音】字段之后，作为提示词最后一行
- ✅ 与角色质感尾缀配合使用（一个正向锁定、一个负向排错）
- ❌ 不前置、不加权重词修饰，末尾裸写即可

### 🔴 强制执行机制（每次生成/优化提示词时）

**触发条件**：用户说"生成提示词"、"优化提示词"、"写提示词"、或类似表达

**前置准备（新）**：
0. **读知识库**：先通读 MEMORY.md 索引和相关条目，确保不遗漏任何已有规则和经验
1. **读规范文档**：读 `docs/knowledge-management-spec.md` 确认知识库状态

**执行步骤**：
2. **先问参考图**：必须先问哪些角色/场景/道具有参考图。有参考图→不写外貌描述，只写位置和动作状态。不问就写→违规
3. **必须先读** `skills/prompt-optimizer/SKILL.md`（含AI理解顺序、180度轴线、OTS肩部占比、18项检查清单、10步执行流程）
3.5. **🔴 技术参数强制输出（2026-06-25 导演要求·每次必用）**：
    - 读 `skills/prompt-optimizer/references/technical-parameters.md`
    - 每段提示词必须写入：**光圈F值 + ISO + 快门角度 + ND（如需）**
    - 每镜景别必须带英文缩写：`全景WS / 中景MS / 近景CU / 特写ECU / 中近景MCU`
    - 每镜角度可带英文标注：`低角度仰拍Low Angle / POV主观视角 / OTS过肩`
    - 不问不查、不写技术参数直接出稿→违规
3.6. **运镜数硬限+搭配底线（2026-06-25 导演+跑片验证）**：
    - 3秒/镜 = 最多2种运镜（不硬凑，内容够了1种即可）
    - 5秒/镜 = 最多3种运镜
    - 7秒+/镜 = 4种以上
    - 超过硬限直接砍
    - 🔴 搭配底线：不叠两种冲击型（whip pan+crash zoom）、不叠两种过程型（缓推+缓拉）
    - ⚠️ 灵活运用：规则是底线不是公式，每镜根据戏本身判断。内容够了1种即可，不硬凑
4. **按10步流程执行**：阅读剧本→15秒分段→选择镜头策略→计算时长→分配镜头→按权重写内容→检查轴线(含OTS)→检查物理锚点→检查一致性→去框架化→过18项检查
5. **输出前检查**：18项检查清单逐条通过（含OTS专项），才能输出

**为什么这样规定**：
- 新会话启动时上下文会被压缩，之前的"记忆"可能丢失
- 只有把规则写在 SOUL.md 里，才能确保每次都执行
- skill 文件是工具，但规则必须锚定在灵魂文件（SOUL.md）

**🔴 段头段尾铁律（多次强调·反复违规·最高优先级）**：
- **段头不铺垫**：不上来先看传送/降临/光团炸开。铠兵已在画面里，秦已经在动。第一帧就是动作进行中。
- **段尾不收招**：不站定不摆pose不定帧。砍完最后一击顺势进入下一状态，敌人倒地碎片飞溅中秦已在移动。

**附加检查项（2026-04-27 周强补充）**：
4. **持剑手检查**：每个持剑/持物镜头必须标注左右手，不可遗漏
5. **站位关系检查**：画面有两人+时，内容描述必须包含相对位置（左/右/前/后/距离）
6. **剧本上下文检查**：每个分镜写完回读剧本原文，确认情绪/关系/状态三者与剧本对齐

**附加检查项（2026-04-28 郑有朋+阿飞补充）**：
7. **OTS肩部占比检查**：过肩镜头必须明确描述前景肩膀占画面比例（左下/右下约20%）
8. **同景别不切换检查**：同一景别、同一个人物、无动作变化时禁止切换镜头
9. **15秒独立输出检查**：按STEP 0分段，每段一个独立提示词不超过15秒，标注1/N
10. **去框架化检查**：丢模型的描述中移除所有框架性文字（全局规则/运镜规则等）

**附加检查项（2026-04-28 郑有朋补充·第二轮）**：
11. **核心约束前置检查**：禁止性要求（不要字幕/不要变形等）必须放在提示词最开头，权重最高
12. **审核规避检查**：
    - 提示词中用身份称呼替代具体人名（男主/男二/大师兄，不用秦轩/林莫）
    - 遇歧义词替换为中性词（"歪脖子树"→"老树""古树"）
    - "真人""写实"等词不超过1次
13. **对话切镜检查**：一段台词超过5秒（15-30字）必须拆成多镜，每镜穿插画面描述
    - 注意：此规则仅适用给郑有朋写提示词时，阿飞的规则相反（不拆镜）
14. **不硬凑15秒检查**：写完每段反问自己——这段的台词+核心动作实际需要多少秒？信息量不够就不撑满，内容收尾就停。10秒/8秒/5秒都可以，不需要补镜头凑15秒
15. **单镜≤5秒硬限检查**：写完每个镜头检查秒数，超过5秒直接砍重排，不能出现6秒/7秒的镜头
16. **画左/画右坐标检查**（2026-05-12 周强示范）：
    - 内容描述中，用**画左/画右**替代世界坐标（餐车方向/主位方向/洛尘方向等）。模型不懂空间名词，只认画面内坐标
    - 角度字段加**精确度数**（`正面偏左10°`，不是只写`正面偏左`）
    - 角度带**仰角/俯角**（`正面微微仰拍`，不是只写`正面`）
    - 景别写**裁切部位**（`特写月帝肩膀以上`/`特写洛尘胸部以上景`，不是只写`近景`）
    - 【人物】字段一句话搞定，括号内写位置+朝向：`月帝（左边蓝色椅子上坐着.正面面对镜头）`
    - 人物名带图编号：`洛尘4`
    - 【道具】不出镜就空着不写
    - OTS构图在**角度**字段直接写肩膀占画比例：`正面偏左10°（沈婉悦@右肩膀出现在画右占10%）`
17. **不出镜不写入检查**：每段【人物】【道具】只列该段镜头实际出现的内容，全局角色表不带到每一段
18. **连续台词合并检查**：同一人同一景别连续台词+动作→合并为一镜不拆（参考21:22导演示范：尝当然可以+刚才+听你们喊谁牛逼，两句台词+两个动作一镜装完，不拆两镜）
19. **场景段约束写法**：可在【场景】中直接写段落约束，如`【场景】末世拍卖厅主位区 【注意群众只在镜头3出现.镜头一和二不要出现群众身影】`

**附加检查项（2026-05-13 周强补充·秒级三查）**：

**查一：每镜台词字数÷时间 合法性检查**
```
每镜完成时：统计台词字数 ÷ 分配秒数

第一步：判断语速类型（情绪决定速度，不是公式决定）
- 情绪爆发/吼叫/崩溃/兴奋 → 4-5字/秒（情绪压着字往外爆，5秒比公式算的还够用）
- 哽咽/回忆/虚弱/重伤/哭泣 → 2-3字/秒（需要换气哽咽停顿）
- 正常对话/叙述 → 3-4字/秒
- 庄严/老人/沉重 → 1.5-2.5字/秒（字间停顿多）

第二步：按语速类型检查
字数 ÷ 语速类 = 合理所需秒数
实际分配秒数 < 合理所需秒数 → 时间不够，加秒
实际分配秒数 > 合理所需秒数 + 2秒 → 太松，压缩

第三步：超5秒的处理（不是硬砍字数，也不是暴力拆为两镜）
先判断：这个角色的台词能不能一气呵成？
- 能（情绪爆发/单人长篇独白/一镜贯穿更有冲击力）→ 中间插反应镜
  -- 台词到一半切观众/对方/环境反应镜(1-1.5秒)，再切回来继续
  -- 说话人的情绪和节奏不断裂
- 不能（台词本身就有明显分段/动作中断）→ 合理分成两镜

⚠️ 核心原则：情绪连贯 > 单镜时长硬限
演员一气呵成的崩溃/爆发/哭戏，拆镜反而破坏情绪，宁可在6-7秒内完整呈现
但5秒内能说清的情绪，不要为了信息量硬拉长
```

**查二：段尾段头 连续性检查**
```
写完一段后，读本段最后1-2镜 + 下段前1-2镜
条件一：同一人、同一景、同一场景 → 应合并，重划段边界
条件二：动作/情绪有直接因果链 → 应合并或加过渡镜
条件三：同一句台词被切断跨段 → 必须合并
通过标准：段尾段头之间如果黑屏/切段，观众不会觉得断，合情合理 → 过
        觉得断、觉得莫名其妙、情绪断了 → 必须合并
```

**查三：跨段分割前自问**
```
这段最后3秒的内容，和下段开头3秒的内容
如果中间没有切段（没有黑屏/字幕/场景切换）
观众看完会感觉：
- "啊就是同一场戏" → 应合并，重划段边界
- "这里断了但合理"（时间跳跃/场景切换）→ 过
- "怎么断了？" → 必须合并
```

**查四：单人连续帧累计≤5秒检查**
```
写完一段所有分镜后，按顺序逐镜扫描——同一人物出现在连续N帧里，累计秒数是否≥5秒？
- 是 → 中间必须插反应镜/环境镜/道具镜隔开
- 单人镜之间有其他人/道具/环境镜隔开 → 重置计数
- 双人中景/全景多人 → 不触发
- 不是只看单镜≤5秒，要看前后帧累计
```

**查零：景别预检——单人说话必须是「纯」单人近景（在查一之前做，不是之后）**
```
写每一组分镜前，先问自己：
这段有几个单人说话/对话的镜头？
这些单人说话镜头用的景别是什么？
画面里还有没有其他人？

硬性规定：
- 单人说话 → 一律用近景或特写
- 「单人近景」= 画面里只有一个人。对面的人不能入画，环境背景不交代，观众视线零分散
- 中近景/中景 = 带入环境背景甚至其他人，分散观众焦点，不准用

- 写每镜景别前强制停1秒：这个镜头观众需要看什么？

  如果是「这个人在环境里/和别人的互动」→ 才用中近景/中景
```

### ⚠️ 写分镜前停一秒（2026-06-16 强化）
**第一停：每写完一个镜头，回看景别字段。这个镜头里有人在说话吗？只有一个人吗？→ 是 → 景别必须是「近景」或「特写」。不准出现「中景」「中近景」「中景→近景」等任何含中景的写法。**

### 🔴 画面里有啥写啥·铁律（2026-06-23 强化·反复犯错点）
**每个镜头的【内容】只能写在当前角度+景别下画面内真实可见的东西。**
- 正面拍一个人 → 画面里就这一个人。不写他背后的暗处、画外的对方、远处的环境。

- 镜头没拍到的 = 画面外，不写进内容。

### 🔴 人脸交代检查·铁律（2026-06-24 导演纠正·全局规则）
**模型不知道"剪影=脸没拍"——它看到整段没交代正脸，就会在动作结束后自己补一个转身正脸，画面崩塌。模型填空的默认值=正脸对镜头。**

每镜写完角度+内容后，必须确认：
1. **角度字段必须同时写两样**：机位位置 + **脸与镜头的关系**（侧脸30°对镜头/四分之三侧脸/正脸/背面/闭眼咬牙）
2. **不给模型填空权**：只要角度字段没写脸朝哪，模型就默认朝镜头
3. **镜头拍不到脸 → 不写表情，不写光打在脸上**：背面/剪影/被遮挡时，只写身体动作，脸不提
4. **侧脸可见 → 可写光打在侧脸**：侧脸轮廓、下颌线、脸颊光影、咬肌绷紧等侧面可见的细节，不写正面才有的（如眉间/鼻翼/双眼同时）
5. **全段扫描**：一段15秒内，关键发力/情绪节点至少有一镜明确拍到秦的正脸（正脸表情可写眉间/鼻翼/双眼）
6. **不留空白**：不说"脸被挡住"但不写被什么挡住，不说"看不清脸"但不写为什么看不清

**每写完一段自问**：这段里模型会不会觉得"秦的脸还没拍完"，然后自己补？→ 会 → 在对应镜头补正脸交代。

### 🔴 落笔前空间逻辑推演·铁律（2026-06-24 导演纠正·全局规则）

**顺序不能反。先建空间关系 → 再选摄影机位置 → 最后写画面内容。**

**Step 0：写任何镜头前，先锁定三个空间事实**
```
事实1：秦在空间中的位置+朝向（画左？画右？面朝画内哪个方向？）
事实2：攻击者/对方在空间中的位置+方向（从哪来？在秦的什么方向？）
事实3：摄影机在轴线的哪一侧（左or右？第几象限？）
```

**Step 1：每镜落笔前，三步推演**
```
① 景别是什么？→ 能写什么粒度不能写什么？（全景=身体轮廓/位移，中全=全身动作+脸方向，近景=表情/眼神，特写=肌肉微动）
② 角度从哪拍？→ 摄影机在空间哪个位置？脸与镜头的关系？（侧脸/四分之三侧脸/正脸/背面/剪影）
③ 内容写什么？→ 只写这个景别+角度下画面里真实可见的东西（画面里有啥写啥）
```

**Step 2："正面"永远不能单独出现——必须标谁的正面**
```
❌ "正面微仰" → 谁的正面？空间关系崩
✅ "从秦右前方仰拍，秦四分之三侧脸对镜头，身体朝向画右大猩猩方向"
```

**铁律**：打斗场景中没有统一的"正面"。两人对峙时，"秦的正面"="背对大猩猩"。如果秦正脸对镜头，必须同时确认：他现在面对的是谁？在打斗中这合理吗？

**Step 3：推进式自查（三连问）**
```
第一问：这个镜头角度下，空间关系对吗？（秦面朝右打大猩猩→不能同时正脸对镜头）
第二问：这个景别下，我能写这个细节吗？（全景不写牙关/目光/下颌微绷）
第三问：画面里实际能看到什么？（中全=全身+动作+脸方向，不写微表情）
```

**内置参考：景别可见粒度速查表**
| 景别 | 能写 | 不能写 |
|------|------|--------|
| 远景/全景 | 身体轮廓、移动轨迹、空间关系、光影方向 | 表情、眼神、牙关、嘴唇、面部肌肉 |
| 中全 | 全身动作、脸的大致朝向、基本姿态、衣服质感 | 面部肌肉细节、眼珠方向、嘴唇微动 |
| 中景 | 腰部以上、手势、脸上大致表情（笑/怒/惊） | 眼球血丝、汗珠、皮肤纹理 |
| 近景 | 胸部以上、表情细节、眼神方向、嘴型 | 远处环境、多人站位、大场面 |
| 特写 | 面部肌肉、眼部微动、单颗汗珠、皮肤纹理 | 环境全貌、空间关系、其他人 |

### 🔴 创作心法（2026-06-09 周强纠正·永久规则）

**不要「糊弄完成任务」，要「带着感觉拍片」。**

当导演提到任何电影/风格/氛围/场景的参考（如"007镜头风格""铁刀""打斗一镜到底"等），不只是记下信息，而是：

## 🔴 2026-06-08 打斗/镜头/台词全维度规则（永久写入）

**以下规则写入了skills/prompt-optimizer/SKILL.md检查清单，每次写分镜必须全部通过才能输出。**

### 打斗因果链
每下攻击写清：A怎么出招（方向/武器/发力）→ B怎么应对（命中/格挡/闪躲三选一写死）
- 命中：打中哪里、B什么反应（后退/踉跄/出血/姿态变形）
- 格挡：用什么格挡、交接点在武器哪个部位、材质反馈（金属碰金属=火花，武器碰骨甲=沉闷震动）
- 闪躲：往哪闪、闪后位置姿态变化

### 打斗动作连贯性
- 发力终点=下一动作发力起点（顺劈到底→借腰拧接横扫）
- 武器轨迹连续弧线不断开
- 写力量感（重心/肩/腰/胯）+ 速度（迅捷不机械）

### 打斗动力链写法 ⚠️ 核心
- ❌ 不写结果：秦牧挥刀砍去 / B被震退
- ✅ 写力量来源：秦牧双脚蹬地，腰部发力带动右肩，手臂送刀从右上斜劈而下
- 完整动力链：脚蹬地→转腰→送肩→臂发力→刀锋落点

### 兵器咬合感
- 交锋不是轻触即分，写清撞击后的角力状态
- 物理反馈：刀身震颤、手臂肌肉绷紧、怪被震退半步/硬顶住

### 攻防节奏
- 不对称设计：我方连续施压，怪被迫应对→找机会反击
- 节奏变化：密集快打中插入重击（慢蓄力→突然爆发），形成对比

### 打斗镜头节奏
- 镜头跟着武器走，动作不断不切镜
- 爆发点（命中/格挡）切1-2秒特写
- 景别不局限于特/近/中/全四档，多用OTS/侧面/斜45°/仰拍/俯拍
- 开场站位：不用大全景俯拍摄数，用横移/后退推/过肩侧拍替代
- **打斗一镜多节拍**：攻→防→转攻→攻→反→攻→收，每秒推进

### 打斗vs叙事镜头差异
- 叙事：多镜头交代时间地点人物
- 打斗：少镜头塞多节拍，一镜到底连杀多只

### 光影每镜单独写
每镜自问：光源从哪来？打谁脸上？什么色温？和环境的明暗关系？
- 光影服务于人物状态和气氛（战斗火光/暗处剪影/谈话柔光侧光/紧张强光切割）
- 每镜写完反问：光影写了吗？没过不算写完

### 台词带情绪说话方式
- 不准写「角色名说：台词」
- 必须写「情绪+状态修饰词+说」（压低声急促说/咬牙嗤笑/喘息道/激动喊道/高冷不抬眼/低沉讥讽等）
- 情绪和状态必须与当前场景气压匹配

### 输出前问导演
- 先问「出全部还是先出一段看看方向」，导演说「发」才能发

## 🔴 2026-06-15 血啸九域生产教训（周强现场纠正·永久记忆）

### OTS默认用近景 ⚠️
- **过肩镜头一律默认近景**，不是中景
- OTS拍对面说话的人，焦点在表情，中景太松、人物小
- 只有导演指定中景时才用中景

### OTS左右不搞反
- 背对镜头时：人物左肩 = 画面右侧，人物右肩 = 画面左侧
- 拍背面人物过肩时先停一秒确认方向

### 站位一遍写清
- 人物字段直接用画面坐标：「画左前方」「画右前方」
- 不写相对词（「秦牧左手边」），写完就定，不来回改

### 删掉的信息不回流通
- 导演让删的内容（如某特征某描述），后面版本不准再出现

### 人物字段写过的分镜不重复
- 刀/服装/外貌等信息【人物】字段提过，分镜内不再重复描述

### 台词必须带情绪状态 ⚠️
- ❌ 不准写「秦牧说」「秦牧自语」
- ✅ 「低声自语」「轻笑一声道」「急促喊道」「带着哭腔慌张喊道」

### 落笔前走三遍法
1. 当观众感受气场 → 2. 当导演选镜头 → 3. 当执行者落笔
- 不跳步骤，不让导演一句一句纠正

### 不硬凑15秒
- 信息量到了就收，10/12/14秒都行，不灌水

### 每镜每人面部三部位铁律 ⚠️
- **每镜里每个入画的人**必须写至少三个面部部位的具体状态
- 格式：眼睛状态 + 嘴巴状态 + 脸部肌肉状态
- ❌ 「沉默看着」「站在旁边」→ 模型给蜡像脸
- ✅ 「嘴角微翘带浅笑，眼睛好奇打量，脸部肌肉自然放松」
- 不说话的人更需要面部描写，模型不会自动"沉默但生动"

### 没让发不准发 ⚠️

- **动作**：放大写，力量感砸实（重心下沉/刀锋轰鸣/地面震裂/迅猛/快速）
- **表情**：压着写，微扬不爆（嘴角微扬/眼中一闪/眉间一紧）

## 🔴 2026-06-18 批量犯错·刻入永久禁止（周强纠正）

以下错误今天反复犯，每条都是之前说过的。以后写分镜前先过这份清单。

### 1. 景别→能见度铁律 ⚠️ 不准再犯
| 景别 | 能写什么 | 不能写什么 |
|------|---------|----------|
| 特写 | 眼睛/嘴角/皱纹/汗珠 | 环境/远处/多人 |
| 近景 | 眼神方向/嘴角/面色 | **瞳孔**/远处/环境全貌 |
| 中景 | 大体表情/身姿/动作 | **瞳孔**/细微面部 |
| 全景 | 身形/站位/动作 | **任何面部细节** |
| 远景/俯拍 | 轮廓/剪影/群体 | **任何面部细节** |

**写每镜前必须停一秒：这个景别能看见什么？写完反问：我写的内容全在画面里吗？**

### 2. 脸看不见不写面部 ⚠️
- ❌ 背影/逆光剪影/俯拍/远景写眼睛嘴巴脸部肌肉
- ✅ 脸在画面里且光线照到 → 才写表情
- ✅ 脸看不见 → 只写身形/动作/站位

### 2B. 这些镜头类型默认拍不到脸 ⚠️
- 虫视/微距（极低角度地面仰拍）→ 镜头距地面一掌高，人脸在远处，拍不到任何表情
- FPV（第一人称视角）→ 写画面里看到的东西，不写自己的脸
- 远景/全景/俯拍 → 按§1表执行
- 背面/侧后方跟拍 → 看不到脸
- 高速追拍/环绕 → 脸一闪而过，不硬塞表情
- 写每镜前先停一秒：这个角度能拍到脸吗？→ 不能 → 跳过表情

### 3. 不写装饰性环境光 ⚠️
- ❌ 磷火、极光、光柱、发光植物等场景装饰光源
- ✅ 只写：时间自然光 + 技能自带光效 → 打在谁脸上 → 明暗关系

### 4. 台词铁律 ⚠️
- ❌ 裸台词丢反引号：`还不错，换上试试。`
- ✅ 谁说 + 情绪语气 + 台词：`秦无奈道：\`还不错，换上试试。\``
- 谁说、情绪、台词三样缺一不可

### 5. 没让发不准发 ⚠️
- 导演说「出」「发」「来吧」→ 才能输出完整提示词
- 导演说「说完再发」「先讨论」→ 只讨论，不输出全段
- 导演让我改 → 只回改的那句，不发全篇，不发自动续

### 6. 时间前后统一 ⚠️
- 同一场景所有段用同一个时间词
- 不自己换时间（段9用「高级地图环境光」→后面不准自己改成「白天」「昏暗压抑」）

### 7. 不套镜头模板 ⚠️
- ❌ 登场→冲撞→半月→俯拍→特写→开天→爆装 循环
- ✅ 每段根据内容选镜头，不是凑五种角度

### 8. 人物名统一 ⚠️
- 导演说用「秦」就全用「秦」，不来回改
- 导演说去掉编号就全部去掉

### 9. 不硬塞技能名 ⚠️
- 剑招是剑招，技能是技能，不混
- 剑舞段就是剑舞，不塞烈火/开天/冲撞

### 10. 写完过脑子 ⚠️
- 每镜写完自问：景别能看到这个吗？时间对吗？场景对吗？
- 不发不经大脑的稿

### 11. 装备掉落描写铁律 ⚠️
- ❌ 只写「光柱」「光团」「金/紫/蓝/赤光柱」不写装备内容
- ❌ 写「光柱从地面炸开」—— 装备是自己发光的，不是光柱
- ✅ 每件装备具体描写：武器/靴子/手镯/项链/戒指/衣服/头盔/腰带 等游戏装备
- ✅ 写清楚每件装备的外观特征和发光颜色
- ✅ 装备「悬浮在空中」「通体泛X光」「缓缓旋转」「表面刻着/流动X纹路」
- ✅ 消失时「光芒收敛」「化作光点散去」
- 格式示例：`一把长剑悬浮在金色光芒中，剑身流动暗金纹路。一双靴子在紫色光中缓缓旋转，靴面泛着幽紫光泽。`

### 12. 没让发不准发 ⚠️ 反复重申
- 导演说「别发了」→ 立刻停，不发任何完整版
- 导演让我改一句 → 只回那一句，不发全段
- 改完等导演确认，不自动补发

### 13. 跨段动作衔接铁律 ⚠️ 所有项目通用
- 连续段落之间，下一段开头动作必须承接上一段结尾
- 写下一段前，先回看上一段最后1-2镜：人物在什么位置、做什么动作、什么状态
- 下一段镜头1直接从这个状态开始，不另起一页、不跳位置、不跳时间
- ❌ 上段人物在跑 → 下段直接站在新场景
- ✅ 上段人物在跑 → 下段人物跑到某处停下/继续跑/喘息回望
- 每次写新段第一镜前停一秒自问：上一段结尾是什么？这一镜从这里接了吗？
- 此规则适用于所有项目所有连续段落

### 14. 15秒不非得杀怪 ⚠️ 所有项目通用
- 15秒段可以是战斗中的一个切片，不一定要怪死才能收
- 可以打到一半、对峙中、互相试探、怪还在动——下一段接着打
- 不要每段都设计成「起→打→杀→爆装」四步收尾
- 打斗节奏有起伏：可快攻、可缠斗、可闪避、可被打退、可撤退
- ✅ 怪物难度梯度：越往后的怪越硬越难打，不能每一波都碾压一剑清一排
- ✅ 前面小怪可以剑气扫一片，后面硬怪要缠斗、被震退、找破绽、换策略
- ✅ 战斗应有挫折感：秦被击退、硬直、喘气、换打法——才有看头
- 写每段前先问：这一段的戏剧功能是什么？（建立威胁/展示机动/被打退/反击/收尾）
- 不是每段都要当最后一刀
- ⚠️ 核心心态：写打斗段时不想着「这段怎么收」，想「下一个15秒从这段尾怎么接」。15秒只是切片，不是小故事。

### 15. 打斗动作不僵硬的写法 ⚠️ 所有项目通用
- ❌ 一拳接一拳（Blow by Blow）：每一招都细写=慢动作，读者累
- ❌ 每个动作都等量描写：挥剑→砍中→火星→震退→再挥剑——这节奏是死的
- ✅ 打斗分级写：不重要的小招用一句带过，关键时刻放大写
- ✅ 每招必须推进：位置变了/怪受伤了/我被震退了/优势易手了——打完一招不能什么都没变
- ✅ 快慢交替：密集快打（三五招浓缩一句话）→ 顿一拍（特写/眼神/喘息）→ 再打
- ✅ 动作写力量来源不写结果：「腰部发力带动右肩，剑从右上斜劈而下」 vs 「一剑劈下」
- ✅ 物理反馈写到位：兵器撞击→手麻/手臂震/剑身颤抖/火星溅射方向
- ✅ 挥砍必带剑气：每次挥剑都带剑气——剑划过的轨迹留下可见能量残留（淡色弧光/空气扭曲/微光拖尾），这是剑的视觉常驻效果，不是只有重招才开
- ✅ 剑气延伸攻击距离：物理剑身之外有能量延伸段——1米剑能砍到1.5-2米外的目标，剑气先到，剑身随后
- ✅ 剑气强度随力度变化：普通挥砍=轻淡弧线一闪、加速重劈=宽幅气刃拖尾、全力爆发=剑气冲出剑身带冲击波
- ✅ 剑气长度不限物理剑身：重招剑气可以拉得很长——一道数米长的弧形能量刃横扫而出，视觉冲击力极强
- ✅ 能量巨剑：爆发时剑身火焰爆燃→剑体肉眼可见地急速膨胀变大至数丈高→巨大化的剑从上方劈下砸下。物理剑和能量是一体的，不是分开两层。描述锚定：剑身膨胀变大至X丈→巨大化剑劈落。不是"能量影叠加"（2026-06-24 导演纠正）
- ✅ 切镜在爆发点：不是在动作做完换招时切，是在拳头打中的瞬间切
- ✅ 每招必有反应：秦冲上去→怪如何反应（后撤/挥爪迎击/硬扛/被震退），不能只写秦冲上去就收
- ✅ 秦出招→怪中招后的状态：踉跄/硬直/甲壳裂开/臂被震开/身形一歪/怒吼——必须写一个

### 15L. 🔴 打斗段落镜头设计总纲 ⚠️ 所有项目通用（2026-06-27 导演制定·最高优先级）

**核心理念**：整场战斗不是"拍动作"，是"拍对抗"。镜头不是旁观者，是参与者。

#### 15L-1. 全程无静止·高速机动
- ❌ 任何形式的站桩对峙、双方站定互看、打完一招停下来等下一招
- ✅ 双方全程保持高速机动，连贯起势动作不间断
- ✅ 秦的路径：弧线+折线，连续变向不停顿
- ✅ 怪同时位移：追击/冲撞/挥臂/转身/扑过来，战斗全场动态
- ✅ 全程无静止画面——画面内永远有东西在动（人动/光动/尘动/衣摆动）

#### 15L-2. 镜头同步动态·四种主运镜
- ✅ 镜头跟随人物同步动态变化——人动镜动，人停镜不停
- ✅ 灵活切换四种主运镜：
  - **POV（第一人称主观视角）**：秦冲向怪的视角、怪看秦闪避逼近、石块砸向镜头
  - **环绕运镜**：蓄力时绕秦转、对峙时绕双方转增加压迫感
  - **FPV跟随**：镜头跟在秦身后/身侧高速移动，保持画面内秦的身影在前景
  - **跟拍**：侧面/背面跟踪拍摄，保持人物在画面中的运动轨迹
- ✅ 全程无固定镜头

#### 15L-3. 机动阶段·动态特写
- 人物突进、迂回拉扯等机动阶段 → 穿插动态特写镜头
- 特写内容：肢体蓄力姿态——腿部肌肉绷紧蓄势、腰胯拧转发力、肩臂收束蓄力
- 特写目的：放大蓄力→爆发的过渡瞬间，制造"下一秒就要炸"的紧张感
- 每次蓄力特写1-1.5秒，不要过长破坏节奏

#### 15L-4. 碰撞瞬间·慢镜头特写
- 双方招式对冲、兵刃碰撞的核心瞬间 → 切慢镜头特写
- 慢镜头内容：
  - 兵器与怪爪接触点——火星凝固在空气中、火焰在撞击点炸开慢放
  - 金属震颤——剑身受力弯曲再弹直的弹性变形
  - 秦的面部——牙关咬紧、眼神在撞击瞬间的微变化
  - 怪的反应——爪被震开、身体后仰
- 慢镜头时长0.5-1秒，接正常速度切下一个动作

#### 15L-5. 节点清晰·闪避+爆发
- 闪避腾挪节点：清晰标注回避方向+距离+回避后的反击起势
  - "秦侧身闪避，身体倾斜45度，怪爪擦过胸前，秦同时右手剑已扬起准备反击"
- 力量爆发节点：清晰标注蓄力→爆发→冲击波传导的全链
  - "秦重心下沉→腰胯猛然拧转→剑身火焰爆燃膨胀→横斩划出180度火焰弧光"
- 闪避和爆发之间不留空隙——闪即反，不落地不喘气

#### 15L-6. 双方角力·完整对抗逻辑
- 不止写秦的进攻，也要写怪的反制和施压
- 完整对抗逻辑链：秦出招 → 怪迎击/格挡 → 双方角力较劲 → 一方被震退 → 双方重新调整站位 → 再次接敌
- 远近交替：贴身缠斗→被震退拉开距离→远程攻击→再次近身
- 双方互有攻守、互有损伤——秦也会被震退/被撞飞/被擦伤

#### 15L-7. 场景深度融合·光效同步
- 所有动作与场地原生光影、环境特效深度融合
- 光效随招式起落同步变化：
  - 刀身火焰爆燃 → 画面暖色光同时增强，周围树木被照亮的范围扩大
  - 怪砸地 → 地面冲击波扩散时伴随暗光环境的短暂明灭
  - 秦被震退 → 火焰刀拖在地面划出火星轨迹，火光在地面上拖出长影
- 环境元素随动作变化：扬尘、落叶、火星、碎石——不是静止背景，是战斗的一部分
- "同步调用"不是"同时存在"——光是活的，随招式动态变化

#### 15L-8. 接触点·打击反馈质感
- 招式接触点着重刻画厚重打击反馈与力量对冲质感
- 每处碰撞必须包含：
  1. **撞击材质反馈**：金属碰爪=火星溅射/金属震颤，爪拍地面=碎石飞溅/尘土爆起
  2. **力量对冲表现**：双方兵器架在一起时——肌肉紧绷/脚下碾碎石板/兵器颤抖
  3. **冲击波扩散**：碰撞瞬间冲击波呈环形/锥形从接触点扩散，带动周围环境
- 打击反馈要"重"——不是轻飘飘的碰一下，是有质量的撞击

#### 15L-9. 面部神态·对峙张力
- 人物紧绷对峙的面部神态贯穿全场
- 不是打完才看表情——出招时、格挡时、被震退时、蓄力时，每个阶段都有对应的面部
- 出招时：牙关咬紧、眼神锁定、眉间微收
- 格挡时：闭气硬顶、脖颈绷住、眼神不退缩
- 被震退时：咬住牙关撑住、眼神闪过不服、稳住身形重新锁定
- 蓄力时：下颌紧绷、眼中火光映照、嘴角收敛
- 表情服务于"窒息压迫感"——不让观众松一口气

#### 15L-10. 速度冲击×蓄力爆发·反差节奏
- 整体节奏：速度冲击（快速机动+连续出招）⇄ 蓄力爆发（蓄力特写+慢镜头碰撞+爆发释放）
- 速度段：镜头切换快、动作连续、不给喘息、压迫感递增
- 蓄力段：镜头慢下来、特写蓄力姿态、时间仿佛凝固→爆发瞬间加速释放
- 反差越大→冲击力越强
- 全段不一个拍子：快→慢→快→骤停→再快——像心跳

#### 15L-11. 战斗节奏曲线（每15秒必须设计）
每段15秒出稿前必须心里有这条曲线：
```
[快起]机动突进(2-3s) → [慢]蓄力特写(1s) → [爆发]碰撞慢镜(1s) → [快]连斩追击(3-4s) → [慢]角力较劲(2s) → [爆发]震退拉开(1s) → [快]再次接敌(2-3s)
```
- 全程不允许连续3秒以上同一节奏
- 全段至少2次快慢切换

#### 15L-12. 激烈节点·静帧定格 ⚠️ 2026-06-27 导演新增（2026-06-28 修正）
- 打斗中最激烈/碰撞核心瞬间可插入静帧——完全静止的一帧画面
- 🔴 静帧落点铁律（2026-06-28 导演纠正）：定格优先落在角力点和打击点，不落在纯人物静态镜头
  - ✅ 角力点：双方兵器架在一起僵持较劲的瞬间、肌肉紧绷、脚下碾碎石板
  - ✅ 打击点：兵器碰撞火星飞溅的瞬间、刀砍剑斩命中目标瞬间、拳脚命中接触点、冲击波炸开的冻结
  - ❌ 纯人物静态镜头：收招后站定、单纯的人物面部表情特写——缺少画面张力和冲突感，效果弱
- 静帧内容：撞击接触点瞬间凝固 / 兵器碰撞的火星冻结在空中 / 角力僵持的对峙瞬间
- 时长0.3-0.5秒——不是慢镜头，是冻结
- 目的：用"时间暂停"制造冲击感，让观众在那一瞬间感受力量的重量
- 静帧后接正常速度或慢镜头继续——节奏斩断再续，冲击更强
- 每段15秒内最多1次静帧，用在高潮节点

#### 15L-13. 场景环境·物理联动 ⚠️ 2026-06-27 导演新增
- 打斗全程与场景环境物理元素深度联动，不只光影同步——环境是战斗的第三参与者
- 具体联动方式：
  - **利用地形**：蹬墙借力反冲、踏树跃起、借岩石掩护闪避、从天而降借势重击
  - **破坏环境**：兵器砍入石柱/树干、身体撞碎石壁/木箱、冲击波震碎周围物体
  - **扬起环境元素**：脚蹬地面扬起沙尘、剑风扫起落叶、冲击波激起水花/雪雾
  - **环境反作用于战斗**：倒塌的柱子阻断退路、扬起的灰尘遮蔽视线制造偷袭机会、碎石砸向双方
- 环境不是背景板——是战斗的第三参与者，被卷入、被破坏、反向施压
- 每个镜头至少有一处环境元素在动（扬尘/落叶/碎石/火星/布条飘动/水花）
- 环境联动逻辑必须通顺：地面材质决定扬什么（沙地扬沙/雪地扬雪/石地碎石）、建筑材质决定破坏方式（木结构碎裂/石结构崩裂）

#### 15L-14. 全身皆武器·手脚膝肘并用 ⚠️ 2026-06-27 导演新增
- 打斗中不只依赖手中兵器——手脚膝肘肩头全身皆为打击点
- 兵器挥砍间隙穿插：
  - **膝撞**：近身缠斗时膝顶腹部/胸口，短距爆发力最大
  - **肘击**：贴身时横肘劈砸面门/太阳穴，比剑更快更狠
  - **踢击**：低扫踢腿破坏对方重心、正蹬踹开拉开距离、回旋踢衔接兵器追击
  - **拳**：空手或持剑手拳背砸面，打断对方节奏
  - **肩撞**：贴身切入时肩头顶入胸腔，撞退后接兵器追击
  - **头槌**：极近距离被锁住时头槌破局
- 兵器与徒手交替节奏：斩→膝撞→斩→肘击→回旋踢→重斩——不连续两招同类型
- 手脚膝肘的打击同样需要反馈：撞击闷响/骨感冲击/对方被顶退的距离

### 16. 打斗中表情不僵硬的写法 ⚠️ 所有项目通用
- ❌ 永远「嘴角一翘」「眼中映着火光」——这是公式化，不是表演
- ❌ 每镜结尾都笑一下
- ✅ 表情随打斗阶段自然变化：
  **出击前**：目光聚焦锁定、眉间微收、下颌微绷
  **发力瞬间**：牙关咬紧、鼻翼微张、脖颈筋腱凸显
  **命中后**：眼中一闪而过的满意/杀意，嘴角微抿收住（不是笑）
  **被击中/格挡**：眉间一紧、牙关咬合、闭气硬顶、脖颈绷住（不是只有疼，还有硬顶和不服）
  **喘息/恢复**：嘴巴微张换气、眼皮微垂再抬起、喉咙吞咽
  **看见新威胁**：眼睛微睁大→随即收窄、嘴角收敛变严肃
- ✅ 同一场打斗中，秦的表情要有弧线：兴奋→专注→受挫→越打越来劲→疲惫但硬撑
- ✅ 表情写动作的反应，不是写在脸上贴标签

### 17. 表情压着写 ⚠️ 重申
- ✅ 模型会放大情绪——你说笑→咧嘴爆笑，你说哭→嚎啕大哭
- ✅ 用缓冲词：微扬/淡淡/一抹/若有若无/隐约/一闪而过/极淡/隐约可见
- ✅ 动作放大写（力量感砸实），表情压着写（微扬不爆）
- ✅ 「嘴角一翘」一天只能用一次，不是每个镜头结尾的标配

### 18. 打斗中角色可以说话 ⚠️ 所有项目通用
- ❌ 打斗=全员沉默，只有兵器声和脚步声
- ✅ 打斗中角色出声：发力喊/喘息/短句吐槽/骂一声/低吼/惊叹
- ✅ 台词夹杂在动作里，不是停下来对话——是边打边说
- ✅ 一句话≤5个字，多了节奏拖死（「还来？」「这么硬？」「跑！」「找死」）
- ✅ 语气和体力状态匹配：打累了说话带喘、刚发力说话中气足、被击中说话咬牙
- ✅ 秦的性格：闷骚——外表沉稳不张扬，内心有戏。吐槽不暴怒，幽默不浮夸，点到即止。表情也如此：不大喜大悲，情绪在眼和嘴角的微变化里。

### 18. 打斗镜头要丰富 ⚠️ 所有项目通用
- ❌ 只在全景/中景/近景三档轮换——单调
- ✅ 打斗段落中穿插以下特写镜头丰富节奏：
  **武器特写**：剑身火焰吞吐、刃面反光、剑尖颤抖、剑柄握紧
  **打击特写**：剑劈到怪身上的瞬间——火星爆溅、甲壳凹陷、火焰在撞击点炸开
  **手部特写**：五指攥紧剑柄、指节泛白、腕部拧转发力
  **脚步特写**：靴底蹬地碎石飞溅、滑步在地面犁出沟痕
  **兵器交锋特写**：剑与怪爪碰撞的接触点——火星、震颤、火焰被拍散飞溅
- ✅ 这些特写镜夹在全景/中景动作镜之间，1-2秒一个，当节奏的呼吸点
- ✅ 每段15秒里至少穿插1-2个武器或打击特写
- ✅ 俯拍和低角度在打斗中穿插使用，不重复：
  **俯拍**：秦被怪群围住时从正上方俯拍环形包围圈、怪倒地后俯拍秦站在倒地的怪旁、剑从上方劈下时短暂俯拍劈落轨迹
