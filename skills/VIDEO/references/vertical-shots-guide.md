# 竖屏镜头语言指南

## 竖屏 vs 横屏的本质区别

### 视觉差异
| 维度 | 竖屏 9:16 | 横屏 16:9 |
|------|----------|----------|
| 视野 | 纵向延伸 | 横向展开 |
| 主体 | 居中为主 | 可偏左/右 |
| 背景 | 展示有限 | 展示充分 |
| 人物 | 适合单人/双人 | 适合多人 |
| 运动 | 纵向运动优先 | 横向运动优先 |

### 心理差异
- **竖屏**：亲密感、沉浸感、个人化
- **横屏**：客观感、场面感、电影感

## 竖屏构图法则

### 1. 三分法则（竖屏版）
```
┌─────────────────┐
│     上1/3       │ ← 天空/天花板/留白
│   (字幕区域)    │
├─────────────────┤
│                 │
│     中1/3       │ ← 主体区域（人物头部）
│   (视觉中心)    │
│                 │
├─────────────────┤
│     下1/3       │ ← 地面/桌面/UI区域
│   (操作区域)    │
└─────────────────┘
```

**应用**：
- 单人镜头：人物眼睛在中1/3偏上
- 对话镜头：人物面部占据中1/3
- 动作镜头：动作在中1/3完成

### 2. 安全区域
```
┌─────────────────┐
│■■■■■■■■■■■■■■■■■│ ← 顶部10%：避免（被UI遮挡）
│                 │
│   ┌─────────┐   │
│   │ 安全区  │   │ ← 中间80%：主体区域
│   │  (主体) │   │
│   └─────────┘   │
│                 │
│■■■■■■■■■■■■■■■■■│ ← 底部10%：避免（被字幕遮挡）
└─────────────────┘
```

### 3. 竖屏专用构图

**中心对称构图**
```
    │
    │    👤
    │   /│\
    │    │
    │   / \
    │
```
- 人物居中，强调主体
- 适合：单人独白、情绪特写

**纵向延伸构图**
```
    │
    │    🏢
    │    🏢
    │    🏢
    │    👤
    │
```
- 利用纵向空间展示高度
- 适合：建筑、楼梯、仰拍

**上下分层构图**
```
    │
    │   👤A
    │───────
    │   👤B
    │
```
- 上下排列展示关系
- 适合：对话、对比、层级

## 竖屏镜头尺寸

### 标准尺寸（竖屏优化）

| 尺寸 | 英文 | 范围 | 竖屏用途 |
|------|------|------|---------|
| **极远景** | EWS | 全身+大量环境 | 开场、场景建立 |
| **远景** | WS | 全身+少量环境 | 全身动作、服装展示 |
| **中远景** | MWS | 膝盖以上 | 走路、动作场景 |
| **中景** | MS | 腰部以上 | 对话、手势 |
| **中近景** | MCU | 胸部以上 | 重要对话、表情 |
| **特写** | CU | 面部 | 情绪、反应 |
| **极特写** | ECU | 眼睛/嘴 | 强调、微表情 |

### 竖屏推荐尺寸组合

**短剧标准组合**：
```
开场：EWS（建立场景）
  ↓
进入：WS（展示人物）
  ↓
对话：MS-MCU（交替使用）
  ↓
情绪：CU-ECU（强调时刻）
  ↓
结束：WS（收束场景）
```

**情绪递进组合**：
```
平静：MS
  ↓
紧张：MCU
  ↓
高潮：CU
  ↓
爆发：ECU
  ↓
回落：MS
```

## 竖屏镜头运动

### 推荐运动（竖屏优化）

| 运动 | 英文 | 效果 | 竖屏应用 |
|------|------|------|---------|
| **升降** | Crane/Boom | 垂直移动 | ⭐⭐⭐ 竖屏最佳 |
| **上下摇** | Tilt | 垂直转动 | ⭐⭐⭐ 展示高度 |
| **推进** | Push in | 向前移动 | ⭐⭐ 强调情绪 |
| **拉远** | Pull back | 向后移动 | ⭐⭐ 揭示环境 |
| **跟随** | Follow | 跟随主体 | ⭐⭐ 运动场景 |
| **环绕** | Orbit | 绕主体旋转 | ⭐ 慎用（易头晕）|
| **平移** | Pan | 水平转动 | ⭐ 少用（竖屏不适合）|

### 竖屏专用运镜技巧

**1. 升降镜头（竖屏王牌）**
```
低角度开始
    ↓
缓缓上升
    ↓
高角度结束
```
- 效果：英雄登场、情绪升华
- 应用：主角亮相、重要时刻

**2. 上下摇镜**
```
从上往下摇
    ↓
展示全身
    ↓
到脚部结束
```
- 效果：展示完整形象
- 应用：服装展示、入场镜头

**3. 垂直推进**
```
远景开始
    ↓
沿垂直方向推进
    ↓
特写结束
```
- 效果：强调面部表情
- 应用：情绪爆发、关键反应

**4. 跟随+升降**
```
跟随人物走动
    ↓
同时轻微升降
    ↓
保持人物在中心
```
- 效果：动态展示
- 应用：走路场景、追逐场景

## 竖屏对话镜头

### 双人对话（竖屏版）

**方案A：上下切换**
```
镜头1：A在上半部分
镜头2：B在下半部分
```
- 优点：保持空间关系
- 缺点：需要后期剪辑

**方案B：居中切换**
```
镜头1：A居中 MCU
镜头2：B居中 MCU
```
- 优点：简单直接
- 缺点：空间感较弱

**方案C：过肩镜头（竖屏版）**
```
前景：A的肩膀（占画面1/4）
后景：B的面部（占画面3/4）
```
- 优点：保持空间关系
- 缺点：构图较紧

### 推荐：短剧对话标准
```
单人镜头：MCU（胸部以上）
情绪镜头：CU（面部特写）
反应镜头：CU（听者反应）
```

## 竖屏情绪镜头

### 情绪递进（竖屏版）

**开心/兴奋**：
```
MS：跳跃/动作
  ↓
MCU：笑容
  ↓
CU：眼睛发亮
```

**悲伤/哭泣**：
```
MCU：低头
  ↓
CU：眼泪
  ↓
ECU：眼泪滑落
```

**愤怒/冲突**：
```
MCU：对峙
  ↓
CU：愤怒表情
  ↓
快速切换：双方反应
```

**心动/暧昧**：
```
MCU：偷看
  ↓
CU：眼神交汇
  ↓
慢动作：微笑
```

## 竖屏卡点技巧

### 视觉卡点

**快切卡点**（配合强拍）：
```
节拍1：镜头A
节拍2：镜头B
节拍3：镜头C
节拍4：回到镜头A
```

**慢放卡点**（配合情绪高点）：
```
正常速度 → 慢动作 → 正常速度
    ↑
   情绪顶点
```

**变焦卡点**（配合重音）：
```
推进 → 定格 → 拉远
  ↑
 重音
```

### 竖屏转场

**1. 上下滑动**
```
画面向上滑出
    ↓
新画面从下滑入
```
- 适合：时间跳跃、场景切换

**2. 快速缩放**
```
快速推进到黑
    ↓
从黑快速拉出
```
- 适合：情绪转换、回忆/现实

**3. 旋转切换**
```
画面旋转90度
    ↓
新画面旋转进入
```
- 适合：强烈情绪转换

## 竖屏常见问题

### 问题1：画面太空
**原因**：主体太小，背景太多
**解决**：
- 使用更大尺寸（MS→MCU）
- 主体居中放大
- 减少环境展示

### 问题2：人物出框
**原因**：横向运动超出画面
**解决**：
- 限制横向运动范围
- 使用跟随镜头
- 改为纵向运动

### 问题3：字幕遮挡
**原因**：底部区域被字幕占用
**解决**：
- 主体上移，避开底部10%
- 使用侧面字幕
- 减少底部重要信息

### 问题4：场景单调
**原因**：竖屏展示环境有限
**解决**：
- 利用纵向空间（高建筑、楼梯）
- 增加镜头运动
- 使用前景遮挡增加层次

## 竖屏提示词模板

### 构图提示词
```
竖屏构图：
"vertical 9:16 composition, subject centered, 
head in upper-middle third, safe margins for subtitles"

安全区域：
"subject positioned in center 80% of frame, 
avoiding top and bottom 10% edges"
```

### 运镜提示词
```
升降镜头：
"crane shot moving vertically, smooth upward movement, 
revealing full figure from low angle"

上下摇镜：
"tilt shot from top to bottom, showing full body, 
vertical camera movement"

推进特写：
"push in vertically, focusing on face, 
intensifying emotion"
```

### 情绪提示词
```
开心：
"joyful expression, bright eyes, upward camera angle, 
sunlight filtering"

悲伤：
"tears streaming down, downward gaze, 
soft lighting, close-up on eyes"

愤怒：
"intense stare, jaw clenched, 
low angle looking up, dramatic shadows"

心动：
"soft focus, eye contact, 
slow motion, warm lighting"
```

## 实操检查清单

### 构图检查
- [ ] 主体是否在安全区域内？
- [ ] 是否避开了顶部和底部10%？
- [ ] 人物面部是否在中1/3偏上？
- [ ] 是否有足够的留白给字幕？

### 运动检查
- [ ] 是否优先使用纵向运动？
- [ ] 运动是否平滑不突兀？
- [ ] 是否避免了快速横向平移？
- [ ] 卡点是否与音乐配合？

### 情绪检查
- [ ] 景别变化是否有节奏？
- [ ] 特写是否用在关键时刻？
- [ ] 情绪递进是否自然？
- [ ] 竖屏是否增强了亲密感？

---

**核心原则**：竖屏不是横屏的裁剪版，而是独立的视觉语言。利用纵向空间，强调主体，控制运动，服务情绪。
