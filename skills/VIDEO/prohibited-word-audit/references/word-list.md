## §6 政治历史类（P2）

### 绝对避免

- 现实政治人物姓名/形象
- 现实政治事件/运动
- 敏感历史时期具体描述
- 民族/宗教敏感议题
- 领土/主权相关表述
- 任何可能影射现实的符号/旗帜/徽章

### 处理原则
- **虚构化**：使用虚构国家、时代、组织
- **模糊化**：不涉及具体细节
- **正能量**：强调团结、和平、发展

## §7 人物称呼类（P2·审核规避）

### 规则：提示词中用身份称呼替代具体人名

| 场景 | ❌ 禁用人名 | ✅ 替代称呼 |
|------|-----------|------------|
| 通用 | 秦轩、林莫、静虚等具体姓名 | 男主、男二、女主、大师兄、老祖、神秘人 |
| 通用 | 任何非必要真实姓名 | 身份+性别（青年男子/中年道人/黑袍长者） |

### 例外
- 打斗项目专用名（秦牧、怪A、怪B 等）：经导演确认的角色代号可使用
- 导演明确说「用真名」→ 按导演指令

### 来源：SKILL.md §九·审核规避补充（郑有朋 2026-04-28）

## §8 歧义词类（P2·审核规避）

AI审核系统可能对某些词做多义解读。以下词一律替换：

| 违禁词 | 原因 | 替代词 |
|--------|------|--------|
| 歪脖子树 | "歪脖"可能被关联异常姿势/政治隐喻 | 老树、古树、盘虬古树 |
| 歪脖 | 同上 | 倾斜、盘虬、扭曲 |
| 独眼 | 可能触发敏感内容 | 单目、一侧遮蔽 |

### 原则
遇到任何可能被多义解读的词 → 立即替换为中性/正向同义词

### 来源：SKILL.md §九·审核规避补充（郑有朋 2026-04-28）

## §9 空词类（P2·禁止使用）

模型无法理解概括性空词，必须替换为具体描述：

| ❌ 空词（禁用） | ✅ 具体描述 |
|----------------|-----------|
| 火焰光效 | 橙红色火焰从剑身喷涌而出，火星四溅 |
| 冲击波光效 | 半透明环形冲击波以剑为中心向外扩散 |
| 能量光效 | 蓝色电弧在拳锋间跳跃闪烁 |
| 爆炸光效 | 橙白光团炸开，碎片和粉尘向外飞散 |
| 战斗光效 | 兵器碰撞迸发橙黄色火星 |
| 技能特效 | （必须写清楚是什么技能的具体视觉表现） |
| 光影效果 | （必须写清楚光源方向+亮度+颜色+投射效果） |

### 规则
- 不允许任何"XX光效""XX特效"的概括性写法
- 每个"光效"必须是具体颜色+具体形态+具体动态

### 来源：prompt-writing-rules.md §禁止空词

## §10 裸动词类（P3·力度锚定）

所有动作必须带力度词+速度词，裸动词一概不用：

### 禁用裸动词清单
```
挥剑、劈下、横扫、闪避、落地、格挡、蹬地、转身
挥刀、出拳、踢腿、跳跃、冲刺、后撤、侧移、翻滚
斩击、刺出、砍落、砸下、冲撞、推开、抵挡、躲避
蓄力、蓄势、发力、爆发（后两者视语境——若独立使用为裸动词）
```

### ✅ 锚定写法
```
快速挥剑、猛烈劈落、凌厉横扫、急速闪避、迅猛落地、
奋力格挡、暴力横抡、凶狠砸地、猛蹬地面、骤然转身、
闪电般冲刺、骤然侧移、凌空翻滚、疾风般斩击
```

### 力度词库
快速/猛烈/暴射/凌厉/疾速/迅猛/凶狠/奋力/狠砸/暴力/急速/疾冲/猛蹬/狠劈/狂扫/暴砍/迅斩/骤然/凶悍/狂暴

### 速度词库
高速/极速/闪电般/瞬间/迅疾/一闪/快如/骤转/刹那间/弹射般/疾风般

### 规则
- 秦的所有动作 + 所有对手动作 = 一律锚定力度词+速度词
- 静态缓慢场景中的动作（呼吸/睁眼/转头）可用缓冲词：缓慢/轻缓/缓缓/徐徐

### 来源：prompt-writing-rules.md §15G + 实战经验

## §11 过度写实类（P3·审核规避）

### 规则
- 「真人」「写实」在整段提示词中出现 ≤ 1次
- 超过1次可能触发"是否违规使用真人肖像"的审核逻辑

### ❌ 违规示例
```
电影级真人实拍质感，真实皮肤纹理，写实摄影风格，非CGI非手绘
```

### ✅ 合规写法
```
电影级光影，自然皮肤质感
```
或
```
写实质感，光影塑造立体感
```

### 替代策略
- 用暗示代替明示：「光影自然」「材质真实」「皮肤质感细腻」
- 不堆叠"真人""写实""真实"等同义词

### 来源：SKILL.md §九·审核规避补充（郑有朋 2026-04-28）

## 附录A：场景处理示例

### 武侠打斗

**原描述（含违禁词）**:
```
黑衣杀手施展血影刀法，刀光带血，杀气腾腾。
一刀斩下，对手鲜血喷溅，倒地身亡。
脖颈中刀，皮开肉绽。
```

**安全版本**:
```
神秘黑衣人施展影刃刀法，刀光如影，气势逼人。
一刀斩落，对手被击退，退出战斗。
侧颈受击，皮肤裂开。
能量碰撞产生红色光效，象征激烈对决。
```

### 魔法战斗

**原描述（含违禁词）**:
```
黑暗巫师召唤地狱火焰，焚烧一切。
灵魂哀嚎，恐怖景象。
```

**安全版本**:
```
神秘法师操控暗色能量，形成高温火焰。
能量场剧烈波动，产生视觉冲击效果。
光影变化象征力量对抗。
```
