## 模板概述

### 核心特征

| 特征 | 说明 |
|------|------|
| 镜头数 | 6镜循环 |
| 总时长 | 约8-10秒（可扩展至15秒） |
| 打斗类型 | 近身徒手格斗（可替换为短兵） |
| 视觉风格 | 写实电影、高对比、侧逆光、脏镜效果 |
| 节奏曲线 | 对峙→攻击→反击→连击→终结→回归 |
| 闭环设计 | 尾帧主角姿势/站位/重心与首帧完全一致 |
| 独创体系 | 冲击波四层描写（碰撞点/介质层/破坏层/镜头反馈层） |

### 冲击波四层体系（模板核心方法论）

```
碰撞点：具体接触部位 + 冲击波形态（圆形/锥形/扇形 + 扩散方向）
介质层：空气/水/尘/光的中间传递效果
破坏层：物理世界的实际损伤（衣物/道具/环境）
镜头反馈层：摄影机本身的"受伤"（震动/失焦/色散/脏镜）
```

**为什么有效**：模型读到的不再是"一拳打中+冲击波"笼统描述，而是从接触点→空气→环境→镜头的完整因果链。每一层在物理世界都有对应画面，AI逐层渲染不容易崩塌。

| 1 | **冲击波四层体系** | 碰撞点→介质层→破坏层→镜头反馈层分级描写，AI逐层渲染因果链清晰，比笼统写"冲击波扩散"精准一个量级 |

## 三、使用指南

### 3.1 角色替换

替换模板中 `陈啸` `铁虎` 为目标角色，同步修改：
- 外貌描述（体型/发型/面部特征）
- 服装（战术背心→目标服装）
- 兵器（铁管→目标兵器；如徒手则删铁管描述）
- 角色质感尾缀中的表情描述（沉冷/凶狠→目标情绪）

### 3.2 场景替换

替换 `城郊废弃汽车修理厂` 为目标场景，同步修改：
- 空间尺寸与布局
- 关键道具（铁柱/玻璃瓶堆→目标场景元素）
- 光源数量与方向
- 积水/路面材质
- 【镜头参数】中的光影描述

### 3.3 兵器替换

| 原兵器 | 替换为 | 需同步修改 |
|--------|--------|-----------|
| 铁管 | 短刀/棍/剑 | 碰撞点描述、音效、冲击波形态 |
| 徒手（主角） | 带兵器 | 攻击方式（拳→刀砍/剑刺）、握持手标注 |
| — | 双持 | 需额外标注左手/右手各自动作 |

### 3.4 时长调整

| 目标时长 | 调整策略 |
|-----------|---------|
| 8秒版 | 镜头1缩至1.0s / 镜头6缩至1.8s |
| **10秒版（当前）** | 保持不动 |
| 15秒版 | 镜头1扩至2.5s（对峙加呼吸节奏变化）/ 镜头6扩至4s（玻璃碎裂慢镜+更长沉降恢复）/ 或插入1个1.5s反应镜 |

### 3.5 冲击波四层调参

| 打击类型 | 冲击波形态 | 介质层 | 破坏层 | 镜头反馈层 |
|---------|-----------|--------|--------|-----------|
| 重拳/膝/肘 | 圆形/锥形气浪 | 空气扭曲+液体飞溅 | 衣物褶皱+身体位移 | 震动+失焦+色散 |
| 兵器交击 | 线形火花爆散 | 空气波纹+金属碎片 | 兵器震颤+地面刮痕 | 震动+高光过曝瞬间 |
| 身体撞墙/物 | 扇形碎片炸开 | 碎片+粉尘扩散 | 物体碎裂+地面拖痕 | 剧烈震动+脏镜 |
| 轻击/连打 | 无可见冲击波 | 汗珠飞溅 | 衣物轻微翻动 | 轻微震动 |
