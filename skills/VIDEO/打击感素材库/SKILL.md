## 一、拳法打击（7组）

### 1. 正面重拳
角色蓄力挥拳，拳头贴近镜头，面部狠厉，衣摆被气流掀起。
- **光效**：前景拳风爆开，强对比边缘光
- **动态**：速度线、冲击波扩散
- **镜头**：镜头轻震
- **适用**：正面交锋起手、破防一击

### 2. 寸拳爆发
极近距离内瞬间出拳，拳程虽短却力量集中，胸肩同时发力。
- **光效**：短促冲击波、前景空气炸裂纹
- **动态**：局部特写
- **镜头**：镜头微震
- **适用**：贴身搏斗、防守反击

### 3. 上勾顶击
自下而上轰出上勾拳，力量把对手直接顶起。
- **光效**：下冲光效
- **动态**：夸张速度线、下巴受击定格
- **镜头**：仰拍镜头
- **适用**：近身破防、将对手打浮空

### 4. 双拳连轰
左右手连续交替出拳，节奏密不透风，拳影交错覆盖画面。
- **光效**：连续冲击纹
- **动态**：多重残影、快速节奏切镜、局部震屏
- **镜头**：快速切换
- **适用**：压制连打、狂暴输出

### 5. 突脸冲拳
突然贴脸冲近后直拳爆出，拳头几乎顶到镜头前，面部杀气集中。
- **光效**：前景拳风
- **动态**：近景透视夸张、高速冲击线
- **镜头**：镜头后仰
- **适用**：突袭起手、压迫式开场

### 6. 反手鞭拳
快速转身后反手甩出鞭拳，拳路狠快、弧线清晰。
- **光效**：空气撕裂线
- **动态**：手臂拖尾、脸部狠厉特写、卡点震屏
- **镜头**：—
- **适用**：转身突袭、出其不意

### 7. 抡臂猛砸
大幅抡臂后猛然重砸，动作幅度大、力量沉，手臂轨迹清晰可见。
- **光效**：受击爆点
- **动态**：重型拖尾、风压弧线
- **镜头**：镜头沉降感
- **适用**：力量型角色、重型打击

## 二、腿法打击（7组）

### 8. 侧身横踢

### 10. 腾空下劈

- **动态**：布料翻飞

### 12. 转体鞭腿

- **适用**：中远距离重击

### 13. 扫堂掀翻

- **光效**：—

- **镜头**：镜头贴地横移
- **适用**：低位突袭、掀翻控制

### 14. 重踏震场
重重踏地后借势爆冲，脚下先产生震场裂纹，再把力量传到下一击。
- **光效**：冲击波扩散、主角高亮
- **动态**：地面开裂
- **镜头**：镜头低机位
- **适用**：蓄力前摇、气场压制

## 十、使用方式

写打斗分镜时：
1. 确定招式类型（拳/腿/膝肘/兵器/摔投/爆发/终结）
2. 在对应分类中选出具体招式编号
3. 将该组的 光效+动态+镜头 三项直接嵌入分镜内容描述
4. 多个招式连段时，逐组叠加效果

**示例**：
```
原分镜：秦牧右拳直击骷髅兵面部
查 #1 正面重拳 →
改为：秦牧蓄力右拳轰出，拳风在镜头前爆开，衣摆被气流掀起，
强对比边缘光勾勒手臂轮廓，冲击波自拳面扩散，镜头轻震
```
