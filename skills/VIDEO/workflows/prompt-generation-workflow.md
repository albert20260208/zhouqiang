## 调用原则

## 详细执行步骤

### Step 1: 需求分析
1. 确认用户需求类型（视频类型、风格、时长）
2. 确定是否需要参考图片/视频
3. 确认输出格式要求
4. 了解用户特殊需求

### Step 2: 故事框架构建
1. 使用三幕结构/Save the Cat!/英雄之旅模板
2. 确定主要人物和关系
3. 设计关键情节节点
4. 规划情绪曲线

### Step 3: 视觉风格确定
1. 选择导演风格参考
2. 确定整体色调和氛围
3. 设计关键镜头序列
4. 规划运镜方式

### Step 4: 场景一致性设计
1. 使用四面墙方法建立场景空间
2. 确定人物位置关系
3. 设计场景转换方式
4. 确保空间逻辑合理

### Step 5: 提示词生成
1. 按照Seedance提示词结构编写
2. 加入@引用系统（如需要）
3. 设置时间分段
4. 添加风格和氛围关键词

### Step 6: 审核优化
1. 敏感词检查
2. 一致性检查
3. 平台适应性检查
4. 最终优化调整

## 示例工作流

### 案例：武侠短剧提示词制作

#### 用户需求
"帮我做一个3分钟的武侠短剧提示词，主角是白衣剑客，在竹林中对战黑衣杀手"

#### 执行流程

**Step 1: 故事层**
1. 使用三幕结构：
   - 第一幕：竹林相遇，对峙
   - 第二幕：激烈对战，招式对决
   - 第三幕：白衣剑客胜出，黑衣杀手逃走
2. 设计情绪曲线：紧张→激烈→释然

**Step 2: 视觉层**
1. 导演风格：王家卫+武侠风格
2. 关键镜头：
   - 开场：竹林全景，白衣剑客静立
   - 中段：快速剪辑对战镜头
   - 结尾：慢动作收剑特写
3. 运镜方式：环绕、快速横摇、慢动作

**Step 3: 场景层**
1. 四面墙方法建立竹林场景
2. 确定人物位置关系
3. 设计打斗路径

**Step 4: 生成层**
1. 编写Seedance提示词
2. 添加@引用（如需要参考图）
3. 设置参数：时长3分钟，4K画质
4. 敏感词检查：将"杀气"改为"气势"

#### 最终输出
```
【场景】竹林，月光如水，竹叶纷飞
【人物】白衣剑客对战黑衣杀手
【动作】剑法对决，招式连绵，剑气纵横
【运镜】快速剪辑，慢动作特写，环绕拍摄
【情绪】紧张激烈，最终释然
【风格】王家卫武侠风，抽帧效果，电影质感
【时长】3分钟，4K画质
```

## 常见问题处理

### Q1: 用户需求不明确
**处理**：主动提问澄清
- 视频类型是什么？
- 目标时长是多少？
- 需要什么风格？
- 有无参考材料？

### Q2: 复杂场景处理
**处理**：分阶段处理
1. 先处理主要场景
2. 再处理细节
3. 最后整合检查

### Q3: 技术限制
**处理**：合理预期管理
- 说明AI生成限制
- 提供替代方案
- 建议分步骤生成

### Q4: 审核风险
**处理**：预先规避
- 使用敏感词库检查
- 提供安全版本
- 准备备选方案
