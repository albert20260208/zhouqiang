# 短剧制作工作流

## 🎬 短剧特点
- **快节奏**：3-5分钟完整故事
- **强情绪**：高冲突、情感张力强
- **竖屏优化**：手机端观看体验
- **场景集中**：减少场景切换，保持一致性
- **人物聚焦**：主角明确，关系清晰

## 📋 制作流程

### 阶段1：剧本创作
#### 1.1 故事构思
- **核心冲突**：明确主要矛盾（爱情、复仇、商战等）
- **人物设定**：主角、反派、配角关系网
- **三幕结构**：
  - **第一幕**（0-1分钟）：引入冲突，建立人物
  - **第二幕**（1-3分钟）：冲突升级，情感爆发
  - **第三幕**（3-5分钟）：解决冲突，情感释放

#### 1.2 剧本格式
```
【场景1：办公室】
时间：白天
人物：林总（霸道总裁）、苏秘书

林总（冷漠）："这份合同，今晚必须签下来。"
苏秘书（紧张）："可是对方要求..."

【场景2：会议室】
时间：晚上
人物：林总、苏秘书、王总

王总（狡猾）："林总，条件很简单，让你的秘书陪我..."
```

### 阶段2：分镜设计
#### 2.1 竖屏镜头语言
- **特写镜头**：情感表达，面部表情
- **中景镜头**：对话场景，人物互动
- **全景镜头**：环境展示，场景建立
- **运动镜头**：推拉摇移，增强动感

#### 2.2 分镜表模板
| 镜头 | 时长 | 景别 | 内容 | 运镜 | 备注 |
|------|------|------|------|------|------|
| 1 | 3秒 | 特写 | 林总冷漠表情 | 固定 | 开场冲击 |
| 2 | 5秒 | 中景 | 林总递合同 | 推近 | 强调动作 |
| 3 | 4秒 | 特写 | 苏秘书紧张表情 | 固定 | 情感反应 |

### 阶段3：角色设计
#### 3.1 人物特征一致性
- **外貌特征**：发型、发色、五官、身高
- **服装风格**：职业装、休闲装、特定时代服装
- **配饰细节**：眼镜、手表、首饰、道具

#### 3.2 角色提示词库
```markdown
### 霸道总裁
- 年龄：28-35岁
- 外貌：冷峻五官，锐利眼神，西装革履
- 气质：强势，自信，掌控感
- 服装：定制西装，名表，皮鞋

### 职场小白
- 年龄：22-26岁  
- 外貌：清秀面容，单纯眼神，朴素打扮
- 气质：青涩，努力，有潜力
- 服装：职业套装，平底鞋，简单配饰
```

### 阶段4：场景生成
#### 4.1 场景一致性保持
- **连续镜头**：使用上一视频最后一帧作为首帧
- **物品布局**：场景内所有物品位置不变
- **色调统一**：光线、色彩风格保持一致
- **人物状态**：服装、表情、位置连续

#### 4.2 场景提示词
```
现代都市办公室，落地窗，简约装修
办公桌上文件整齐，电脑屏幕亮着
窗外城市天际线，白天自然光
电影质感，浅景深，专业摄影
```

### 阶段5：视频生成
#### 5.1 Seedance提示词结构
```
【场景】现代都市办公室，白天
【人物】林总，28岁霸道总裁，黑色西装
【动作】冷漠地递出合同，眼神锐利
【运镜】中景推近特写
【对白】"这份合同，今晚必须签下来。"
【风格】职场剧，紧张氛围，电影质感
```

#### 5.2 多镜头衔接
1. **镜头1**：特写林总表情
2. **镜头2**：中景递合同动作
3. **镜头3**：特写苏秘书反应
4. **镜头4**：双人对话场景

每个镜头使用上一镜头的最后一帧作为首帧参考。

### 阶段6：后期整合
#### 6.1 音频处理
- **背景音乐**：情绪配乐（紧张、浪漫、悲伤）
- **音效**：环境音、动作音、转场音
- **对白**：清晰可辨，情绪到位

#### 6.2 字幕添加
- **位置**：底部安全区域
- **样式**：白色带黑边，易读字体
- **时机**：与对白同步，提前0.5秒出现

#### 6.3 转场效果
- **硬切**：场景切换
- **淡入淡出**：时间过渡
- **划变**：地点转换
- **匹配剪辑**：动作连贯性

## 🎯 质量控制

### 一致性检查
1. **人物一致性**：发型、服装、配饰是否变化
2. **场景一致性**：物品位置、光线、色调是否统一
3. **时间连续性**：动作、表情、状态是否连贯
4. **物理合理性**：重力、运动、交互是否符合常识

### 情感表达检查
1. **表情匹配**：面部表情是否符合对白情绪
2. **肢体语言**：动作是否传达正确情感
3. **节奏控制**：快慢节奏是否符合剧情需要
4. **高潮处理**：情感爆发点是否足够强烈

## 🔧 技术规范

### 视频规格
- **分辨率**：1080×1920（竖屏9:16）
- **帧率**：30fps
- **时长**：3-5分钟（分15-25个镜头）
- **格式**：MP4，H.264编码

### 音频规格
- **采样率**：48kHz
- **比特率**：192kbps
- **声道**：立体声
- **格式**：AAC

### 字幕规格
- **字体**：思源黑体/苹方
- **大小**：36-42px
- **颜色**：白色带2px黑边
- **位置**：底部1/8处

## 🚀 快速开始

### 1. 选择剧本模板
从`templates/`目录选择适合的剧本模板

### 2. 修改人物设定
根据剧情需要调整角色特征

### 3. 生成分镜表
按照竖屏镜头语言设计分镜

### 4. 生成参考图片
使用Seedream生成角色和场景参考图

### 5. 生成视频片段
按分镜顺序使用Seedance生成视频

### 6. 后期整合
添加音乐、音效、字幕，输出成品
