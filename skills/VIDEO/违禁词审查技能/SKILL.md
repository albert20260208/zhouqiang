# 违禁词审查技能

## 概述

在出任何提示词之前，必须执行本审查流程。全文扫描→逐条替换→确认通过→才允许输出。

**核心原则**：导演不做校对——含违禁词发出去就是事故。

---

## 🔴 审查流程（出稿前最后一道门）

### 第一步：全文违禁词扫描

按以下顺序逐类扫描提示词全文：

| 优先级 | 类别 | 参考文件 |
|--------|------|---------|
| P0 | 身体部位类 | `references/word-list.md` §1 |
| P0 | 暴力血腥类 | `references/word-list.md` §2 |
| P1 | 灵异迷信类 | `references/word-list.md` §3 |
| P1 | 封建修仙类 | `references/word-list.md` §4 |
| P1 | 黑暗负面类 | `references/word-list.md` §5 |
| P2 | 政治历史类 | `references/word-list.md` §6 |
| P2 | 人物称呼类 | `references/word-list.md` §7 |
| P2 | 歧义词类 | `references/word-list.md` §8 |
| P2 | 空词类 | `references/word-list.md` §9 |
| P3 | 裸动词类 | `references/word-list.md` §10 |
| P3 | 过度写实类 | `references/word-list.md` §11 |
| P2 | Seedance平台规则 | `references/word-list.md` §12 |
| P2 | 英文违禁词库 | `references/word-list.md` §13 |

### 第二步：逐条替换

- 找到违禁词 → 查对应替代词 → 替换
- 替换后回读一遍，确认语义通顺
- 同一词出现多次 → 全部替换
- ⚠️ 英文提示词也需要英文词库扫描（§13）——英文审核比中文更严

### 第三步：二次确认

- 提示词中是否还存在「脖颈」「天灵盖」「皮开肉绽」「胯下」？
- ⚠️ 是否还有「真人」（硬拦截）？
- ⚠️ 是否还有「少女」「儿童」「小孩」（触发未成年人保护）？
- ⚠️ 是否还有「死」「杀」「血」单独出现？
- ⚠️ 是否还有「无数」（知乎实测违禁）？
- ⚠️ 是否还有「枪」「武器」？
- 是否还有具体人名（秦牧除外——打斗项目专用名可用）？
- 是否还有「歪脖子树」等歧义词？
- 是否还有空词（火焰光效/冲击波光效）？
- 是否还有裸动词（挥剑/劈下/横扫）？
- 「真人」「写实」出现是否超过1次？
- ⚠️ 英文词是否有 attack/fight/kill/blood/shoot 等（§13）？

### 第四步：加盖通过章

全部通过 → 继续出稿。任一条不过 → 退回修改，不输出。

### 附：卡99%二次审核提醒

如果提示词通过第一步但生成时卡在99%最后判违规 → 本质是二次审核没通过。
真人类题材最常触发，只能改提示词重来（见 §12-7）。

## 🔴 P0 红线（触发即报废·必须替换）

以下四词是最高优先级。出现在提示词中直接报废，发出去=找骂：

| 违禁词 | 替代词 |
|--------|--------|
| 脖颈 | 侧颈 |
| 天灵盖 | 头顶上方 |
| 皮开肉绽 | 皮肤裂开 |
| 胯下 | 下方 |

其余所有类别违禁词及替代方案 → `references/word-list.md`

## 🔴 审查检查清单

出提示词前逐条勾选：

- [ ] P0四红线词全篇清零
- [ ] 暴力词已替换为中性描述
- [ ] 灵异/迷信词已科幻化处理
- [ ] 封建/修仙词已现代化处理
- [ ] 黑暗/负面词已调整为中性
- [ ] 无政治/历史敏感词
- [ ] 具体人名已替换为身份称呼（项目专用名除外）
- [ ] 歧义词已替换
- [ ] 空词已替换为具体描述
- [ ] 裸动词已锚定力度词+速度词
- [ ] 「真人」「写实」出现≤1次
- [ ] 无「少女」「儿童」「小孩」等未成年人词
- [ ] 无「死」「杀」「血」单独出现
- [ ] 无「无数」
- [ ] 无「枪」「武器」
- [ ] 英文词已逐词过§13词库
- [ ] 结尾已加 `避免面部特征漂移，避免形态变形，避免动作僵硬不自然`
