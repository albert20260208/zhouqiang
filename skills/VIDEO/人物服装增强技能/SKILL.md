# 人物服装增强技能

> **创建日期**：2026-06-26
> **来源**：全网搜索（火山引擎官方/Reddit/Threads/Instagram/博客园）+ 现有知识库去重整合
> **用途**：为 Seedance 视频提示词提供系统化的人物（皮肤/眼睛/头发）和服装质感增强方案
> **与现有体系关系**：本技能是 `prompt-writing-rules.md §角色质感尾缀` 的扩展增强版，尾缀为核心锚点（必加），本技能为可选增强模块

---

## 一、概述

### 问题诊断

AI 视频生成人物常见的"塑料感"源于：
| 问题 | 根因 | 解决方向 |
|------|------|---------|
| 皮肤光滑如塑料 | 模型默认均值平滑，缺少微纹理 | 加毛孔/纹理/次表面散射 |
| 眼神空洞呆滞 | 虹膜无纹理、无眼神光、无微动 | 加虹膜细节/眼神光/眨眼微动 |
| 头发整整齐齐 | 无碎发/毛躁/飘动，违反物理 | 加碎发/飞丝/逆光发丝 |
| 服装全新完美 | 无褶皱/磨损/面料纹理 | 加编织纹理/自然折痕/面料特性 |
| 面部特征漂移 | 帧间五官位移 | avoid约束+皮肤锚定词 |

### 核心原则

**不是加细节，是加"不完美"。** AI 默认输出太完美 = 一眼假。真人感来源于可控的不完美。

## 二、四维质感增强模块

### 🟢 维度一：皮肤（Skin）

#### 基础版（适用于所有景别，已含在现有尾缀中）

```
皮肤毛孔纹理全程保留，肤质前后帧保持一致，不会自动磨皮
```

#### 高配版（近景/特写追加，叠加在基础版之后）

```
皮肤毛孔纹理全程保留，皮下微血管隐约可见，颧骨自然高光，T区微油光，
肤质前后帧保持一致，不会自动磨皮，自然皮肤微纹理和细纹可见，
非CG光滑，subsurface scattering次表面散射光感，轻微肤色不均保留真实感
```

#### 特写增强词库

| 英文（模型理解更准） | 中文 | 效果 |
|---------------------|------|------|
| `subsurface scattering` | 次表面散射 | 光穿透皮肤再反射，通透感 |
| `visible pores, pore depth` | 可见毛孔，有深度 | 非平面皮肤 |
| `vellus hair catching sidelight` | 桃色绒毛侧光可见 | 脸颊/额头真实微毛发 |
| `uneven sebum sheen` | 不均匀皮脂光泽 | 对抗AI均匀高光 |
| `micro-wrinkles` | 微皱纹 | 表情纹/眼角细纹 |
| `subtle blemishes, realistic skin tone variation` | 轻微瑕疵，肤色不均匀 | 真实肤色 |
| `natural skin glow, dewy but not oily` | 自然水光，非油腻 | 健康肤质 |
| `peach fuzz visible in rim light` | 轮廓光下绒毛可见 | 逆光人像真实感 |

#### 场景适配

| 场景 | 皮肤策略 | 推荐模块 |
|------|---------|---------|
| 全景/远景 | 不写皮肤细节（模型看不到） | 基础版即可 |
| 中景/中近景 | 皮肤质感一句话带过 | 基础版 |
| 近景（胸部以上） | 毛孔+肤质+光影 | 高配版 |
| 特写/大特写 | 毛孔深度+绒毛+微纹理+次表面散射 | 高配版+特写词库择2-3个 |

### 🟢 维度二：眼睛（Eyes）

#### 基础版（已含在现有尾缀中）

```
眼神灵动，缓慢自然眨眼，眼球轻微转动，眼神有神
```

#### 高配版（近景/特写追加）

```
眼神灵动，双眼均有眼神光catchlights不跳动，虹膜放射状纤维纹理可见，
自然缓慢眨眼，眼球轻微转动，下眼睑湿润线wetline可见，
睫毛根根分明自然卷翘，眼白略带微血管非纯白，瞳孔光线折射自然
```

#### 特写增强词库

| 英文 | 中文 | 效果 |
|------|------|------|
| `catchlights in both eyes` | 双眼均有眼神光 | 对抗AI常忘一只眼 |
| `iris radial fibers visible` | 虹膜放射状纹理 | 瞳孔不空洞 |
| `wetline on lower eyelid` | 下眼睑湿润线 | 眼睛不干涩 |
| `pupil light refraction natural` | 瞳孔自然光线折射 | 玻璃体通透 |
| `eyelashes individually visible` | 睫毛根根分明 | 不粘连成块 |
| `subtle scleral veins` | 眼白微血管 | 非纯白假眼 |
| `micro eye darting movements` | 眼球微颤/扫视 | 活人眼神 |

#### 眼神情绪速查

| 情绪 | 眼睛描述 |
|------|---------|
| 坚定 | 眼神锐利，瞳孔聚焦前方，不闪躲 |
| 悲伤 | 眼神黯淡，下眼睑微红，目光下垂 |
| 愤怒 | 眼神凌厉，眉弓下压，瞳孔收缩 |
| 恐惧 | 瞳孔放大，眼白露出增多，快速扫视 |
| 温柔 | 眼神柔和，眼角微弯，目光温暖 |
| 冷漠 | 眼神淡然，瞳孔无焦点，半垂眼帘 |
| 狡黠 | 眼角微挑，目光斜瞥，瞳孔灵动 |

### 🟢 维度三：头发（Hair）

#### 通用基础（全新，现有尾缀未覆盖）

```
发丝根根分明，发丝纹理清晰可见，头发自然光泽，
高光沿发丝走向流动，碎发自然散落，鬓角绒毛可见
```

#### 按发型增强

**散发/长发：**
```
长发自然垂落，发丝间透光，发尾微卷，碎发在脸侧飘动，
逆光下形成发丝光晕，个别飞丝flyaway hairs自然散出，
发丝纹理从发根到发梢方向一致，避免头发像塑料假发，
避免发丝粘连成块
```

**束发/发髻：**
```
发髻紧致，发丝纹理清晰，额前碎发轻拂，
碎发修饰脸颊和鬓角，后颈碎发自然散落，
发根蓬松立体不贴头皮
```

**短发：**
```
短发利落，发根蓬松立体，发丝纹理分明，
发梢自然不规则，鬓角碎发自然，
头顶发丝有层次感，非均匀一刀切
```

#### 特写增强词库

| 英文 | 中文 | 效果 |
|------|------|------|
| `individual hair strands visible` | 单根发丝可见 | 不糊成一片 |
| `flyaway hairs catching light` | 飞丝在光中可见 | 动态真实感 |
| `natural frizz and loose strands` | 自然毛躁松散发丝 | 对抗整齐假发 |
| `hair specular highlights along strand direction` | 高光沿发丝走向 | 不反光成一团 |
| `vellus hair at temples and hairline` | 太阳穴/发际线绒毛 | 发际线真实 |
| `baby hair wisps at hairline` | 发际线胎毛碎发 | 自然发际线 |
| `scalp subtly visible through parting` | 发缝处头皮隐约可见 | 头发厚度真实 |

#### 避坑词（必加）

```
避免头发像塑料假发，避免发丝粘连成块，避免发际线整齐如刀切
```

### 🟢 维度四：服装（Clothing）

#### 基础版（已含在现有尾缀中）

```
服装面料细节拉满，褶皱丰富，层次分明，布料肌理清晰，
丝绸高光通透，薄纱半透质感，色彩冷暖层次丰富，光影塑造立体感
```

#### 按面料类型增强

##### 丝绸/Satin
```
丝绸高光沿褶皱走向流动，表面柔光散射，
不同褶皱面反光角度不同形成明暗层次，
面料自然垂坠，折叠处形成软阴影
```
- 追加关键词：`silk specular highlights follow fold direction`, `soft light diffusion on fabric surface`

##### 棉麻/Cotton & Linen
```
布纹经纬纹理可见，自然褶皱形成软阴影，
面料微粗糙质感，亚麻特有微皱纹理，
cotton weave texture visible at thread level
```
- 追加关键词：`fabric weave visible`, `natural crease patterns`, `thread-level texture`

##### 皮革/Leather
```
皮质纹理微反光，折痕处自然磨损光泽，
表面微颗粒纹理，边缘磨损做旧，
leather grain texture with natural patina
```
- 追加关键词：`leather grain surface`, `natural wear patina`, `crease highlights`

##### 铠甲/Armor
```
金属表面哑光处理，铆钉高光点独立反光，
甲片间皮革衬垫纹理，金属边缘自然磨损，
叠合处形成缝隙阴影，铠甲关节处活动褶皱
```
- 追加关键词：`metal surface matte finish`, `individual rivet highlights`, `segmented plate shadows`

##### 薄纱/Chiffon & Organza
```
薄纱半透，叠层处透光度递减，边缘轻飘，
纱质面料在微风中轻微起伏，
sheer fabric light transmission varies by layer count
```
- 追加关键词：`sheer fabric transparency gradient`, `lightweight hem flutter`

##### 毛呢/羊毛/Wool
```
毛呢面料表面微毛绒纹理，自然厚重垂坠，
折痕处面料厚度可见，
wool surface nap texture, heavy drape, visible fabric thickness at folds
```
- 追加关键词：`wool nap directional texture`, `heavy fabric drape physics`

##### 机能/科技面料/Techwear
```
面料哑光质感，版型简约利落，
缝线处有微张力褶皱，拉链/卡扣金属件独立反光，
matte technical fabric, seam tension creases, hardware specular highlights
```

#### 通用增强词库

| 英文 | 中文 | 效果 |
|------|------|------|
| `fabric weave visible, thread-level detail` | 面料编织纹理可见 | 非光滑平面 |
| `natural wear and crease patterns` | 自然磨损折痕 | 非全新完美 |
| `cloth catching light differently on folds` | 褶皱不同角度反光差异 | 立体感 |
| `seam and stitch detail visible` | 缝线针脚可见 | 工艺质感 |
| `fabric draping physics natural` | 面料自然垂坠物理 | 符合重力 |
| `edge fray and natural hem irregularity` | 边缘自然毛边/卷边 | 真实做工 |

## 三、整体增强魔咒（One-Liner）

### 通用真实感增强（源自Reddit 114赞验证）

在提示词中追加以下一句话可整体提升真实感：

```
Include natural imperfections: flyaway hairs, slight skin blemishes, 
uneven fabric texture, subtle wrinkles, real light scattering effects
```

中文版：
```
包含自然不完美细节：飞散碎发、轻微皮肤瑕疵、不均匀面料纹理、
细微皱纹、真实光线散射效果
```

### 摄影质感增强（源自博客园高频组合）

```
8K超高清，photorealistic，真实皮肤纹理，发丝蓬松有空气感，
胶片颗粒质感，轻微暗角，保留轻微胶片颗粒感，自然噪点
```

## 四、场景适配矩阵

### 文戏/对话场景

重点：表情微动+眼睛细节+面部肤质

```
追加模块：皮肤高配 + 眼睛高配 + 头发通用
景别优先：近景/特写 → 模块全开
```

### 打斗/动作场景

重点：服装物理+头发甩动+汗水反光（皮肤精简）

```
追加模块：服装增强（按材质选）+ 头发通用（飞丝/甩动）+ 皮肤基础
景别注意：全景/中景 → 皮肤眼睛精简，服装头发重点写
          特写切镜 → 追加汗水/皮肤高配
```

### 情绪特写/高潮

重点：眼睛全部词库 + 皮肤最高配 + 微表情

```
追加模块：皮肤特写词库 + 眼睛高配+眼神情绪 + 头发飞丝
```

### 开场/定场（远景/全景）

```
无需追加质感模块。
目光影方向、空间关系、人物轮廓和位移。
```

## 五、使用方式

### 方式一：嵌入角色质感尾缀（推荐·日常使用）

将所需增强模块的关键词植入【人物】段末尾的角色质感尾缀中。

```
【人物】
秦轩（画左，侧身站立，右手持剑）服装面料细节拉满，褶皱丰富，
层次分明，布料肌理清晰... [原有尾缀全部保留] ...
皮肤毛孔纹理全程保留，肤质前后帧保持一致，不会自动磨皮，
subsurface scattering次表面散射光感...[皮肤高配追加]...
眼神灵动，双眼均有眼神光，虹膜纹理可见...[眼睛高配追加]...
发丝根根分明，碎发自然飘动，飞丝在侧光中可见...[头发追加]...
8K高清，肤质统一
```

### 方式二：写入【要求】层（全局生效）

在 `1.【要求】` 行中加入全局质感要求：

```
1.【要求】不要字幕.不要背景音乐.人物无变形.禁止广角.实拍.
全画幅手持+稳定器混合.真人写实，电影质感，高对比度.
皮肤毛孔纹理/发丝分明/眼神灵动/面料高质感.
包含自然不完美细节：飞散碎发、轻微皮肤瑕疵、面料纹理、真实光散射.
```

### 方式三：分镜内局部追加（精准控制）

在特定镜头的内容段追加质感关键词：

```
镜头3（8-11秒）：特写秦轩面部
景别：ECU大特写
角度：侧面平拍
镜头：固定镜头
内容：秦轩侧脸，汗水沿颞部滑落，太阳穴微血管隐现，
皮肤毛孔纹理在侧光下清晰立体，眼白微血管隐约可见，
虹膜纹理在暗光中仍保留细节 `秦轩咬牙低声道：...`
光效：单侧柔光在颧骨形成自然高光，阴影侧皮肤纹理无损
```

## 六、避坑指南

| 常见问题 | 原因 | 解法 |
|---------|------|------|
| 远景写皮肤毛孔 | 模型看不到，浪费token | 全景/远景只写轮廓和光影 |
| 全部镜头用高配 | token爆炸，冲突 | 按场景适配矩阵选模块 |
| 写"完美无瑕皮肤" | 触发塑料感 | 写"自然微纹理，轻微肤色不均" |
| 头发写"整齐柔顺" | AI给你戴假发 | 写"碎发/飞丝/毛躁/松散" |
| 服装写"全新整洁" | 无折痕无阴影 | 写"自然褶皱/磨损纹理/折痕光影" |
| 眼睛只写"有神" | 太抽象模型不懂 | 写"眼神光/虹膜纹理/湿润线" |
| 肤质词与景别不匹配 | 特写不写细节，全景写太多 | 对照景别可见粒度表 |

### 景别可见粒度速查

| 景别 | 能写 | 不能写 |
|------|------|--------|
| 远景/全景 | 身体轮廓、移动轨迹、空间关系、光影方向 | 表情、眼神、皮肤纹理、牙关、嘴唇 |
| 中景 | 腰部以上、手势、脸上大致表情（笑/怒/惊） | 眼球血丝、汗珠、皮肤毛孔 |
| 近景 | 胸部以上、表情细节、眼神方向、嘴型、肤质 | 远处环境、多人站位、大场面 |
| 特写 | 面部肌肉、眼部微动、汗珠、皮肤纹理、毛孔 | 环境全貌、空间关系、其他人 |
| 大特写 | 眼睫毛、虹膜纹理、嘴唇纹理、单颗汗珠 | 同上，画面中不存在 |

## 八、快速决策流

```
开始写提示词
  ↓
[景别是什么？]
  ↓
远景/全景 → 只写轮廓和光影，不追加质感模块
  ↓
中景 → 基础版皮肤 + 通用头发 + 服装按材质
  ↓
近景 → 高配版皮肤 + 眼睛高配 + 头发通用 + 服装按材质
  ↓
特写/大特写 → 特写词库全开（皮肤/眼睛/头发各选2-3个最强词）
  ↓
[场景类型？]
  ↓
文戏/对话 → 眼睛优先
打斗/动作 → 服装物理+头发甩动优先
情绪高潮 → 眼睛+皮肤最高配
```
