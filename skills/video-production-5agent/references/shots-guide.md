## 📐 一、镜头尺寸 (Shot Size)

镜头尺寸决定观众与主体的心理距离,是画面构成的基础。

### 完整尺寸表 (从远到近)

| 缩写 | 英文 | 中文 | 画面范围 | 心理效果 | 使用场景 |
|------|------|------|----------|----------|----------|
| **EWS** | Extreme Wide Shot | 极远景 | 主体在环境中很小 | 渺小、史诗、孤独 | 开场建立、结尾收束、展示壮观环境 |
| **WS/LS** | Wide/Long Shot | 远景 | 全身+大量环境 | 客观、交代 | 展示人物与环境关系、场景转换 |
| **FS** | Full Shot | 全身镜头 | 头到脚,少量环境 | 展示全貌 | 全身动作、多人场景、服装展示 |
| **MWS** | Medium Wide Shot | 中远景 | 膝盖以上 | 过渡、平衡 | 动作+环境平衡、走路场景 |
| **CS** | Cowboy Shot | 牛仔镜头 | 大腿中部以上 | 展示腰部 | 西部片拔枪、展示腰带/武器 |
| **MS** | Medium Shot | 中景 | 腰部以上 | 日常、交流 | 对话场景标准镜头、最常用 |
| **MCU** | Medium Close-Up | 中近景 | 胸部以上 | 亲近、关注 | 重要对话、手势表达、保持距离感的亲密 |
| **CU** | Close-Up | 特写 | 面部 | 情绪、紧张 | 情感高潮、反应、重要时刻 |
| **ECU** | Extreme Close-Up | 极特写 | 眼睛/嘴/细节 | 强调、压迫 | 关键细节、心理压迫、微表情 |

### 尺寸选择决策树

```
要表达什么?
    │
    ├─ 史诗/壮观/孤独 → EWS极远景
    │
    ├─ 交代环境/人物关系 → WS远景
    │
    ├─ 全身动作/服装 → FS全身
    │
    ├─ 日常对话/动作 → MS中景
    │
    ├─ 重要对话/亲密 → MCU中近景
    │
    ├─ 情绪/反应/紧张 → CU特写
    │
    └─ 极度强调/心理压迫 → ECU极特写
```

### 尺寸节奏原则

**1. 熟悉度递进**
```
初见 → 远景 (保持距离)
熟悉 → 中景 (正常交流)
亲密 → 特写 (深入情感)
```

**2. 避免单调**
- ❌ 连续5个中景
- ✅ 远景→中景→特写→中景

**3. 情绪升级配合景别收紧**
```
平静 → MS中景
紧张 → MCU中近景
高潮 → CU特写
爆发 → ECU极特写
```

## 📷 二、镜头角度 (Camera Angles)

角度决定观众对主体的心理感受——权力、脆弱、平等。

### 完整角度表

| 英文 | 中文 | 摄像机位置 | 心理效果 | 使用场景 |
|------|------|------------|----------|----------|
| **Eye Level** | 平视 | 与眼睛同高 | 中立、平等、真实 | 日常对话、中性呈现 |
| **Shoulder Level** | 肩高 | 与肩膀同高 | 更电影感 | 大部分电影实际使用(比平视更常见) |
| **Low Angle** | 仰拍 | 低于眼线向上 | 权力、威胁、崇高 | 反派登场、英雄时刻、建立威严 |
| **High Angle** | 俯拍 | 高于眼线向下 | 弱小、脆弱、俯视 | 受害者、失败时刻、展示孤立 |
| **Hip Level** | 腰高 | 与腰部同高 | 展示腰部动作 | 西部片拔枪、掏口袋 |
| **Knee Level** | 膝高 | 与膝盖同高 | 轻度优越感 | 走路场景、潜行 |
| **Ground Level** | 地面 | 地面高度 | 神秘、不露脸 | 脚步特写、神秘人物 |
| **Dutch Angle** | 荷兰角/倾斜 | 画面倾斜 | 不安、疯狂、紧张 | 精神状态不稳、恐怖/悬疑 |
| **Overhead** | 俯瞰/鸟瞰 | 正上方90度 | 上帝视角、全局 | 战争场面、布局展示 |
| **Aerial** | 航拍 | 高空 | 史诗、壮观 | 开场、大场面、地标展示 |

### 角度组合技巧

**对话场景权力对比:**
```
角色A (强势): 仰拍 Low Angle
角色B (弱势): 俯拍 High Angle
→ 形成视觉上的权力差异
```

**荷兰角使用原则:**
- ✅ 角色精神状态不稳定
- ✅ 紧张悬疑高潮时刻
- ✅ 主观混乱感
- ⚠️ 慎用!容易显得做作,用在关键时刻

## 🎬 三、镜头运动 (Camera Movement)

运动为画面增加动态感和情绪节奏。

### 完整运动表

| 中文 | 英文 | 运动方式 | 效果 | 使用场景 |
|------|------|----------|------|----------|
| **推进** | Push in / Dolly in | 镜头向前移动 | 聚焦、强调、进入情绪 | 强调细节、情绪升级、重要时刻 |
| **拉远** | Pull back / Dolly out | 镜头向后移动 | 揭示、疏离、结束 | 展示环境、结束场景、揭示真相 |
| **平移** | Pan | 水平转动(机位不动) | 跟随、扫视 | 跟随移动、展示宽度、扫描场景 |
| **摇镜** | Tilt | 垂直转动(机位不动) | 展示高度 | 从上到下或相反、展示建筑高度 |
| **横移** | Track / Dolly | 平行于主体移动 | 伴随、跟踪 | 跟随角色走路、展示空间 |
| **跟拍** | Follow / Tracking | 跟随主体移动 | 紧跟、沉浸 | 运动场景、追逐、保持主体在画面中 |
| **环绕** | Orbit / Arc | 绕主体旋转 | 360度展示、炫技 | 全方位展示、英雄时刻、产品展示 |
| **升降** | Crane / Boom | 垂直升降 | 史诗、揭示 | 大场面、开场、揭示全景 |
| **手持** | Handheld | 手持摄像机 | 真实、紧张、纪实 | 战争片、纪录片、追逐场景 |
| **稳定器** | Steadicam | 稳定器跟拍 | 流畅、沉浸 | 一镜到底、跟随长镜头 |

### 特殊运动技巧

| 技巧 | 英文 | 说明 | 效果 |
|------|------|------|------|
| **一镜到底** | One-take / Oner | 无剪辑连续拍摄 | 真实感、时间连贯、技术炫技 |
| **希区柯克变焦** | Dolly Zoom / Vertigo | 推进+焦距拉远(或相反) | 眩晕、不安、空间扭曲 |
| **甩镜** | Whip Pan | 快速水平甩动 | 转场、时间跳跃、惊吓 |
| **冲击变焦** | Crash Zoom | 快速推进变焦 | 强调、惊讶、70年代风格 |

## 🖼️ 四、镜头构图 (Shot Framing)

如何在画面中安排主体的位置和关系。

### 按人数分类

| 类型 | 英文 | 说明 | 用途 |
|------|------|------|------|
| **单人镜头** | Single Shot | 画面只有一个主体 | 独白、内心戏、特写 |
| **双人镜头** | Two Shot | 两人同框 | 对话、关系展示 |
| **三人镜头** | Three Shot | 三人同框 | 三人对话、三角关系 |
| **群戏镜头** | Group Shot | 多人同框 | 团队、群众场面 |

### 干净 vs 脏

| 类型 | 英文 | 说明 | 用途 |
|------|------|------|------|
| **干净单人** | Clean Single | 画面只有一人,无遮挡 | 强调孤立、专注 |
| **脏单人** | Dirty Single | 单人但有其他元素入画 | 保持空间关系、对话场景 |

### 特殊构图

| 类型 | 英文 | 说明 | 用途 |
|------|------|------|------|
| **过肩镜头** | Over-the-Shoulder (OTS) | 从一人肩后拍另一人 | 对话标配、保持空间关系 |
| **主观镜头** | Point of View (POV) | 角色视角看出去 | 代入感、悬疑、恐怖 |
| **反应镜头** | Reaction Shot | 拍角色对事件的反应 | 情绪传递、强调重要性 |
| **插入镜头** | Insert Shot | 特写某个重要细节 | 强调道具、手部动作、关键物品 |
| **对称构图** | Symmetrical | 画面左右对称 | 秩序、压迫、库布里克风格 |
| **一点透视** | One-point Perspective | 线条汇聚到中心一点 | 纵深感、引导视线、库布里克风格 |

## 🎯 五、Seedance提示词应用

### 镜头尺寸提示词

```
极远景: "extreme wide shot, tiny figure in vast landscape, epic scale"
远景: "wide shot, full body with environment, establishing"
全身: "full shot, head to toe, showing full costume"
中景: "medium shot, waist up, conversational framing"
中近景: "medium close-up, chest up, intimate but distanced"
特写: "close-up shot, face filling frame, emotional"
极特写: "extreme close-up, eyes only, intense detail"
```

### 镜头角度提示词

```
平视: "eye level shot, neutral perspective"
仰拍: "low angle shot, looking up, powerful, dominant"
俯拍: "high angle shot, looking down, vulnerable, small"
荷兰角: "dutch angle, tilted frame, unsettling, disoriented"
鸟瞰: "overhead shot, bird's eye view, god perspective"
航拍: "aerial shot, drone view, sweeping landscape"
```

### 镜头运动提示词

```
推进: "dolly in, push in, camera moving forward, intensifying"
拉远: "pull back, dolly out, revealing environment"
平移: "pan shot, horizontal camera turn, scanning"
跟拍: "tracking shot, following subject, steadicam"
环绕: "orbit shot, circling around subject, 360 view"
一镜到底: "one-take, oner, continuous shot, no cuts"
希区柯克变焦: "dolly zoom, vertigo effect, disorienting"
```

### 构图提示词

```
过肩: "over-the-shoulder shot, OTS"
主观: "POV shot, first person view, subjective"
双人: "two shot, both characters in frame"
对称: "symmetrical composition, centered, Kubrick style"
一点透视: "one-point perspective, lines converging"
```

## 📊 六、速查对照表

### 情绪→镜头映射

| 情绪/效果 | 推荐尺寸 | 推荐角度 | 推荐运动 |
|----------|----------|----------|----------|
| 史诗/壮观 | EWS极远景 | 航拍/俯瞰 | 升降/环绕 |
| 权力/威严 | MS-CU | 仰拍 | 推进 |
| 脆弱/渺小 | WS/FS | 俯拍 | 拉远 |
| 紧张/悬疑 | MCU-ECU | 荷兰角 | 手持/推进 |
| 亲密/情感 | CU-ECU | 平视 | 缓慢推进 |
| 日常/中性 | MS | 肩高/平视 | 静止/轻微跟拍 |
| 动作/追逐 | MS-MWS | 变化多端 | 跟拍/手持 |
| 揭示/惊喜 | 从CU到WS | 变化 | 快速拉远 |

### 场景类型→镜头组合

| 场景类型 | 建议组合 |
|----------|----------|
| **对话场景** | MS中景 + OTS过肩 + CU特写(反应) |
| **追逐场景** | WS远景(路线) + MS中景(跟拍) + 手持 |
| **情感高潮** | MS→MCU→CU→ECU递进 |
| **环境建立** | EWS航拍 → WS远景 → FS全身 |
| **产品展示** | FS全身 + 环绕 + ECU细节 |
| **人物登场** | WS远景 → 推进到CU特写 |
