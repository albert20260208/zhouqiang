# 视频制作经验总结 (2026-02-25 实战提炼)

基于贾维斯出场视频制作过程的经验总结。

---

## ⚠️ 常见问题与避坑指南

### 1. 图片与视频比例匹配
- ❌ 错误: 3:1宽图 → 视频拒绝(宽高比需在0.4-2.5之间)
- ✅ 正确: 16:9 (2848x1600) → 完美适配视频
- **教训**: 图生视频前,先确认图片比例

### 2. 模型ID要准确
- ❌ `doubao-seedream-5.0-lite` (文档里的名字)
- ✅ `doubao-seedream-5-0-260128` (实际调用ID)
- **教训**: 从控制台代码示例复制,不要猜

### 3. 多人场景的人物高度比例 ⭐重要
- **问题**: 多人同框时,身高比例不协调(主角太高或太矮)
- **解决**: 提示词中明确写入人物高度比例
- **示例**:
  ```
  主角身高约175cm,与周围行人身高比例协调一致
  人群中主角不突兀,自然融入环境
  前景人物与背景人物保持透视比例正确
  ```
- **检查点**: 生成后检查人物大小是否合理

### 4. 视频时长限制
- Seedance 1.5 Pro: 5秒或10秒
- **策略**: 复杂运镜用5秒,避免失败

### 5. 图生视频的平衡
- 提示词太多 → 可能偏离参考图
- 提示词太少 → 动态不够
- **平衡点**: 明确运镜+关键动态,其他依赖参考图

## ✅ 成功要点

### 1. 场景一致性控制
**方法**: 图生图/图生视频 + 参考图输入

**关键词**: 
```
保持参考图中XXX完全一致
保持参考图中的场景、建筑、色调、霓虹风格高度完全一致
```

### 2. 提示词结构化
借鉴官方示例的分块结构效果更好:
```
【场景】具体环境描述
【人物/动态】角色和动作(含身高比例)
【运镜】镜头运动方式
【风格】导演风格参考
【色彩与光线】色调+光效
【质感】渲染方式+分辨率
```

### 3. 运镜描述要具体
- ❌ 模糊: "镜头移动"
- ✅ 清晰: "镜头快速斜向上拉升爆发,如火箭般冲天,掠过建筑侧面"
- **关键**: 动作+速度+路径+效果

### 4. 导演风格融入
**王家卫风格关键词**:
- 抽帧跳跃感
- 霓虹色彩流转(红蓝绿粉紫)
- 高对比度光影
- 胶片颗粒质感
- 浅景深背景虚化

### 5. 细节丰富化
- 街道场景加入: 行人+雨伞+AR眼镜+小贩+飞行器
- 环境动态: 霓虹闪烁+数据流+云层飘动+雨后反射
- **效果**: 画面更有生命力

## 🎯 提示词技巧

### 技巧1: 三段式描述
```
1. 开场画面 (1-2秒)
2. 运镜过程 (核心动作)
3. 结尾定格 (最后1-2秒)
```

### 技巧2: 运镜+场景动态分开
```
【运镜】镜头如何移动
【场景动态】场景内元素如何变化(霓虹闪烁、人群移动)
```

### 技巧3: 镜头语言术语
| 中文 | 英文 | 效果 |
|------|------|------|
| 升降 | Crane up/down | 史诗感、揭示 |
| 推进 | Dolly in / Push in | 聚焦、强调 |
| 拉远 | Pull back / Dolly out | 揭示全景 |
| 环绕 | Orbit | 360度展示 |
| 一镜到底 | One-take | 流畅、真实感 |
| FPV视角 | FPV drone | 穿越感、沉浸 |

### 技巧4: 视觉冲击关键词
- 史诗感: epic scale, sweeping
- 震撼: stunning, breathtaking
- 流畅: smooth, fluid
- 加速: accelerating, rushing
- 电影级: cinematic, film-grade

### 技巧5: 人物比例关键词 ⭐
```
人物高度比例协调一致
主角与周围人物身高自然
透视比例正确
人物在画面中的相对大小合理
```

## 🔄 工作流总结

```
1. 构思故事/场景
   ↓
2. 生成16:9参考图 (Seedream 5.0)
   - 尺寸: 2848x1600 (16:9适配视频)
   - 结构化提示词
   - 多人场景注意身高比例
   ↓
3. 图生视频 (Seedance 1.5 Pro)
   - 引用参考图
   - 明确运镜(动作+速度+路径)
   - 场景一致性要求
   ↓
4. 检查与迭代
   - 检查人物比例
   - 检查运镜是否符合预期
   - 需要则调整提示词重新生成
   ↓
5. 多段拼接(如需要)
   - 保持场景一致
   - 运镜衔接自然
```

## 📊 推荐尺寸参数

### 图片 (Seedream 5.0)
| 用途 | 尺寸 | 比例 |
|------|------|------|
| 视频首帧 | 2848x1600 | 16:9 |
| 竖屏视频 | 1600x2848 | 9:16 |
| 正方形 | 2048x2048 | 1:1 |

### 视频 (Seedance 1.5 Pro)
| 参数 | 推荐值 |
|------|--------|
| duration | 5 (复杂运镜) / 10 (简单运镜) |
| camerafixed | false (运镜) / true (固定) |
| watermark | false (无水印) |
