# 人物提示词指南

## 人物提示词结构

### 标准结构
```
[主体] + [外貌] + [表情] + [服装] + [姿态] + [环境] + [光影] + [风格] + [质量]
```

### 示例
```
一位23岁年轻女性，
外貌：长发及腰，乌黑亮丽，瓜子脸，大眼睛，皮肤白皙，
表情：冷漠高傲，嘴角微微上扬带着不屑，
服装：白色高定礼服，露肩设计，裙摆拖地，佩戴钻石项链，
姿态：双手抱胸，微微仰头，居高临下地看着前方，
环境：豪华宴会厅，水晶吊灯，
光影：侧光，轮廓光，
风格：现代都市，霸总短剧风格，
质量：8K高清，电影级画质
```

## 人物一致性控制

### 外貌一致性
```
固定描述：
- 年龄：[固定数字]
- 身高：[固定数字]
- 发色：[固定颜色]
- 瞳色：[固定颜色]
- 特征痣/疤：[固定位置]

变化描述：
- 服装：根据场景变化
- 表情：根据情绪变化
- 姿态：根据动作变化
```

### 服装一致性
```
建立服装档案：
- 日常装：[详细描述]
- 工作装：[详细描述]
- 礼服：[详细描述]
- 特殊场景：[详细描述]

使用提示：
"wearing her signature [服装名称], [详细描述], 
same as previous scenes"
```

### 表情一致性
```
人物性格决定表情范围：
- 高冷型：表情变化小，微表情为主
- 活泼型：表情丰富，变化大
- 内敛型：表情含蓄，眼神传递情绪
```

## 提示词优化技巧

### 1. 负面提示词
```
避免的问题：
- "deformed, bad anatomy, extra limbs"
- "blurry, low quality, pixelated"
- "inconsistent features, changing appearance"
- "modern elements in ancient setting (unless intended)"
- "wrong hairstyle for era"
```

### 2. 权重调整
```
强调重要特征：
- (blue eyes:1.3) - 强调蓝眼睛
- [modern clothing:0.5] - 弱化现代服装
- {{ancient hairstyle}} - 强调古代发型
```

### 3. 风格统一
```
添加风格词：
- "consistent character design"
- "same person, different pose"
- "character reference sheet"
- "maintain facial features across images"
```

## 实操检查清单

### 人物设定检查
- [ ] 年龄、身高、外貌特征是否明确？
- [ ] 人物性格标签是否清晰？
- [ ] 服装风格是否符合身份？
- [ ] 表情范围是否符合性格？

### 提示词质量检查
- [ ] 描述是否具体可执行？
- [ ] 是否有负面提示词避免畸形？
- [ ] 风格是否统一？
- [ ] 质量词（8K, photorealistic）是否添加？

### 一致性检查
- [ ] 同一人物在不同场景是否一致？
- [ ] 服装是否随场景合理变化？
- [ ] 表情是否符合情绪但不改变基本面貌？
- [ ] 是否建立了人物档案方便复用？

---

**使用建议**：根据人物类型选择对应模板，填入具体细节，使用检查清单确保质量。
