# 主流AI模型特性数据库

## 图片生成模型

### Midjourney
- **版本**: v6
- **特点**: 艺术感强，美学优秀，默认风格化较高
- **优化策略**: 强调风格、氛围、光影；服装材质描述很重要
- **提示词结构**: `[主体] + [详细外貌] + [服装/材质] + [场景] + [光线/氛围] + [艺术风格] + [质量词]`
- **有效关键词**: cinematic lighting, photorealistic, 8K, masterpiece
- **注意事项**: 避免过于技术化的参数描述，强调情感和氛围

### Stable Diffusion (SDXL / SD 1.5)
- **特点**: 可控性强，权重调整，需要负面提示词
- **优化策略**: 详细描述，使用权重，结构化提示词
- **提示词结构**: `(Quality tags), (Subject:1.2), (Detailed description), (Style), (Lighting)`
- **权重使用**: (keyword:1.3)强调, [keyword:0.8]弱化

### DALL-E 3
- **特点**: 理解力最强，自然语言描述效果好，逻辑一致性强
- **优化策略**: 自然语言描述，像讲故事，强调逻辑关系
- **提示词结构**: `[场景描述], [主体在做什么], [周围环境], [光线和氛围], [风格要求]`

### 即梦 Seedream 5.0
- **特点**: 中文理解好，本土化，针对亚洲审美优化
- **优化策略**: 使用中文提示词，强调网文/短剧风格
- **有效关键词**: 网文风格, 短剧风格, 电影感, 写实, 古装, 仙气飘飘

## 视频生成模型

### 即梦 Seedance 2.0 / 1.5 Pro
- **特点**: 多模态输入（图+视频+音频+文本），@引用系统强大，本土化
- **输入限制**: 图片≤9张，视频≤3个，音频≤3个，总文件≤12
- **输出**: 4-15秒，480p-720p
- **限制**: 不支持写实真人脸部
- **参见**: `references/seedance-guide.md`, `references/seedance-zh-guide.md`

### Runway Gen-3
- **特点**: 质量最高，运动流畅，支持Motion Brush
- **优化策略**: 强调动作描述，分时段描述运动，使用Motion Brush

### Pika Labs
- **特点**: 特效强，扩展性好，支持Modify Region
- **优化策略**: 强调特效和转场，适合快速迭代

### Luma Dream Machine
- **特点**: 真实感强
- **优化策略**: 强调真实感、物理规律

### Vidu
- **特点**: 国产模型，16:9画幅，武侠片效果好
- **优化策略**: 中文提示词，青色主色调，电影感武侠风格

## 视频模型核心提示词策略

### Seedance（胡导风格）
参见 `references/hu-director-style.md` 完整8部分格式：
```
不要字幕
【场景】...@场景ID
【时间】...
【人物】...@人物ID
【道具】...@道具ID
【位置】...
【声音】...@音频ID
【分镜设计】...
```

### Seedance @引用系统
```
@Image1 作为首帧 / @Image2 作为尾帧
@Image1 的人物作为主体 / 场景参考 @Image3
参考 @Video1 的运镜 / 完全参考 @Video1 的特效和转场
背景BGM参考 @Audio1 / 穿着 @Image2 的服装
```

### 分时段描述（10秒以上推荐）
```
0-3秒：[开场画面描述、运镜、动作]
3-6秒：[中段发展]
6-10秒：[高潮或关键动作]
10-15秒：[收尾、定格画面、品牌文字]
```

### 画质增强（Seedance）
在提示词开头加上：`画质强化:4K直出+极致细节+锐利焦点`
生成后先点HD按钮重绘，导出选尺寸倍率2x。
