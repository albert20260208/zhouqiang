# 胡导风格标准格式指南

## 概述
胡导风格是专门为即梦（Seedance）视频生成设计的标准提示词格式，特点：全部固定镜头，15秒不超过9个镜头。

## 标准格式模板

```
不要字幕
【场景】...@场景ID
【时间】...
【人物】
@人物1（特征描述）
@人物2（特征描述）
【道具】...@道具ID
【位置】...
【声音】@音频ID1（描述），@音频ID2（描述），环境音效
【分镜设计】
镜头1（0-2秒）：
景别：中景
角度：正面
运镜：固定镜头
内容：...，台词："..."

镜头2（2-4秒）：
景别：特写
角度：侧面
运镜：固定镜头
内容：...，台词："..."

...（最多9个镜头）
```

## 核心规则

### 1. 格式结构（必须8部分）
1. **开头**：`不要字幕`
2. **场景**：描述场景+@引用场景ID
3. **时间**：时间设定（白天/夜晚/具体时间）
4. **人物**：列出所有人物+@引用人物ID+特征描述
5. **道具**：列出道具+@引用道具ID
6. **位置**：位置描述
7. **声音**：音频引用+环境音效
8. **分镜设计**：镜头详细描述

### 2. ⚠️ 镜头设计要求（2026-05-25 更新）
**胡导风格：全部使用固定镜头** 这个规则仅适用于 **胡导本人的短剧风格**（文戏/对话/节奏紧凑的15秒片段）。

**⚠️ 对于有动作/打斗/冲突升级/追逐等动态场面，不能死守固定镜头！**
用动态运镜（手摇、跟拍、环绕、POV/FPV、缓推）来传递力量感和冲击速度。

详见 `production-tips.md` →「动态场面运镜规则」章节。

- **数量**：15秒不超过9个镜头
- **描述**：每个镜头必须包含：
  - 景别（远景/中景/近景/特写）
  - 角度（正面/侧面/背面/俯拍/仰拍）
  - 运镜（根据场景类型选择固定或运动）
  - 内容（动作+表情+台词）
- **时长**：每个镜头1-3秒，总时长15秒

### 3. @引用系统
- **场景引用**：`@场景ID`
- **人物引用**：`@人物ID（特征描述）`
- **道具引用**：`@道具ID`
- **音频引用**：`@音频ID（描述）`

### 4. 台词标注
- **直接台词**：`台词："..."`
- **内心台词**：`内心台词："..."`
- **环境音效**：在内容中描述（脚步声、开门声等）

## 示例：公司总裁紧急停止合作场景

```
不要字幕
【场景】公司前台区域@公司前台ID，电梯间@电梯间ID
【时间】白天，商务工作时间
【人物】
@男主（面容急切，西装革履）
@助理（表情紧张，职业装）
@前台（表情专注，职业装）
【道具】前台电脑、电梯按钮、文件夹、电梯
【位置】前台区域→电梯间
【声音】@音频ID1（前台声音），@音频ID2（助理声音），@音频ID3（总裁声音），电梯提示音，急促脚步声
【分镜设计】
镜头1（0-2秒）：
景别：中景
角度：正面
运镜：固定镜头
内容：前台工作人员低头看了一眼电脑，表情专注，内心台词："He is the president of Liandong Company, Sun Xun."

镜头2（2-4秒）：
景别：中景
角度：侧面
运镜：固定镜头
内容：两人站在前台前，助理说话，台词："He is a supplier of the your branch. A small company."

镜头3（4-6秒）：
景别：特写
角度：正面
运镜：固定镜头
内容：前台视角，男主转身急切离开，台词："Stop all cooperation with Sun Xun immediately."

镜头4（6-8秒）：
景别：中景
角度：侧面
运镜：固定镜头
内容：助理急忙跟上，点头回应，台词："Yes."，表情紧张

镜头5（8-10秒）：
景别：远景
角度：背面
运镜：固定镜头
内容：男主和助理一前一后从远处来到电梯间，背影，急促的脚步声

镜头6（10-12秒）：
景别：特写
角度：低角度
运镜：固定镜头
内容：背影男主着急的按电梯，手部特写，急促的呼吸声

镜头7（12-14秒）：
景别：中景
角度：正面
运镜：固定镜头
内容：电梯到达楼特写，电梯门打开，电梯提示音"叮"

镜头8（14-15秒）：
景别：中景
角度：侧面
运镜：固定镜头
内容：男主和助理进电梯，背影，电梯门开始关闭
```

## 常见场景模板

### 1. 商务冲突场景
- 场景：公司环境
- 情绪：紧张、急切
- 节奏：快速切换
- 声音：对话声、脚步声、环境音

### 2. 情感冲突场景
- 场景：咖啡店、酒店
- 情绪：愤怒、厌恶、痛苦
- 节奏：中等偏快
- 声音：对话声、环境音、音效

### 3. 动作冲突场景
- 场景：房间、走廊
- 情绪：愤怒、暴力、震惊
- 节奏：快速激烈
- 声音：击打声、倒地声、喘息声

## 使用流程
1. 确认场景需求
2. 准备@引用资源（图片、音频）
3. 按照8部分结构填写
4. 设计8-9个固定镜头
5. 添加台词和音效描述
6. 生成即梦提示词

## 注意事项
1. 严格遵守格式，不要缺项
2. 全部使用固定镜头
3. 15秒不超过9个镜头
4. 台词要明确标注
5. @引用要准确对应已有资源
