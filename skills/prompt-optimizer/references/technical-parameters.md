## 一、曝光三角（Exposure Triangle）

曝光由三个变量共同决定，改变其一必须调整另外两个以保持亮度不变。

### 1. 光圈（Aperture）

**定义**：镜头孔径大小，控制进光量和景深。

**F-stop 级数（每档进光量差一倍）**：
```
F1.0 → F1.4 → F2.0 → F2.8 → F4.0 → F5.6 → F8.0 → F11 → F16 → F22
  ←—— 光圈越大（数值越小），进光越多，景深越浅 ——→
  ←—— 光圈越小（数值越大），进光越少，景深越深 ——→
```

**各档位的影像属性**：

| F值 | 景深 | 进光量 | 适用场景 | 跟焦难度 |
|-----|------|--------|---------|---------|
| F1.0-F1.4 | 极浅，对焦点外全虚化 | 极大 | 暗光人像、梦幻虚化、夜景 | 极难 |
| F2.0-F2.8 | 浅，背景明显虚化 | 大 | 电影人像、近景对话、特写 | 难 |
| F4-F5.6 | 中等，背景可辨 | 中 | 中景、环境人像、纪录片 | 适中 |
| F8-F11 | 深，前后景清晰 | 小 | 风景、全景、建筑、大场面 | 易 |
| F16-F22 | 极深，全画面锐利 | 极小 | 强光大场面、追求全景锐度 | 极易 |

**画质注意**：
- 全开光圈（F1.4-F2.8）→ 边缘锐度下降、暗角出现、可能有色散（紫边）
- 最佳光学区间（F5.6-F8）→ 中心到边缘最佳锐度
- 小光圈（F11+）→ 衍射开始影响整体锐度
- 极小光圈（F16+）→ 点光源产生星芒效果

### 2. 快门 / 快门角度（Shutter Speed / Shutter Angle）

**摄影用快门速度**（秒/分之一秒）：
```
1/8000 → 1/4000 → 1/2000 → 1/1000 → 1/500 → 1/250 → 1/125 → 1/60 → 1/30 → 1/15 → 1/8
  ←—— 越快，冻结运动越强，进光越少 ——→
  ←—— 越慢，运动模糊越多，进光越多 ——→
```

**电影用快门角度**（基于24fps）：

| 快门角度 | 等效快门速度 | 动态模糊量 | 适用场景 |
|---------|------------|-----------|---------|
| 45° | 1/192s | 极少，动作割裂 | 《拯救大兵瑞恩》战斗场面 |
| 90° | 1/96s | 少，动作锐利 | 动作片/打斗高潮 |
| **180°** | **1/48s** | **自然，标准电影感** | **绝大多数电影（行业默认）** |
| 270° | 1/32s | 多，拖影明显 | 梦境/回忆/醉酒主观镜头 |
| 360° | 1/24s | 极大，光轨拖尾 | 极致梦幻/失重感 |

**180° 快门法则**：快门角度 = 180° 时，动态模糊量与人眼自然感知最接近。这是电影行业默认标准。改变快门角度是风格化选择，不是技术失误。

**打斗场景快门选择**：
- 标准打斗 → 180°（1/48s），自然的动作模糊让观众觉得流畅
- 高速砍杀/想要余影轨迹感 → 270°（1/32s），剑光拖影更长
- 碎片飞溅/震动瞬间想强调冲击 → 90°（1/96s），碎片不糊，打击感锐利

### 3. ISO / 感光度

**定义**：传感器对光线的放大倍率。ISO越高画面越亮，但噪点越多。

**级数**：
```
ISO 100 → 200 → 400 → 800 → 1600 → 3200 → 6400 → 12800
  ←—— 每档进光等效翻倍，噪点递增 ——→
```

**各档位表现**：

| ISO | 噪点 | 适用场景 |
|-----|------|---------|
| 100-200 | 几乎无 | 晴天户外、强光场景 |
| 400 | 极轻微 | 阴天户外、室内窗边 |
| 800 | 轻微 | 室内灯光、黄昏 |
| 1600 | 可见 | 夜景、暗光室内 |
| 3200+ | 明显 | 极暗环境，接受噪点换画面 |

**电影摄影的 ISO 策略**：
- 电影机通常有**原生ISO（Native ISO）**：此值下动态范围最大，噪点最低
- 常见原生ISO：800（ARRI Alexa）、800/3200（Sony Venice双原生）
- 户外强光放ND也不提ISO（维持800保证画质），降ISO会损失动态范围
- **Seedance场景默认假设**：ISO 800（电影机原生）

## 二、F-stop vs T-stop

**这是电影摄影和静态摄影的本质区别。**

| 维度 | F-stop | T-stop |
|------|--------|--------|
| 是什么 | 理论几何值：焦距÷孔径直径 | 实测透光率值 |
| 计算 | 纯数学公式 | 实际测量通过镜片组的光通量 |
| 差异原因 | 忽略镜片镀膜/玻璃吸收的光损 | 计入所有光损 |
| 一致性 | 不同镜头F2.8进光量可能差1/3档 | 所有镜头T2.8进光量完全一致 |
| 使用场景 | 静态摄影 | **电影/视频拍摄** |

**实际影响**：
- 两支F2.8镜头，一只10片镜片组+老镀膜，一只6片镜片组+新镀膜 → F值相同但实际画面亮度不同
- 电影组换镜头不重测曝光，靠T-stop保证一致性
- F2.8的镜头实测约等于 T3.0-T3.2（取决于镜片数量和镀膜质量）

**Seedance提示词建议**：写F值或T值都可以，模型能理解。如果强调曝光精确性，用T-stop。一般场景用F-stop足够。

## 三、ND滤镜（Neutral Density Filter）

**定义**：减光但不改变颜色。户外强光下大光圈拍摄的必需品。

**常见ND值**：

| ND标号 | 减光档数 | 透光率 | 适用场景 |
|--------|---------|--------|---------|
| ND0.3 (ND2) | 1档 | 50% | 阴天 F2.8 |
| ND0.6 (ND4) | 2档 | 25% | 薄云晴天 F2.8 |
| ND0.9 (ND8) | 3档 | 12.5% | 晴天 F2.8 |
| ND1.2 (ND16) | 4档 | 6.25% | 强光晴天 F2.8 |
| ND1.5 (ND32) | 5档 | 3.1% | 正午烈日 F2.8 |
| ND1.8 (ND64) | 6档 | 1.6% | 极端强光+大光圈 |

**实用公式**（晴天F2.8，ISO 800，180°快门）：
- 正午直射阳光 ≈ 需ND1.5（减5档）
- 下午3-5点 ≈ 需ND0.9-ND1.2（减3-4档）
- 阴天 ≈ 可能不需要ND，或ND0.3

**Seedance提示词中**：如果指定F2.8+正午阳光→必须加ND或降ISO，否则过曝。

## 四、镜头焦段与透视

### 焦段分类及影像特性

| 类别 | 焦段（全画幅） | 视角 | 透视感 | 变形 | 常见用途 |
|------|--------------|------|--------|------|---------|
| 超广角 | 14-24mm | 114°-84° | 夸张，近大远小剧烈 | 边缘拉伸 | 大场面、空间压迫感 |
| 广角 | 24-35mm | 84°-63° | 明显 | 边缘有 | 定场镜头、环境人像 |
| 标准 | 35-50mm | 63°-46° | 自然 | 几乎无 | 人眼视角、日常拍摄 |
| 中长焦 | 50-85mm | 46°-28° | 轻微压缩 | 无 | 人像、OTS、近景对话 |
| 长焦 | 85-135mm | 28°-18° | 空间压缩明显 | 无 | 特写、情感孤立 |
| 超长焦 | 135mm+ | <18° | 强压缩，背景贴脸 | 无 | 远距离偷窥感、压制 |

### 焦段选择对画面的影响

| 焦段 | 面部表现 | 背景关系 | 运动速度感 |
|------|---------|---------|-----------|
| 24mm | 面部膨胀，鼻子大耳朵小 | 背景远小 | 横向运动显得极快 |
| 35mm | 自然偏宽 | 背景关系正常 | 运动速度自然偏快 |
| 50mm | 最接近人眼 | 正常 | 自然 |
| 85mm | 面部扁平化，五官收敛 | 背景压缩靠近主体 | 横向运动显得慢 |
| 135mm | 面部高度扁平 | 背景紧贴，空间压缩 | 横向运动显得极慢 |

### 电影常用焦段搭配

| 镜头类型 | 推荐焦段（S35等效） | 原因 |
|---------|-------------------|------|
| 全景/定场 | 24-35mm | 交代空间关系 |
| 双人中景 | 35-50mm | 两人同框不变形 |
| OTS过肩 | 50-85mm | 压缩空间，焦点在说话人 |
| 近景/特写 | 85-135mm | 面部不变形，背景虚化自然 |
| 动作/打斗 | 35-50mm | 看得清全身动作，不变形 |

**你的「平角镜头、无变形、禁止广角」对应**：焦段 ≥ 50mm（全画幅）或 ≥ 35mm（S35）。

## 五、传感器画幅与景深换算

### 常见画幅

| 画幅 | 尺寸（mm） | 裁切系数 | F2.8等效景深 |
|------|-----------|---------|-------------|
| 全画幅（FF） | 36×24 | 1.0x | F2.8 |
| Super 35（S35） | 24.9×14.0 | ~1.5x | F4.0等效 |
| APS-C | ~23.6×15.6 | ~1.5x | F4.0等效 |
| M4/3 | 17.3×13.0 | 2.0x | F5.6等效 |

**景深等效公式**：要实现全画幅F2.8的景深 → S35需要F2.0 → M4/3需要F1.4

**Seedance默认假设**：大概率模拟S35画幅（电影行业标准），F2.8在S35上景深比全画幅稍深，跟焦更轻松。

## 六、综合曝光速查表

### 户外场景（ISO 800, 180°快门 + ND补偿）

| 场景 | 推荐光圈 | 需ND | 画幅建议 |
|------|---------|------|---------|
| 正午直射+全景打斗 | F5.6-F8 | ND0.6 | S35 35mm |
| 正午直射+近景表情 | F2.8 | ND1.5 | S35 85mm |
| 晴天下午+中景打斗 | F4-F5.6 | ND0.6-0.9 | S35 50mm |
| 晴天下午+特写 | F2.8 | ND0.9-1.2 | S35 85mm |
| 阴天+全景 | F4-F5.6 | 可能不需要 | S35 35mm |
| 黄昏+近景 | F2.8 | 不需要 | S35 50mm |
| 夜景暗光 | F1.4-F2.0 | 不需要 | 提ISO到1600-3200 |

### Seedance提示词中技术参数的写法

**推荐写法**（直接简洁）：
```
光圈2.8，ISO 800，快门角度180°，ND1.2
```

**不需要写的内容**（模型不懂/冗余）：
- ❌ "按照曝光三角法则计算" — 模型不理解曝光三角这个词
- ❌ "F-stop = 焦距÷有效孔径" — 这是公式不是指令
- ✅ 直接写数字：F2.8、ISO 800、1/48s 或 180°快门

## 七、不同景别的技术参数建议

| 景别 | 推荐焦段 | 推荐光圈 | 注意 |
|------|---------|---------|------|
| 大全景/远景 | 24-35mm | F8-F11 | 深景深让环境清楚 |
| 全景 | 35-50mm | F5.6-F8 | 人物+环境都清晰 |
| 中景 | 50-85mm | F4-F5.6 | 背景可辨认但不抢眼 |
| 近景 | 85mm | F2.8 | 背景虚化，焦点在表情 |
| 特写 | 85-135mm | F2.8 | 极浅景深，焦点在局部 |
| 打斗全景 | 35-50mm | F5.6-F8 | 动作幅度大，深景深保焦 |
| 打斗近景 | 50-85mm | F2.8-F4 | 稍深景深给跟焦留余量 |

## 八、快速决策流

```
拿到一个镜头，按顺序走：

1. 首先要什么感觉？
   → 虚化背景突出主体 → 大光圈 F1.4-F2.8
   → 环境与人物并重 → 中小光圈 F5.6-F8

2. 打斗还是文戏？
   → 打斗 → 光圈不低于F4（给跟焦留余量）
   → 文戏 → 光圈可以全开F2.8

3. 强光还是暗光？
   → 强光+F2.8 → 必须加ND或降ISO
   → 暗光+F2.8 → 可能需要提ISO

4. 全景还是近景？
   → 全景+浅景深 = 不搭 → 收光圈
   → 近景+深景深 = 不搭 → 开光圈

5. 写进提示词：F值 + ISO + 快门角度 + ND（如需要）
```
