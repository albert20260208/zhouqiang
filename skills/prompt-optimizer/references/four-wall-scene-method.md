# 四面墙场景一致性方法

## 核心概念

### 什么是四面墙方法？
- 想象场景是一个房间的四堵墙
- 确定每面墙的位置和特征
- 所有镜头必须遵守这个空间布局
- 确保场景的空间逻辑一致性

### 为什么需要四面墙方法？
- 避免空间逻辑错误（穿帮）
- 保持场景布局一致性
- 确保人物位置关系合理
- 增强场景真实感

## 应用步骤

### 步骤1：绘制场景平面图
```
┌─────────────────┐
│     北墙        │
│  窗户、书架     │
├─────────────────┤
│西墙      │东墙  │
│门、柜子  │沙发、│
│          │茶几  │
├─────────────────┤
│     南墙        │
│  电视、装饰画   │
└─────────────────┘
```

### 步骤2：标注四面墙特征

#### 北墙特征
- **主要元素**: 大窗户、书架
- **窗户细节**: 百叶窗、窗台有绿植
- **书架内容**: 书籍、装饰品、相框
- **光线来源**: 主要自然光源

#### 南墙特征
- **主要元素**: 电视、装饰画
- **电视型号**: 55寸液晶电视
- **装饰画**: 抽象艺术画，金色画框
- **其他**: 电视柜、音响设备

#### 东墙特征
- **主要元素**: 沙发、茶几
- **沙发样式**: L型灰色布艺沙发
- **茶几材质**: 木质，圆形
- **其他**: 地毯、抱枕、遥控器

#### 西墙特征
- **主要元素**: 门、储物柜
- **门类型**: 白色木门，带门把手
- **储物柜**: 三层书架式储物柜
- **其他**: 开关、插座、挂钟

### 步骤3：确定角色位置关系

#### 位置坐标系统
```
(0,0) ──── (10,0)   → X轴（东西方向）
   │          │
   │   场景    │
   │          │
(0,10) ─── (10,10)  ↓ Y轴（南北方向）
```

#### 关键位置点
- **沙发位置**: (3,2) 到 (7,5)
- **茶几位置**: (5,3)
- **电视位置**: (5,8)
- **门位置**: (0,5)
- **窗户位置**: (5,0)

### 步骤4：建立空间逻辑规则

#### 视线规则
- 从沙发看向电视：视线经过茶几
- 从门看向窗户：视线经过沙发区域
- 从书架看向装饰画：对角线视线

#### 移动路径
- 门→沙发：直线距离约3米
- 沙发→电视：直线距离约4米
- 窗户→茶几：对角线距离约5米

#### 遮挡关系
- 沙发遮挡部分电视柜
- 书架遮挡部分窗户
- 茶几不遮挡电视视线

## 镜头设计应用

### 镜头1：全景展示
- **机位**: 西南角，高度2米
- **视角**: 覆盖整个房间
- **可见元素**: 所有四面墙特征
- **目的**: 建立空间认知

### 镜头2：人物对话
- **机位**: 东墙方向，高度1.5米
- **视角**: 沙发区域特写
- **可见元素**: 沙发、茶几、部分电视
- **背景**: 南墙电视和装饰画
- **目的**: 展示对话场景

### 镜头3：人物移动
- **起始位置**: 门（西墙）
- **移动路径**: 西墙→沙发区域
- **机位跟随**: 平移跟拍
- **可见变化**: 从门特征过渡到沙发特征
- **目的**: 展示空间转换

### 镜头4：细节特写
- **焦点**: 茶几上的物品
- **背景虚化**: 沙发和电视
- **可见细节**: 茶几材质、物品摆放
- **空间关系**: 显示与周围家具的距离
- **目的**: 强调细节，保持空间感

## 一致性检查清单

### 空间逻辑检查
- [ ] 人物位置是否符合坐标系统
- [ ] 视线方向是否合理
- [ ] 移动路径是否连贯
- [ ] 遮挡关系是否一致

### 场景元素检查
- [ ] 四面墙特征是否保持不变
- [ ] 家具位置是否固定
- [ ] 物品摆放是否一致
- [ ] 光线方向是否统一

### 镜头连续性检查
- [ ] 前后镜头空间关系是否连贯
- [ ] 人物位置是否合理过渡
- [ ] 背景元素是否保持一致
- [ ] 视线方向是否连续

## 常见问题与解决方案

### 问题1：空间跳跃
**表现**: 人物突然出现在不可能的位置
**解决**: 严格遵守坐标系统和移动路径

### 问题2：元素突变
**表现**: 家具位置或样式突然改变
**解决**: 建立详细的四面墙特征文档

### 问题3：视线混乱
**表现**: 视线方向不符合空间逻辑
**解决**: 绘制视线关系图，确保合理性

### 问题4：比例失调
**表现**: 人物与空间比例不协调
**解决**: 使用统一的尺度参考（如门高2米）

- 南墙: 55寸电视，抽象装饰画，电视柜

光线从北窗射入，形成柔和自然光。

## 进阶技巧

### 1. 动态空间
- 可移动元素（如椅子）标注移动范围
- 临时物品（如手提包）记录出现时间
- 状态变化（如开灯）记录时间点

### 2. 多场景衔接
- 不同场景建立独立坐标系
- 转换时明确空间关系变化
- 使用过渡元素（如门、走廊）连接

### 3. 复杂空间
- 多层空间建立Z轴坐标
- 楼梯、阳台等特殊结构单独标注
- 视线穿透（如透过窗户看外面）单独处理

### 4. 时间变化
- 不同时间段的灯光变化
- 物品位置的时间性变化
- 人物状态的时间连续性

## 总结

四面墙场景一致性方法是确保视频场景空间逻辑合理性的有效工具。通过：

1. **明确空间结构** - 绘制平面图，标注四面墙
2. **建立坐标系统** - 确定位置关系
3. **制定逻辑规则** - 视线、移动、遮挡规则
4. **严格检查执行** - 使用检查清单确保一致性

可以显著提升视频场景的真实感和专业度，避免常见的空间逻辑错误。
