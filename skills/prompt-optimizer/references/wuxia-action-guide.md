# 武侠动作指导

## 动作设计原则

### 1. 起承转合
每个动作有开始、发展、高潮、结束
- **起**: 起手式，蓄势待发
- **承**: 动作展开，招式连绵
- **转**: 变招转折，出其不意
- **合**: 收招定式，气定神闲

### 2. 虚实结合
真实武术 + 夸张特效
- **实**: 真实的武术动作基础
- **虚**: 艺术化的特效表现
- **结合**: 以实为基础，以虚为升华

### 3. 兵器特性
不同兵器的独特表现方式

#### 剑法特点
- 灵动飘逸，变化多端
- 以巧取胜，以快制敌
- 招式: 点、刺、挑、削、劈、撩

#### 刀法特点
- 刚猛霸道，势大力沉
- 以力破巧，一往无前
- 招式: 劈、砍、撩、剁、扫、斩

#### 枪法特点
- 长兵优势，控制距离
- 灵活多变，攻防一体
- 招式: 扎、刺、挞、抨、缠、圈

## 打斗场景设计

### 一对一单挑
```
【起手】双方对峙，气势碰撞
【试探】快速交手，互相试探
【升级】招式渐猛，内力比拼
【高潮】绝招对拼，胜负分晓
【收尾】胜者收招，败者认输
```

### 一对多群战
```
【开场】主角被围，四面楚歌
【破局】找准弱点，逐个击破
【混战】招式频出，眼花缭乱
【突围】杀出重围，扬长而去
```

### 高手对决
```
【气势】未动手先比气势
【试探】虚招试探，寻找破绽
【对拼】实招对拼，内力碰撞
【变招】奇招迭出，出人意料
【决胜】抓住破绽，一击制胜
```

## 特效表现

### 剑气刀光
```
剑气: 白色/蓝色光刃，锐利逼人
刀光: 金色/红色光弧，霸道凌厉
枪芒: 银色光点，迅疾如电
```

### 内力特效
```
真气: 白色/金色气流，环绕周身
气劲: 透明冲击波，扩散开来
护体: 光罩护身，刀枪不入
```

### 轻功特效
```
残影: 快速移动留下残影
踏空: 空中借力，如履平地
飘落: 缓缓飘落，衣袂飘飘
```

## 情绪与动作结合

### 愤怒战斗
```
招式: 大开大合，不顾防守
表情: 怒目圆睁，咬牙切齿
特效: 红色气劲，火焰特效
```

### 冷静战斗
```
招式: 精准高效，以巧破力
表情: 面无表情，眼神锐利
特效: 蓝色/白色气劲，冷静克制
```

### 悲伤战斗
```
招式: 沉重缓慢，威力惊人
表情: 眼神悲戚，招式带泪
特效: 灰色/黑色气劲，压抑沉重
```

```
竹叶纷飞，月光如水，意境深远

```

## 注意事项

### 物理合理性
- 轻功要有起落点，不能无限飞行
- 内力消耗要有表现，不能无限使用
- 受伤要有恢复过程，不能瞬间痊愈

### 艺术夸张
- 在合理范围内夸张表现
- 特效服务于情感表达
- 动作要有美感，不仅仅是暴力

### 文化内涵
- 体现武侠文化的精神内涵
- 招式名称要有文化底蕴
- 动作设计要符合人物性格
