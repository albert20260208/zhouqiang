# 胡导文件夹

## 简介
这是为胡导（导演）专门定制的提示词优化文件夹，包含他的个人偏好、风格模板、案例库和学习记录。

## 文件夹结构

```
hu-director/
├── README.md                    # 本文件
├── preferences/                 # 个人偏好设置
│   ├── style-preferences.json  # 风格偏好
│   ├── model-preferences.json  # 模型偏好
│   └── workflow-preferences.json # 工作流程偏好
├── templates/                   # 个人模板库
│   ├── scene-templates/        # 场景模板
│   ├── character-templates/    # 人物模板
│   └── shot-templates/         # 镜头模板
├── cases/                      # 个人案例库
│   ├── successful-cases/       # 成功案例
│   └── learning-cases/         # 学习案例
├── styles/                     # 个人风格库
│   ├── visual-style.json       # 视觉风格
│   ├── narrative-style.json    # 叙事风格
│   └── editing-style.json      # 剪辑风格
└── learning/                   # 学习记录
    ├── evolution-log.md        # 进化日志
    └── patterns-learned.json   # 学习到的规律
```

## 胡导风格特点

### 视觉风格偏好
- **色调**：偏好冷色调、蓝绿色调
- **光影**：喜欢强烈的明暗对比
- **构图**：偏爱不对称构图、框架构图
- **运动**：喜欢缓慢的推拉镜头

### 叙事风格偏好
- **节奏**：慢节奏，注重情绪铺垫
- **情绪**：忧郁、孤独、怀旧
- **主题**：时间、记忆、城市、人际关系

### 技术偏好
- **常用模型**：即梦 Seedance、Midjourney
- **分辨率**：偏好4K或更高
- **格式**：偏好竖屏短视频格式

## 使用说明

1. **个性化优化**：系统会根据胡导的偏好自动调整提示词
2. **风格继承**：新提示词会继承胡导的视觉和叙事风格
3. **持续学习**：每次交互都会更新胡导的学习记录
4. **模板调用**：可以快速调用胡导常用的模板
