# 视频制作经验总结 (2026-02-25 实战提炼)

基于贾维斯出场视频制作过程的经验总结。

## 🔴 光线气氛营造法则（2026-05-25 周强亲授）

### 核心原则
**【时间】字段定死全局光线基调后，不靠改整体光线来变气氛，而靠局部脸部光影烘托情绪。**

### 三个切入点

#### 1. 脸部光影描写
- 不需要改场景基准光线，在分镜内容里写具体人物脸上的光影变化
- ✅ `晨光照在他脏污疲惫的侧脸上`（人物脸部光线叙事）
- ✅ `阳光在他额头眉骨处投下阴影，遮住半张脸`（半明半暗表现内心冲突）
- ✅ `侧逆光勾勒出他脸部轮廓，正面暗部隐去表情`（压抑/神秘氛围）
- ❌ `光线变暗了`（改全局光线，和大段时间冲突）
- ❌ `场景光线略显阴沉`（在分镜里直接改场景光，破坏一致性）

#### 2. 人物定位与光线的关系
- 人物在场景中的站位决定他受什么光：
  - `铁刀坐在树下阴影中，只有几缕光斑落在膝盖上`（弱势/压抑）
  - `横肉男站在阳光里，晨光从背后照来在他脸上投下阴影`（压迫感/居高临下）
  - `铁刀脸埋在暗处，只露出一只眼睛反光`（阴郁/蓄势）

#### 3. 单一描写人物脸部光影
- 聚焦一个人脸的明暗变化就能传递情绪变化
- `阳光从他脸上慢慢移开，半张脸沉入阴影`（情绪低落/放弃）
- `一缕光恰好落在她闪动的睫毛上`（细腻/柔和）
- `光线从他眉骨上方打下，在眼窝处形成深色阴影`（严肃/阴鸷）

### 示例对照

**❌ 错误写法（改全局光线）**
```
【时间】清晨，光线略显阴沉
镜头：横肉男站在铁刀面前，光线变暗了，铁刀脸上表情看不清楚
```

**✅ 正确写法（人物脸部光影叙事）**
```
【时间】清晨，晨光明亮
镜头：横肉男粗壮身形挡住部分晨光，在铁刀脸上投下一道阴影
```

### 要点总结
- 【时间】段定了就定了，不改
- 气氛变化通过分镜里人物的光影描写实现

## 🔴 提示词算力效率法则（2026-05-25 周强纠正·重要）

### 核心原则
**模型不认感觉/触觉/温度/情绪形容词，只认视觉特征。**
每句描述都问自己：模型能不能画出来？不能就是浪费算力。

#### 无效描述（模棱两可/感觉类）
- ❌ `冰冷的剑柄` → 模型画不出"冰冷"，只画出一个剑柄
- ❌ `沉重的脚步声` → 模型画不出"沉重"，只画出脚落地
- ❌ `温暖的笑容` → 模型画不出"温暖"，只画出一个笑容
- ❌ `坚毅的眼神` → 模型画不出"坚毅"，只画出一个眼神
- ❌ `气氛紧张` → 模型不知道什么叫紧张

#### 有效替代（视觉可渲染）
- ✅ `指节因用力泛白` → 可见的物理现象
- ✅ `鞋底踩碎枯枝，尘土向两侧溅起` → 可见的物理效果
- ✅ `嘴角慢慢上扬，眼角挤出笑纹` → 可见的表情变化
- ✅ `眉头紧锁，目光直视镜头不动` → 可见的面部状态
- ✅ `横肉男粗壮身形挡住晨光，在铁刀脸上投下阴影` → 可见的光影关系

### 检查技巧
写每句描述后问自己：
> 这句话删了，生成的画面会不一样吗？
- 删了没区别 → 无效描述，删
- 删了画面会不同 → 保留

尤其注意**感觉类形容词**：冰冷、沉重、温暖、柔和、紧张、压迫——它们都是浪费算力的大户。

### 🔴 景别-细节匹配法则（2026-05-25 周强纠正）
**画面细节的精细度必须和景别匹配。** 非特写镜头写微细节=浪费算力，画面根本体现不出来。

| 景别 | 能写的细节级别 | 不能写的 |
|------|---------------|---------|
| 全景/远景 | 空间关系、大动作轨迹、光线氛围 | 手指动作、表情微变化 |
| 中景 | 身体动作、相对位置、主要物体状态、光影 | 指节泛白、瞳孔收缩 |
| 近景 | 上半身动作、表情、饰品 | 血迹纹理、毛孔细节 |
| 特写 | 微细节：指节泛白、血迹纹路、眼神光、衣料纤维 | 不限制 |

**检查方法**：写完后问自己——
这个景别的镜头，观众能看清这个细节吗？
看不清就删。

## 🔴 动态场面运镜规则（2026-05-25 周强纠正·重要）

### 核心原则
**动态场面（打斗、冲突升级、追逐等）不用固定镜头。**
根据场景类型选择对应的运动镜头，固定镜头只用于文戏/对话。

### 场景类型→镜头策略

| 场景类型 | 推荐运镜 | 不合适的运镜 |
|---------|---------|------------|
| 文戏/对话/对峙 | 固定镜头为主（稳，给演员表演空间） | 手摇、跟拍（分散注意力） |
| 冲突升级/紧张积蓄 | 缓推+轻微手摇（积蓄，火药味） | 快切、快速摇镜（破坏氛围） |
| 打斗/爆发 | 跟拍+手摇+时而快速横摇（力量感） | 全程固定（没有冲击力） |
| 关键瞬间/重击 | POV/FPV（极致的代入感） | 固定远景（观众感受不到冲击） |
| 追逐/跑动 | 跟拍+低角度+轻微摇晃（速度感） | 固定中景（丢失运动轨迹） |
| 关键道具/物体 | 缓推+特写环绕（引导观众视线） | 固定远景（丢失焦点） |

### 常用的动态运镜

1. **手摇（Handheld）**：冲突升级时轻微晃动，让观众感受到紧绷感和火药味
2. **跟拍（Tracking）**：打斗中跟随动作轨迹，增强代入感
3. **POV/FPV**：重击的关键一刻，用施害者视角制造极致冲击力
4. **环绕（Arc）**：蓄力或对峙时围绕人物半圈，增加压迫感
5. **缓推（Push-in）**：引导观众注意力到关键表情/道具上
6. **快速横摇（Whip Pan）**：动作爆发时快速甩镜头跟上动作，创造速度感

## ⚠️ 常见问题与避坑指南

### 1. 图片与视频比例匹配
- ❌ 错误: 3:1宽图 → 视频拒绝(宽高比需在0.4-2.5之间)
- ✅ 正确: 16:9 (2848x1600) → 完美适配视频
- **教训**: 图生视频前,先确认图片比例

### 2. 模型ID要准确
- ❌ `doubao-seedream-5.0-lite` (文档里的名字)
- ✅ `doubao-seedream-5-0-260128` (实际调用ID)
- **教训**: 从控制台代码示例复制,不要猜

### 3. 多人场景的人物高度比例 ⭐重要
- **问题**: 多人同框时,身高比例不协调(主角太高或太矮)
- **解决**: 提示词中明确写入人物高度比例
- **示例**:
  ```
  主角身高约175cm,与周围行人身高比例协调一致
  人群中主角不突兀,自然融入环境
  前景人物与背景人物保持透视比例正确
  ```
- **检查点**: 生成后检查人物大小是否合理

### 4. 视频时长限制
- Seedance 1.5 Pro: 5秒或10秒
- **策略**: 复杂运镜用5秒,避免失败

### 5. 图生视频的平衡
- 提示词太多 → 可能偏离参考图
- 提示词太少 → 动态不够
- **平衡点**: 明确运镜+关键动态,其他依赖参考图

## ✅ 成功要点

### 1. 场景一致性控制
**方法**: 图生图/图生视频 + 参考图输入

**关键词**: 
```
保持参考图中XXX完全一致
保持参考图中的场景、建筑、色调、霓虹风格高度完全一致
```

### 2. 提示词结构化
借鉴官方示例的分块结构效果更好:
```
【场景】具体环境描述
【人物/动态】角色和动作(含身高比例)
【运镜】镜头运动方式
【风格】导演风格参考
【色彩与光线】色调+光效
【质感】渲染方式+分辨率
```

### 3. 运镜描述要具体
- ❌ 模糊: "镜头移动"
- ✅ 清晰: "镜头快速斜向上拉升爆发,如火箭般冲天,掠过建筑侧面"
- **关键**: 动作+速度+路径+效果

### 4. 导演风格融入
**王家卫风格关键词**:
- 抽帧跳跃感
- 霓虹色彩流转(红蓝绿粉紫)
- 高对比度光影
- 胶片颗粒质感
- 浅景深背景虚化

### 5. 细节丰富化
- 街道场景加入: 行人+雨伞+AR眼镜+小贩+飞行器
- 环境动态: 霓虹闪烁+数据流+云层飘动+雨后反射
- **效果**: 画面更有生命力

### 技巧1: 三段式描述

### 6. 人物素材审核问题处理 ⭐重要
- **问题**: 使用真实人物素材时，即梦可能因肖像权或内容审核拒绝生成
- **解决方案**: 使用PS等工具处理人物脸部特征
  - **技巧1**: 缩小脸部特征（眼睛、鼻子、嘴巴等）
  - **技巧2**: 轻微调整面部轮廓
  - **技巧3**: 改变发型或发色
  - **技巧4**: 添加眼镜、帽子等配饰
- **注意事项**:
  - 脸部变形是必然的副作用，需在可接受范围内
  - 保持人物基本辨识度与角色特征
  - 处理后的人物应更像"艺术创作"而非真实肖像
- **应用场景**:
  - 短剧制作中使用真实演员素材
  - 商业广告中避免肖像权问题
  - 需要特定外貌但无法通过纯文本描述实现时

### 7. 审核规避策略
1. **预处理素材**: 在输入前对人物图片进行艺术化处理
2. **特征弱化**: 减少过于明显的个人特征
3. **风格化**: 添加滤镜或艺术效果
4. **组合处理**: 使用多个参考人物的混合特征

### 8. 裸体场景审核技巧 ⭐重要 (2026-04-23 新增)
- **问题**: 即梦对裸体内容审核严格，直接描述"裸体"、"裸露上半身"等会被拒绝
- **解决方案**: 使用艺术化、专业化的描述替代
  - **技巧1**: "展示人物上半身肌肉" - 用于男性角色裸露上半身场景
  - **技巧2**: "展示人物身体线条" - 用于健身、艺术人体场景
  - **技巧3**: "展示人物背部/肩部肌肉结构" - 用于特定部位展示
  - **技巧4**: "人物身着轻薄贴身衣物" - 用于若隐若现效果
- **注意事项**:
  - 避免直接使用"裸体"、"裸露"等敏感词汇
  - 使用专业健身、艺术、医学等领域的描述词汇
  - 结合场景上下文，如"健身训练中展示肌肉线条"
  - 保持艺术性和专业性的描述角度
- **应用场景**:
  - 健身/运动类视频中展示肌肉
  - 艺术人体摄影/绘画参考
  - 医疗/解剖学教学视频
  - 古装/武侠剧中受伤包扎场景

**示例对比**:
- ❌ 错误: "男人裸体躺在床上"
- ✅ 正确: "男人展示上半身肌肉线条，躺在床上休息，电影级光影效果"
- ❌ 错误: "女人裸露上半身"
- ✅ 正确: "女人展示身体线条，身着轻薄贴身衣物，艺术人体摄影风格"

## 🚫 物理崩塌 & 剧情脱节检查清单 (2026-04-22 新增)

### 核心原则
**写提示词前，必须逐条检查以下规则，确保每一帧都符合真实物理规律和剧情逻辑。**

### 一、物理规律类检查项

#### 1. 动作逻辑连贯性
- [ ] 拧瓶盖→拔瓶塞→开盖→上药（不能跳步骤）
- [ ] 拿东西→手部要有明确的抓取动作，不能凭空出现在手里
- [ ] 放东西→要有放下/松手的动作序列
- [ ] 物体交互→手和物体接触位置要合理
- [ ] 穿衣/脱衣→要有明确的穿脱过程
- [ ] 开关门→手触门把→旋转→推/拉→门开/关

#### 2. 物体状态一致性
- [ ] 玻璃碎了就是碎了，不能再恢复成完整花瓶
- [ ] 水倒了就是倒了，杯子不能自己立起来
- [ ] 破碎/撕裂/变形→不可逆状态变化
- [ ] 消失/出现→必须有明确动作解释
- [ ] 液体泼洒后衣服上的湿痕要持续
- [ ] 被撕碎的文件不能恢复完整

#### 3. 空间/透视逻辑
- [ ] 人物在画面中的相对位置保持一致
- [ ] 近大远小透视正确
- [ ] 遮挡关系合理（人走后背景恢复，不能留半个人影）
- [ ] 影子方向、倒影位置符合光源
- [ ] 多人同框时身高比例协调
- [ ] 前后景人物大小符合透视关系

#### 4. 重力与运动
- [ ] 物体自由落体速度合理（不是慢慢飘）
- [ ] 被打倒后不会立刻站起（需要合理时间）
- [ ] 跑步时衣摆/头发摆动方向符合运动方向
- [ ] 水的流动、烟的飘散符合物理
- [ ] 跳起→落下→有缓冲动作

### 二、剧情/情绪类检查项

#### 5. 表情-语气-剧情三角一致
- [ ] 愤怒的语气 → 愤怒的表情 → 愤怒的原因（前情铺垫）
- [ ] 哭泣 → 泪痕 → 红眼眶 → 哽咽声音
- [ ] 惊喜 → 眼睛睁大 → 嘴角上扬 → 呼吸加快
- [ ] 恐惧 → 瞳孔收缩 → 身体后倾/颤抖
- [ ] 台词的情绪和面部表情必须匹配

#### 6. 时间线/因果关系
- [ ] 前一秒还重伤躺地，不能下一秒活蹦乱跳
- [ ] 前一集刚吵架决裂，不能这一集开场就卿卿我我
- [ ] 道具状态要延续（上一幕被摔坏的东西不能完好出现）
- [ ] 受伤的人要有伤口/绷带延续
- [ ] 时间跨度（白天→晚上）要有过渡

#### 7. 环境一致性
- [ ] 天气、光照、季节不能突变
- [ ] 室内外温度感觉连贯（从室外进室内不能还有风雪吹）
- [ ] 同一场景的家具摆设、墙面色调保持一致
- [ ] 同一场戏中角色的服装不能无故变换
- [ ] 杯中的饮料量、桌上的物品位置要有连续性

### 三、提示词防呆锚点

在提示词中主动加入以下"物理锚点"来约束模型：

```markdown
【物理规则】
- 拧瓶盖时手部有明确的旋转动作，瓶盖不会凭空消失
- 物体落地后保持在地面，不会自动回到原位
- 人物动作有完整的起止过程，没有跳跃步骤

【状态延续】
- 上一幕被摔碎的玻璃花瓶，这一幕保持碎片状态
- 角色受伤部位持续有包扎/血迹
- 桌面上被碰倒的物品保持倒下的状态

【情绪延续】
- 男主怒气未消，语气低沉但克制，与前一幕的愤怒保持一致
- 角色哭泣后眼眶发红，声音哽咽
- 情绪转换需要合理的触发事件

【剧情逻辑】
- 角色之间的恩怨关系与前情一致
- 秘密/误会尚未揭露前，角色行为符合当前信息量
- 角色性格前后一致，不能为推进剧情做出违和反应
```

## 🔴 打斗场景设计指南（2026-06-10 周强纠正后学习整理）

### 核心原则

**1. 动作密度铁律**
- 9秒一镜到底至少塞6-8个攻防节拍
- 不留对峙/缓冲/呼吸时间，动手就是连续攻防
- 每秒都要推进：攻→防→转攻→攻→防→反→攻→收
- 参考：电影《极冻之城》7分钟长镜头由40颗镜头合成，平均每颗约10秒

**2. 对抗张力（兵器咬合感）**
- 每下交锋不是轻触即分，而是写清兵器撞击后的角力状态
- 格挡时的冲击力传递：被震退、用脚后踏卸力、硬顶住
- 力量感描述：动力链法（脚蹬地→转腰→送肩→发力），写清力量从哪来到哪去
- 物理反馈：剑身弯颤、手臂肌肉绷紧、两股力对抗时的僵持

**3. 攻防节奏**
- 不对称设计：一方主攻、一方被迫防守→然后攻防转换
- 不要写成对称的"你一刀我一刀"，而是一方连续施压，另一方被迫应对，然后找机会反击
- 节奏变化：密集快打中插入一次重击（慢蓄力→突然爆发），形成节奏对比

**4. 动力链描述法（写力量感的核心技巧）**
```
❌ 只写动作：A挥剑劈向B
✅ 写力量来源：A双脚蹬地，腰部发力带动右肩，手臂送剑从右上斜劈而下
❌ 只写结果：B被震退
✅ 写力量传导体感：剑刃在棍身炸出火花，B双臂一震，身体后仰半步，右脚后踏才稳住
```

**5. 环境交互利用**
- 利用场景元素增加打斗丰富度：踩碎枯叶、棍尖带起泥土、剑风扫落树叶
- 一棵树可以作为格挡/借力/闪避的中介物
- 但注意：使用场景元素是为了强化打斗张力，不是堆砌视觉效果

**6. 运镜与打斗的配合（一镜到底特化）**
- 运镜路线跟着武器轨迹走，武器画弧→镜头跟着弧线移动
- 景别变化的节奏：中景（看清全身招式）→中近景（看清兵器碰撞细节）→中景（看攻防转换全景）
- 角度变化：正拍看正面对抗→侧拍看动作方向和轨迹
- 不要在兵器碰撞的高潮瞬间做大的运镜动作（绕背/大范围移动），会丢焦点

**7. 一镜到底打斗的节拍框架示例（9秒）**
```
节拍1（0-1.5s）：A主动进攻，剑劈/B格挡→火花
节拍2（1.5-3s）：B受力后仰，借反弹转棍横扫
节拍3（3-4.5s）：A后撤避让，立即踏步反击/剑斜撩
节拍4（4.5-6s）：B举棍格挡，两人力量角力
节拍5（6-7.5s）：B拨开剑身，棍尾反打/A后仰闪
节拍6（7.5-9s）：最后一下冲击/被逼退/没有喘息
```

- 电影《John Wick》系列（以干净利落、高密度的打斗编排著称）
- 电影《英雄》中李连杰对甄子丹（纯兵器格斗的动作设计标杆）
