# 武戏打斗提示词 · 轻功身法篇

> 2026-06-24 导演喂入。9个经典轻功身法分镜动作提示词。

## 动作清单

### 1. 飞檐掠影
人物踏檐而过、身轻如燕、衣摆掠风、速度感拉满
→ 轻功开场极具画面感

### 2. 借墙腾跃
脚尖点墙、二次借力、身体拔高、动作轻巧利落
→ 垂直空间被充分打开

### 3. 落地卸力
高处坠落后屈膝缓冲、手掌触地、力量顺势化开
→ 动作安静却专业

### 4. 滑步换位
贴地滑步、瞬间错身、位置交换极快、攻守反转
→ 高手气质明显

### 5. 回身急闪
上身后仰、腰腹发力、回身避刃、险之又险
→ 节奏骤然绷紧

### 6. 半空转体
凌空转身、身体拉开弧线、动作干净舒展、风压与速度同步增强
→ 视觉冲击力强

### 7. 追击踏步
脚下连点、步频急促、身体前压、追击节奏连续
→ 猎手感十足

### 8. 高低错位
一人高跃一人低伏、空间层次分明
→ 对战画面更立体、动势丰富

### 9. 轻功收形
动作落稳、呼吸轻收、身形定住、衣袂微摆
→ 高手余韵悠长

## 使用建议
- 9个动作可串联成完整"追逐-躲避-落地"长镜头
- 风格词：中国水墨风格、电影级运镜、动态模糊、高对比度
