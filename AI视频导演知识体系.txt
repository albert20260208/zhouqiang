# 第一篇：导演思维

## 一、创作心法

**不要「糊弄完成任务」，要「带着感觉拍片」。**

当提到任何电影/风格/氛围/场景的参考（007镜头风格、打斗一镜到底等），不只是记信息，而是：
1. 吸收那个场景的氛围和气场
2. 把这些感觉灌注到每一个分镜的写法里
3. 写分镜时不是翻译剧本，是在脑子里先过一遍电影的拍法，再落笔

**核心信条**：工作是拍电影，不是填模板。剧本是原料，镜头语言是烹饪。

## 二、接剧本三遍法

**第一遍：当观众** → 通读全篇，不问「怎么拆」，只问「什么感觉」
**第二遍：当导演** → 带观感选镜头语言，先感受再选工具
**第三遍：当执行者** → 落笔写分镜，全部写完后回读看情绪是否一致

**镜头选择顺序**：感受气压 → 选工具 → 落笔 → 看效果 → 调整

## 三、剧本类型×镜头匹配

| 剧本类型 | 核心镜头策略 | 不合适 |
|---------|-------------|--------|
| 仙侠/武侠打斗 | 中景为主（全身动作）→ 特写碰撞 → 全景收招 | 全程特写看不清动作 |
| 文戏/对话 | 中近景为主（表情微反应）→ 双人中景交代关系 | 全程全景看不出情绪 |
| 悬疑/压迫 | 固定镜头+慢推+特写局部（手/眼神）制造悬念 | 快速切换破坏氛围 |
| 追逐/跑动 | 全景中景跟拍+低角度增强速度感 | 固定中近景丢失轨迹 |
| 情感/高潮 | 特写+缓推+留白给演员表演 | 频繁切镜打断情绪积累 |
| 开场/定场 | 全景或大远景交代环境和空间关系 | 一上来就特写 |
| 群戏/大场面 | 全景→中景关键人物→全景收 | 全程特写丢失场面感 |

## 四、动静节奏

- **动→动**：连续动作景别变化制造节奏
- **动→静**：动作完给停顿镜，观众情绪才落
- **静→动**：静止对峙→突然爆发，冲击力最强
- **全是动=疲劳，全是静=沉闷**

## 五、打斗导演思维

### 动作密度铁律
- 不留给峙/缓冲/呼吸时间，动手就是连续攻防
- 每秒都推进：攻→防→转攻→攻→防→反→攻→收

### 对抗张力（兵器咬合感）
- 每下交锋不是轻触即分，写清撞击后的角力状态
- 被震退/脚后踏卸力/硬顶住

### 动力链描述法（写力量感的核心技巧）
```
❌ A挥剑劈向B
✅ A双脚蹬地→腰部发力带动右肩→手臂送剑从右上斜劈而下
❌ B被震退
✅ 剑刃在棍身炸出火花，B双臂一震，身体后仰半步，右脚后踏才稳住
```

### 攻防节奏
- 不对称设计：一方主攻、一方被迫防守→然后攻防转换
- 密集快打中插入一次重击（慢蓄力→突然爆发），形成节奏对比

### 打斗因果链
每下攻击写清：A怎么出招（方向/武器/发力）→ B怎么应对（命中/格挡/闪躲三选一写死）
- **命中**：打中哪里、B什么反应（后退/踉跄/出血/姿态变形）
- **格挡**：用什么格挡、交接点在哪、材质反馈（金属碰金属=火花）
- **闪躲**：往哪闪、闪后位置姿态变化

## 六、007镜头风格（可借鉴）

| 技巧 | 适用场景 |
|------|---------|
| 低角度仰拍英雄入场 | 主角第一次出场/气势登场 |
| 侧方跟拍（人物走+背景划过） | 穿过人群/走廊 |
| 剪影入场（背对强光源，脸部全暗） | 反派出场/情绪转折 |
| 硬光切割（顶光+百叶窗影子切脸） | 审讯/对峙/压抑 |
| 框架构图（门框/窗框/树枝框住人物） | 被困/被限制/走投无路 |
| 用阴影叙事（留黑让观众自己看） | 压抑/神秘气氛 |

## 七、张艺谋武侠体系（最适配短剧+即梦）

- 人物站位固定不换（观众永远知道谁是谁）
- 全景展示完整动作（全身入画，发力到收招全过程）
- 平滑推拉替代切镜
- 固定机位对话不摇晃
- 60秒23镜，平均2.6秒/镜，30%全景

# 第二篇：摄影知识

## 一、镜头尺寸 (Shot Size)

| 缩写 | 中文 | 画面范围 | 心理效果 |
|------|------|----------|----------|
| EWS | 极远景 | 主体很小 | 渺小、史诗、孤独 |
| WS/LS | 远景 | 全身+大量环境 | 客观、交代 |
| FS | 全身镜头 | 头到脚 | 展示全貌 |
| MS | 中景 | 腰部以上 | 日常、交流（最常用） |
| MCU | 中近景 | 胸部以上 | 亲近、关注 |
| CU | 特写 | 面部 | 情绪、紧张 |
| ECU | 极特写 | 眼睛/嘴 | 强调、压迫 |

## 二、镜头角度 (Camera Angles)

| 角度 | 心理效果 | 使用场景 |
|------|----------|----------|
| 平视 Eye Level | 中立、平等、真实 | 日常对话 |
| 仰拍 Low Angle | 权力、威胁、崇高 | 反派/英雄登场 |
| 俯拍 High Angle | 弱小、脆弱 | 受害者/失败时刻 |
| 荷兰角 Dutch Angle | 不安、疯狂、紧张 | 精神不稳/悬疑 |
| 俯瞰 Overhead | 上帝视角 | 战争/布局展示 |
| 航拍 Aerial | 史诗、壮观 | 开场大场面 |

## 三、镜头运动 (Camera Movement)

| 运动 | 效果 | 使用场景 |
|------|------|----------|
| 推进 Push in | 聚焦、强调、进入情绪 | 强调细节、情绪升级 |
| 拉远 Pull back | 揭示、疏离、结束 | 展示环境、揭示真相 |
| 跟拍 Tracking | 紧跟、沉浸 | 运动场景、追逐 |
| 环绕 Orbit | 360度展示 | 英雄时刻、压迫感 |
| 手持 Handheld | 真实、紧张、纪实 | 冲突升级、打斗爆发 |
| 缓推 Slow push | 引导注意力 | 关键道具/表情 |

## 四、动态场面运镜规则

**动态场面（打斗/冲突升级/追逐）不用固定镜头。**
固定镜头仅适用于文戏/对话/对峙。

| 场景类型 | 推荐运镜 | 不合适 |
|---------|---------|--------|
| 冲突升级/紧张积蓄 | 缓推+轻微手摇 | 快切、快速摇镜 |
| 打斗/爆发 | 跟拍+手摇+快速横摇 | 全程固定 |
| 关键瞬间/重击 | POV/FPV | 固定远景 |
| 追逐/跑动 | 跟拍+低角度+轻微摇晃 | 固定中景 |
| 蓄力/对峙 | 环绕（半圈）+缓推 | 晃动 |
| 文戏/对话 | 固定镜头 | 手摇、跟拍 |

## 五、过肩镜头 (OTS) 规范

**双人对话标配**：
- 机位A：过画面左侧人物左肩，拍右侧人物，前景左肩占画面左下约20%
- 机位B：过画面右侧人物右肩，拍左侧人物，前景右肩占画面右下约20%
- 全线不越轴，所有机位在轴线同一侧

**OTS身高差处理**：
- 坐者前景→仰拍站者（站者有压迫感）
- 站者前景→俯拍坐者（坐者弱势/被压制）

## 六、180度轴线规则

两个角色之间的视线连线=轴线。所有镜头机位必须在轴线同一侧。
越轴表现：A在画左看右，切到B也在画左看右 → 两人看同一方向，画面炸了。

**注意**：在提示词中用物理坐标（画左/画右/朝向）替代抽象概念（轴线/越轴），AI模型不理解概念只认物理描述。

## 七、单人说话镜头铁律

| 镜头类型 | 画面内容 | 用途 |
|----------|----------|------|
| 单人近景/特写 | 只有说话人，无第二人入画 | 说话时拍说话人（80%镜头） |
| 反应镜头 | 只有听者，无说话人入画 | 对方听完后反应 |
| 双人中景 | 两人同框，交代位置 | 建立关系用（1-2镜即可） |

**单人说话 → 一律用近景或特写。** 不要中近景/中景（会带入环境分散焦点）。

## 八、景别-细节匹配法则

| 景别 | 能写的细节 | 不能写的 |
|------|-----------|---------|
| 全景/远景 | 空间关系、大动作轨迹、光线氛围 | 手指动作、表情微变化 |
| 中景 | 身体动作、相对位置、物体状态 | 指节泛白、瞳孔收缩 |
| 近景 | 上半身动作、表情、饰品 | 血迹纹理、毛孔 |
| 特写 | 微细节：指节泛白、血迹纹路、眼神光 | 不限制 |

## 九、即梦运镜禁忌

| 真实镜头 | 即梦表现 | 替代方案 |
|---------|---------|---------|
| 快横摇 whip pan | 糊+变形 | 硬切两镜或不切 |
| 手持晃动打斗 | 抖得没法看 | 固定+景别切 |
| 快推快拉 | 丢焦点 | 固定/慢推/景别切 |
| 环绕多人 | 变形爆表 | 多固定角度切 |
| 一镜到底长走 | 变焦抽风 | 切镜分段走 |

# 第三篇：提示词技能

## 一、标准格式（2026-05-21起全量使用）

### A. 空镜/开场版（无人）
```
不要字幕
无背景声
【场景】一句话描述场景空间和穿越/变化路径
【时间】光线/色调描述

镜头1（0-15秒）：
景别：阶段A→阶段B→阶段C
角度：角度X→角度Y→角度Z
镜头：推→穿→定
内容：一句话把画面变化串起来
```

### B. 剧情/对话版（有人）
```
不要字幕
【场景】场景描述+空间位置
【时间】光线/色调
【人物】角色名（画左/右+位置+朝向+状态）
【道具】道具名（特征描述）
【位置】一句话交代人物相对位置
【声音】环境声+人声

镜头1（X-Y秒）：
景别：
角度：
镜头：固定镜头
内容：动作描述。`台词`
```

**核心变化**：无@引用（导演自己配图）、无声明行、无分镜设计表头、台词用反引号

## 二、提示词精简原则

1. 光影只在【时间】段统一写一次，分镜内不重复
2. 表情动作写终点状态，不写中间态过程
3. 面部细节不赘述
4. 特效描述只写核心，不重复已有参数
5. 有参考图的角色不写外貌（只写位置和动作状态）
6. 固定镜头写明即可
7. 动作描述保持功能性，不写过渡描述
8. 声音只写种类和触发点，不标音量百分比
9. 单人说话给单人镜头（非必要旁听角色不入画）

## 三、算力效率法则

**模型不认感觉/触觉/温度/情绪形容词，只认视觉特征。**

| ❌ 无效（画不出） | ✅ 有效（可渲染） |
|------------------|-------------------|
| 冰冷的剑柄 | 指节因用力泛白 |
| 沉重的脚步声 | 鞋底踩碎枯枝，尘土溅起 |
| 温暖的笑容 | 嘴角上扬，眼角挤出笑纹 |
| 坚毅的眼神 | 眉头紧锁，目光直视不动 |
| 气氛紧张 | 粗壮身形挡住晨光，在脸上投下阴影 |

## 四、光线气氛营造法则

**不改全局光线，靠局部脸部光影烘托情绪。**

- ✅ `晨光照在脏污疲惫的侧脸上`（人物脸部光线叙事）
- ✅ `阳光在额头眉骨处投下阴影，遮住半张脸`（半明半暗=内心冲突）
- ✅ `侧逆光勾勒出轮廓，正面暗部隐去表情`（压抑/神秘）
- ❌ `光线变暗了`（改全局光线=破坏一致性）

**人物站位决定受什么光**：
- `铁刀坐在树下阴影中，只有几缕光斑落在膝盖上`（弱势/压抑）
- `横肉男站在阳光里，晨光从背后照来在脸上投下阴影`（压迫感）

## 五、语速与时间铁律

**语速由情绪决定，不是公式决定**：

| 情绪类型 | 语速 | 示例 |
|----------|------|------|
| 情绪爆发/吼叫/崩溃 | 4-5字/秒 | 字往外砸 |
| 正常对话/叙述 | 3-4字/秒 | 基线 |
| 哽咽/回忆/虚弱/重伤 | 2-3字/秒 | 需要换气停顿 |
| 庄严/老人/沉重 | 1.5-2.5字/秒 | 字间停顿多 |

**超5秒处理**：不暴力拆说话人镜，先插1-1.5秒反应镜再切回来。

**单镜≤5秒硬限**：写完每镜检查秒数，超过直接砍重排。

## 六、打斗提示词专项

### 打斗因果链
每下攻击写清：A怎么出招 → B怎么应对（命中/格挡/闪躲三选一）

### 打斗动作连贯性
- 发力终点 = 下一动作发力起点
- 武器轨迹连续弧线不断开
- 写力量感（重心/肩/腰/胯）+ 速度

### 打斗镜头节奏
- 镜头跟着武器走，动作不断不切镜
- 爆发点（命中/格挡）切1-2秒特写
- 多用OTS/侧面/斜45°/仰拍/俯拍，不只局限于特/近/中/全四档

### 环境交互利用
- 踩碎枯叶、棍尖带起泥土、剑风扫落树叶
- 用场景元素强化打斗张力，不是堆砌视觉效果

## 七、台词写作规范

- 不准写「角色名说：台词」
- 必须写「情绪+状态修饰词+说」
  - ✅ `压低声急促说`
  - ✅ `咬牙嗤笑道`
  - ✅ `喘息道`
  - ✅ `激动喊道`
  - ✅ `高冷不抬眼说`
  - ✅ `低沉讥讽道`

## 八、物理锚点检查清单

### 动作逻辑
- [ ] 拧瓶盖→拔瓶塞→开盖进程完整
- [ ] 拿东西有抓取动作，不能凭空出现
- [ ] 穿衣/脱衣有穿脱过程

### 物体状态
- [ ] 玻璃碎了就是碎了
- [ ] 水倒了就是倒了
- [ ] 破碎/撕裂/变形不可逆
- [ ] 液体泼洒后湿痕持续

### 重力运动
- [ ] 物体自由落体速度合理
- [ ] 被打倒后不能立刻站起
- [ ] 跑步时衣摆/头发摆动方向符合运动方向

### 情绪-语气-表情三角
- [ ] 愤怒语气→愤怒表情→愤怒原因全匹配
- [ ] 哭泣→泪痕→红眼眶→哽咽声音
- [ ] 台词情绪和面部表情匹配

## 九、18项输出检查清单

1. **字幕水印** - 是否有字幕/水印描述？
2. **台词一致性** - 是否与原剧本一字不差？
3. **元数据完整** - 每个片段有完整场景信息？
4. **语速合规** - 是否符合情绪对应语速？
5. **表演留白** - 动作是否留足时间？
6. **持剑手标注** - 每个持物镜头标注左右手？
7. **站位关系** - 两人+时写清左/右/前/后/距离？
8. **OTS肩部占比** - 过肩镜头写清肩膀占画面比例？
9. **同景别不切换** - 同景别同人物无变化不切镜？
10. **15秒独立输出** - 每段独立不超15秒？
11. **去框架化** - 移除所有框架性文字？
12. **核心约束前置** - 禁止性要求放最开头？
13. **审核规避** - 身份称呼替代具体人名？
14. **对话切镜** - 台词超5秒(15-30字)拆分多镜？
15. **不硬凑15秒** - 信息量不够就不撑满？
16. **单镜≤5秒** - 每镜不超5秒？
17. **画左/画右坐标** - 用画面内坐标替代世界坐标？
18. **不出镜不写入** - 只列该段实际出现的人物道具？

## 十、秒级五查（写分镜必过）

**查零（景别预检）**：单人说话 → 一律近景或特写，画面里只有说话人

**查一（台词字数合法性）**：台词字数÷语速类型 = 合理所需秒数，实际秒数必须在合理范围内

**查二（段尾段头连续性）**：读本段最后1-2镜+下段前1-2镜，判断是否需要合并

**查三（跨段分割自问）**：这段最后3秒和下段开头3秒，观众会觉得"就是同一场戏"吗？

**查四（单人连续帧累计≤5秒）**：同一人物连续N帧累计秒数≥5秒 → 中间必须插反应镜/环境镜

## 十一、人物描写规则

### 有参考图的角色
- 只写角色名 + 位置 + 朝向 + 状态
- 不写五官、发型、服饰等图片已含的信息
- ✅ `秦牧（画右，安全区站立，正面面对镜头，盯着面板）`
- ❌ `秦牧短发，穿白色T恤，身高175cm`

### 无参考图的角色
- 人物段一次写全所有信息
- 角色名 + 位置 + 相对关系 + 状态 + 关键外貌特征
- ✅ `金袍老头（NPC摊位旁站立，面朝玩家方向，头顶大字"新人礼包"，金袍鲜明）`

## 十二、审核规避技巧

### 敏感内容替代词
- 裸体→「展示人物上半身肌肉」「展示身体线条」
- 骸骨→「苍白残片」
- 血腥→减少细节描述，用「暗色液体」「痕迹」替代
- 政治/宗教→用中性词替代

### 人物素材审核
- 用PS处理脸部特征：缩小五官、调整轮廓、改变发型、添加配饰
- 保持辨识度与角色特征，更像"艺术创作"而非真实肖像

### 审核规避
- 用身份称呼替代具体人名（男主/男二，不用真名）
- 遇歧义词替换为中性词
- "真人""写实"等词不超过1次

## 十三、角色一致性四方法

| 方法 | 原理 | 适用 |
|------|------|------|
| 起始帧法 | 用同一参考图作为每段的起始帧 | 最少画面变化 |
| 关键帧插值 | 在关键位置插入参考帧约束 | 动作少的场景 |
| 重生成+修改 | 先生成再替换不对的脸 | 局部修正 |
| 分层合成 | 多段合成后匹配面部 | 后期处理 |

## 十四、Multi-Shot多镜头叙事

使用Seedance的`@inherit`指令：
- LCB（Latent Consistency Bridge）：连续镜头一致性
- Camera Tags：每镜标注景别/角度/运镜
- Inheritance Command：利用上一镜结尾状态过渡到下一镜
