# 即梦（Seedance）审核通过速查手册

## 🚨 立即解决问题

### 人脸审核失败？用这些关键词：
```
# 动漫风格（最安全）
动漫风格，二次元，卡通渲染，非写实人脸
日系动漫，大眼睛，简化面部特征

# 艺术化处理
油画质感，明显笔触，艺术化面部
水彩画风格，色彩融合，非照片写实

# 低多边形
Low-poly风格，几何化面部，数字艺术
```

### 敏感词导致失败？立即替换：

| 原词 | 安全替代词 |
|------|------------|
| 巫师 | 神秘黑袍法师 |
| 魔法 | 能量操控 |
| 鬼魂 | 光影化身 |
| 血腥 | 红色液体/光效 |
| 死亡 | 失去意识/退出战斗 |
| 修炼 | 训练/修行 |

### 技术参数设置（复制使用）：
```
--style artistic --realism 0.3 --detail face:low --abstraction 0.7
```

## 📋 审核失败原因速查

### 1. 人脸问题（60%失败原因）
- ❌ 真实人脸特征
- ❌ 肖像权风险  
- ❌ 面部过于写实
- ✅ 解决方案：风格化处理

### 2. 敏感内容（30%失败原因）
- ❌ 暴力、血腥、恐怖
- ❌ 政治、历史敏感
- ❌ 封建迷信
- ✅ 解决方案：意象表达+科技化处理

### 3. 技术限制（10%失败原因）
- ❌ 文件超限（>12个）
- ❌ 格式不支持
- ❌ 大小超限
- ✅ 解决方案：检查参数

## 🎯 三步解决法

### 第一步：风险评估（2分钟）
1. 检查是否有人脸需求
2. 检查是否有敏感词
3. 确定安全等级

### 第二步：提示词优化（3分钟）
```
原提示词 + 风格化关键词 + 安全声明 + 技术参数
```

### 第三步：生成测试（5分钟）
1. 生成15秒测试视频
2. 检查审核风险
3. 调整优化


## 💡 创意解决方案

### 如果必须有人脸：
1. **动漫化** - 转为动漫风格
2. **艺术化** - 转为油画/水彩风格
3. **抽象化** - 转为几何/符号风格
4. **遮挡法** - 用道具/光线/动作遮挡

### 完全避免人脸：
1. **第一人称** - 主观视角
2. **背影叙事** - 用背影讲故事
3. **道具象征** - 物品代表人物
4. **环境主导** - 场景成为主角
