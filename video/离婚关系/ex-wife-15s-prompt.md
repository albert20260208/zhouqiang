# 前妻对话 - 15秒情感爆发提示词

## 场景分析

### 对话内容
```
女主："But..."
男主："You are my ex-wife. I'm James boss. The three from the Shen family, they dare not tell others about our marriage to the public."
```

### 情感变化
1. **女主**：微微怔愣 → 眼眶泛红 → 低落难过
2. **男主**：神色认真 → 语气坚定 → 略带复杂

### 时间分配（15秒）
- 0-5秒：对话开始，女主说"But..."
- 5-10秒：男主陈述事实
- 10-15秒：情感反应，特写表情变化

## 导演技巧应用

### 王家卫风格元素
1. **抽帧效果**：动作不连贯，增强情感梦幻感
2. **霓虹色调**：冷蓝办公室 + 窗外暖色霓虹
3. **慢动作特写**：表情变化慢动作处理
4. **内心独白感**：通过表情传达未说出口的话

### 情感张力构建
1. **距离控制**：1米距离，亲密又疏远
2. **视线交流**：眼神接触与回避的节奏
3. **微表情层次**：从控制到崩溃的情感递进
4. **沉默运用**：对话间的沉默比语言更有力

### 镜头语言
1. **特写力量**：面部特写传递内心情感
2. **推进压迫**：镜头推进增强情感压力
3. **环绕复杂**：环绕镜头展现关系复杂
4. **光线情绪**：冷光理性 vs 暖光情感
