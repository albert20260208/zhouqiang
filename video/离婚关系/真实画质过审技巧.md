# 即梦真实画质视频过审完全指南

## 🎯 核心挑战

### 即梦平台限制
- **明确禁止**：真实人脸（平台合规要求）
- **审核机制**：人脸检测算法识别"过于真实"的面部
- **技术限制**：无法完全绕过，但可以巧妙处理

### 用户需求矛盾
- ❌ 平台：不要真实人脸
- ✅ 用户：想要真实画质
- 🎯 目标：在限制内最大化真实感

## 🛡️ 解决方案体系

### 策略一：真实画质 + 非真实人脸

#### 1. 风格化人脸 + 写实环境
```
【环境】超写实场景，电影级摄影，真实质感
【人物】动漫风格角色（保持环境真实，只改变人脸）
【技巧】人脸风格化，环境保持写实
```

#### 2. 半脸处理技术
```
【镜头】3/4侧面，部分脸部在阴影中
【光线】侧逆光，面部一半明亮一半阴影
【角度】俯拍或仰拍，减少正面暴露
```

#### 3. 遮挡艺术
```
【道具】帽子、眼镜、口罩、围巾
【环境】窗户栏杆、植物枝叶、雨滴雾气
【动作】转身、低头、抬手遮挡
```

#### 1. 照片级环境 + 风格化人物
```
# 提示词示例

#### 3. 第一人称视角

## 🎨 具体实施技巧

### 技巧1：环境极致真实化
```
# 环境描述要点
- 材质细节：砖墙纹理、水泥裂缝、铁锈痕迹
- 光线效果：自然光散射、阴影渐变、反射光
- 空间感：景深控制、透视关系、空气透视
- 动态元素：飘动的灰尘、滴落的水珠、摇曳的光影

# 示例
照片级真实暗巷，砖墙每块砖的纹理清晰可见
水泥地面裂缝中的青苔细节，垃圾堆的真实质感
自然光从巷口斜射入，形成明显的光影对比
空气中飘浮的微尘在光线下可见，增强真实感
```

### 技巧2：人物巧妙处理
```
# 面部处理方案
方案A：低多边形风格化
  - 保持人体比例真实
  - 面部简化为几何形状
  - 保留表情基本特征

方案B：艺术滤镜效果
  - 油画笔触质感
  - 水彩晕染效果  
  - 素描线条风格

方案C：光影遮挡
  - 强烈逆光剪影
  - 斑驳树影遮挡
  - 雨雾朦胧效果
```

### 技巧3：情感表达转移
```
# 不依赖面部的表达方式
1. 身体语言：姿势、手势、肢体张力
2. 环境互动：与场景的物理交互
3. 声音设计：呼吸声、脚步声、环境音
4. 镜头语言：运镜节奏、剪辑节奏
5. 色彩情绪：色调、饱和度、对比度
```

### 模板1：环境真实+人物风格化（推荐）
```

## 🔧 技术参数优化

### 参数组合策略
```
# 方案A：环境高真实+人脸低真实
--style cinematic          # 电影风格
--realism 0.8             # 整体真实度80%
--detail face:low         # 面部细节低
--detail environment:high # 环境细节高
--texture quality:high    # 纹理质量高

# 方案B：艺术化平衡
--style artistic          # 艺术风格
--realism 0.6             # 真实度60%
--abstraction 0.4         # 抽象度40%
--stylization face:high   # 面部风格化高
--stylization env:low     # 环境风格化低

# 方案C：技术导向
--renderer path_tracing   # 路径追踪渲染
--ray_bounces 8           # 光线反弹次数
--global_illumination on  # 全局光照开启
--ssr_quality high        # 屏幕空间反射高质量
```
